# 有机反应中的控温

## 一、为什么需要控温

有机反应中，温度同时影响两个关键因素：
- **热力学**：决定反应能否发生、产物是否稳定（平衡位置）
- **动力学**：决定反应速率快慢

两者往往相互矛盾——高温有利于克服能垒（动力学），但可能让反应停留在热力学不稳定的状态，或引发副反应。

---

## 二、常见需要控温的反应类型

### 1. 放热剧烈的反应
| 反应类型 | 控温原因 | 典型温度 |
|---------|---------|---------|
| 硝化反应 | 放热剧烈，温度过高易多硝化或氧化 | 0~50°C |
| 磺化反应 | 高温导致多磺化或逆反应 | 室温~200°C（视目标产物） |
| 卤化反应（自由基） | 光照/高温引发链式反应，需控制选择性 | -20~80°C |
| 格氏试剂制备 | 反应放热，乙醚沸点低（34.6°C），需防止冲料 | 回流温度 |

### 2. 对温度敏感的选择性反应
| 反应 | 低温产物 | 高温产物 |
|-----|---------|---------|
| 1,3-丁二烯的加成 | 1,2-加成（动力学控制） | 1,4-加成（热力学控制） |
| α,β-不饱和羰基化合物的加成 | 1,2-加成（动力学） | 1,4-加成（热力学） |
| 烯烃与HBr的加成（有过氧化物） | 反马氏产物 | 可能回归马氏 |

### 3. 涉及活泼中间体的反应
- **重氮盐反应**：重氮盐在低温（0~5°C）稳定，高温分解
- **Vilsmeier-Haack反应**：需在0°C以下滴加POCl₃，防止副反应
- **低温锂卤交换**：-78°C（干冰/丙酮浴）进行，避免Wurtz偶联

### 4. 酶催化/生物模拟反应
- 酶的最适温度通常在25~40°C
- 高温导致蛋白质变性失活

## 三、热力学控制 vs 动力学控制

```
能量
  ↑
  │    反应物 ──────→ 产物A（动力学产物，活化能低）
  │         \      /
  │          \    /
  │           \  /
  │            过渡态1（低）
  │             |
  │            过渡态2（高）
  │           /  \
  │          /    \
  │         /      \
  │    产物B（热力学产物，能量更低）
  └────────────────────────────→ 反应进程
```

| 控制类型 | 条件 | 产物特点 | 实例 |
|---------|------|---------|------|
| **动力学控制** | 低温、短时间 | 活化能低，生成快 | 1,2-加成 |
| **热力学控制** | 高温、长时间、可逆 | 能量最低，最稳定 | 1,4-加成 |

**关键公式**：
- 阿伦尼乌斯方程：$k = A \cdot e^{-E_a/RT}$（温度↑，速率↑）
- 平衡常数：$K = e^{-\Delta G°/RT}$（温度影响平衡位置）

## 四、实用控温手段

| 温度范围 | 冷却/加热方式 |
|---------|-------------|
| -78°C | 干冰 + 丙酮 |
| -40~-20°C | 乙腈/乙醇 + 液氮 |
| 0~5°C | 冰水浴 |
| 室温~100°C | 水浴/油浴 |
| >150°C | 沙浴、金属浴、微波 |

---

## 五、一句话总结

> **低温抢速度（动力学），高温追稳定（热力学）；放热反应防失控，敏感中间体要冷藏。**
