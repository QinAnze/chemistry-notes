## 卤代烃
### 命名 [[命名]]
1. 烃为母体，卤原子为取代基，与烷烃命名相近 
2. 选连有取代基(含卤原子)最多的最长链为主链
3. 主链编号：从靠近取代基一端开始编号，使取代基位次最小； 有选择余地时，使英文首字母排前的取代基编号较小

### 反应
1. 亲核取代 [[取代]]
	1. 原理：R-X中$-R为\delta+$而$-X为\delta-$，所以C容易受负离子或带有孤对电子的中性分子的进攻，此类分子为Nu [[术语#Nu和E]] 此时-X为L [[术语#L]]
	2. 反应式：$$\mathrm{R-X} + \mathrm{Nu} \rightarrow \mathrm{R-Nu} + \mathrm{X^-}$$
	3. 常见反应：
		1. 水解：$\mathrm{RX} + \mathrm{H_2O} \rightarrow \mathrm{ROH} + \mathrm{HX}$，在碱性条件可以中和HX，使得反应正向进行 [[控pH反应]] 反应活性：RI>RBr>RCl
		2. 醇解：Williamson反应 [[人名反应]]：合成醚 $$\mathrm{RX} + \mathrm{R'ONa} \rightarrow \mathrm{ROR'} + \mathrm{NaX}$$ 适用于伯卤代烷烃，仲的效果差，叔走消除
		3. 氨解：$$\mathrm{NH_3} \xrightarrow{\mathrm{RX}} \mathrm{RNH_2} \xrightarrow{\mathrm{RX}} \mathrm{R_2NH} \xrightarrow{\mathrm{RX}} \mathrm{RN} \xrightarrow{\mathrm{RX}} \mathrm{R_4N^+X^-}$$ 最终生成季铵盐
		4. 氰解：$$\mathrm{RX} + \mathrm{NaCN} \xrightarrow[\text{醇}]{} \mathrm{RCN} \xrightarrow[\text{水解}]{} \mathrm{RCOOH}$$ 用于增长碳链
		5. 和炔基钠反应：$$\mathrm{RX} + \mathrm{R'C \equiv CNa} \rightarrow \mathrm{R'C \equiv C-R}$$" 适用于伯卤代烷烃
		6. 与硝酸银反应：鉴别卤代烷 $$\mathrm{RX} + \mathrm{AgNO_3} \rightarrow \mathrm{R-ONO_3} + \mathrm{AgX} \downarrow$$ 可以根据沉淀颜色鉴别卤代烷，反应活性：叔>仲>伯，伯需要Δ [[控温]]，RI>RBr>RCl，不适用于Ar-X
	4. 机理 [[术语#亲核取代 取代 立体选择性]] [[立体选择性]] [[控pH反应]]
		1. $S_{N}1$
			1. 单分子亲核取代，反应速率仅与R-Br浓度有关，反应两步进行
			2. 机理为-X离去，出现$R^+$碳正离子，经历碳正离子重排，Nu再进攻碳正离子，形成R-Nu键
			3. 决速步骤是R-X的断裂出现碳正离子
			4. 立体化学上，进攻碳正离子可以从任意方向进攻，导致最后R和S型的量几乎等量，外消旋化 [[立体选择性]] [[术语#外消旋/内消旋 立体化学]]
			5. 碳正离子稳定性决速，所以3'C>2'C>1'C
		2. $S_{N}2$
			1. 双分子亲核取代，反应速率与R-Br和Nu浓度均有关，反应一步协同进行
			2. 机理是Nu和R-X形成中间过渡态Nu-R-X后X离去，形成R-Nu和$X^{-}$，是一个协同的步骤，即R-Nu形成和R-X断裂同时发生
			3. 决速步骤是中间协同这一步
			4. 立体化学上，Nu从背面进攻，会导致构型翻转，称为Walden转化 [[人名反应]] [[立体选择性]] 注意：骨架构型发生改变，R/S符号 有可能不变
	5. 影响因素
		1. -R的影响 [[术语#决定因素]]
			1. -R对$S_{N}1$的影响：碳正离子稳定性，所以3'C>2'C>1'C
			2. -R对$S_{N}2$的影响：空间位阻，位阻越大，Nu越不易进攻，所以1'C>2'C>3'C
			3. 特殊的：
				1. 烯烃：C=C-X的C-X难以断裂，比如氯乙烯，不活泼，不走$S_{N}$反应
				2. 烯丙型：更稳定，更易于进行，走$S_{N}1$有重排，走消除得共轭二烯烃
				3. 苄基：更易反应
				4. 桥环得卤原子在桥头碳上，自身位阻大，且难离去，所以$S_{N}$都难以进行
		2. Nu的影响
			1. 概念：Nu性：亲核试剂对带有正电性碳原子的亲和力或者置换离去基团的能力
			2. Nu性大小，对$S_{N}1$无影响，对$S_{N}2$影响显著，Nu性增大，利于$S_{N}2$
			3. Nu性大小比较：
				1. 反应中心相同，碱性越强，亲核性越强 eg:$RO^{-}>HO^->ArO^- >RCOO^-$
				2. 负离子>共轭酸
				3. 同主族与溶剂极性有关：
					1. 极性质子性溶剂：Nu性与碱性相反，比如Nu性（羧酸，醇，水中）：$I^- >Br^- >Cl^-$
					2. 极性非质子性溶剂：Nu性与碱性相同，比如DMSO（二甲亚砜），乙腈，丙酮
					3. 非极性溶剂：Nu难溶，不适用
				4. 体积越小，位阻越小，Nu性越强 [[立体选择性]]
		3. 溶剂极性的影响
			1. 非极性：Nu难溶，不使用
			2. 极性质子性：利于$S_{N}1$，利于RX的解离，且和Nu形成HB，不利于$S_{N}2$进行
			3. 极性非质子性：利于$S_{N}2$进行，使负离子裸露，活性高
		4. L的影响
			1. L即X，C-X的强度决定了X的L性
			2. 一般来说，L能力：$R-SO_{3}Ph$>RI>RBr>RCl>$ROOC-CH_{3}$
			3. L能力强，对于$S_N1$更有利
			4. 判断离去能力：看离去基团的碱性，基团碱性强，离去倾向小，所以：$RO^-,HO^-,H_{2}N^-$不是好的离去基团
2. 消除 [[消除]]
	1. X为EWG，所以R-X的β-H有酸性，可以在NaOH/醇中消去HX得到碳碳双键三键
	2. Zaitsev规则 [[人名反应]]：主要消含H少的C上的H原子
	3. 邻二卤代物和偕二卤代物在强碱，乙醇，Δ环境下会脱两分子H形成炔烃，如果不加热，停留在烯卤代烃 [[控温]] [[控pH反应]]
	4. 邻二卤代物在Zn下的反应：$$\mathrm{R_1R_2CX-CXR_3R_4} \xrightarrow[\text{EtOH, }\Delta]{\mathrm{Zn}} \mathrm{R_1R_2C=CR_3R_4} + \mathrm{ZnX_2}$$
	5. 机理 [[术语#消除 消除 立体选择性]]
		1. E1
			1. 单分子消除，同样经历碳正离子阶段，第一步为α-C脱去X形成碳正离子并重排，第二步为β-C被强碱拔H，α-C与β-C形成双键
			2. 第一步是慢反应，决速步骤
			3. E1和$S_{N}1$第一步一样，所以会竞争，需要根据反应条件判断，eg:和水溶液，有氢氧根，走$S_{N}1$，和乙醇，有$EtO^-$，走E1
		2. E2
			1. 双分子消除，一步完成，同样有过渡态
			2. 立体条件 [[立体选择性]]：β-H和L必须反式共平面，此时能量更低，对于Nu的进攻更有利
			3. 立体专一性：赤式以cis产物为主，苏式以trans产物为主 [[立体选择性]]
			4. E2和$S_{N}2$也类似，如果Nu进攻β-H走E2，如果Nu进攻α-C走$S_{N}2$
	6. 影响因素
		1. -R
			1. 碳正离子稳定性影响E1
			2. R的β-H越多，更容易走E2，所以活性：3'C>2'C>1'C
		2. L
			1. 好的离去基团利于${S_{N}1}$和E1，反之利于$S_{N}2$和E2
			2. 活性RI>RBr>RCl
		3. 溶剂：极性溶剂利于E1，不利于E2
		4. 温度：高温有利于消除 [[控温]]
	7. 如果分子上有双键三键，那消烯丙位的H，这样形成共轭烯炔烃，更加稳定
	8. 大位阻试剂：适合末端碳碳双键和三键的形成，位阻小易进攻 [[立体选择性]]
	9. Hofmann规则 [[人名反应]]：动力学控制，适用于强碱弱碱化物（如氨、肼、肼衍生物）和环状氨碱（如四甲基氢氧化铵），和扎伊采夫规则的对比：![[屏幕截图 2026-06-01 175216.png]]
3. 消除和取代的竞争：![[屏幕截图 2026-06-01 175446.png]]
	1. RX结构影响：3'C-X易消除，1'C-X易取代
	2. Nu和碱性影响：Nu性强的利于$S_{N}2$，碱性强的利于E2

4. 与金属反应
	1. Wurts反应 [[人名反应]] $$\mathrm{RX} + 2\mathrm{Na} \rightarrow \mathrm{NaX} + \mathrm{RNa} \xrightarrow{\mathrm{RX}} \mathrm{R-R} + 2\mathrm{NaX}$$ 适用于伯卤代烷烃，X=Br,I
	2. Grignard试剂 [[人名反应]] $$\mathrm{RX} + \mathrm{Mg} \xrightarrow[\text{乙醚}]{} \mathrm{RMgX}$$ $$\mathrm{Ar-X} + \mathrm{Mg} \xrightarrow[\text{THF}]{} \mathrm{Ar-MgX}$$ THF为四氢呋喃，可以使得芳基卤化物被活化，从而制备格氏试剂 格氏试剂要严格无水无氧，现配现用，否则会水解 RX的产率：1'C>2'C>3'C
	3. 格氏试剂参与的反应：![[屏幕截图 2026-06-01 180527.png]]
	4. Li试剂
		1. 第一步反应 $$\mathrm{RX} \xrightarrow[\text{醚}]{\text{2 equiv. Li}} \mathrm{R-Li} + \mathrm{LiX}$$ 此时R-Li已经可以反应，但无立体选择性，会外消旋化 [[立体选择性]] 会过度反应，发生多取代，而且活性太高
		2. 所以，第二步反应：$$\mathrm{R-Li} + \mathrm{CuI} \xrightarrow[\text{低温}]{\text{THF}} \mathrm{R_2CuLi} + \mathrm{LiX}$$ $\mathrm{R_2CuLi}$叫做Gilman试剂 [[人名反应]]
		3. Corey-House反应 [[人名反应]]：Gilman试剂和卤代烃反应 $$\mathrm{R_2CuLi} + \mathrm{R-X} \xrightarrow[\text{醚}]{} \mathrm{R-R'}$$ 也适用于Ar-X，Gilman试剂的碱性，Nu性，反应速率都低于R-Li，使得其仅进行一取代，稳定性高，毒性小，而且立体选择性高，不会改变立体架构 [[立体选择性]]
		4. Kumada偶联 [[人名反应]] $$\mathrm{CH_2=CH-X} \xrightarrow[\mathrm{Pd}]{\mathrm{RMgX} \text{或} \mathrm{RLi}} \mathrm{CH_2=CH-R}$$
		5. 另外几个偶联：![[屏幕截图 2026-06-14 122651.png]]                     [[人名反应]]
		6. 重点：Suzuki偶联 [[人名反应]] $$\mathrm{R_1-X} + \mathrm{R_2B(Y)_2} \xrightarrow[\text{碱}]{\mathrm{Pd}} \mathrm{R_1-R_2}$$ Y为OH或烷基，这个偶联选择性好 [[控pH反应]]
5. 芳香亲核取代$S_{N}Ar$
	1. 机理：$$\mathrm{EWG-Ph-X} \xrightarrow{\mathrm{Nu}} \mathrm{EWG-Ph-Nu}$$ EWG可有可无，有了反应更快
	2. 应用：Sanger测序 [[人名反应]]：$\mathrm{pro.-NH_2}$和$\mathrm{F-Ph-(NO_2)_2}$反应，后续水解生成黄色沉淀，从而检测蛋白质
6. 制备卤代烃
	1. 自由基取代 [[取代]]
	2. 烯烃的自由基卤代 [[取代]]：用NBS在烯丙位取代Br，防止直接和双键反应 $$\mathrm{R_1R_2C=CH-CH_3} \xrightarrow[(\mathrm{PhCO})_2\mathrm{O}]{\mathrm{NBS}} \mathrm{R_1R_2C=CH-CH_2Br}$$
	3. Ar-X制备：催化卤代 [[取代]]
	4. 醇的取代 [[取代]]：$$\mathrm{ROH} + \mathrm{HX} \rightarrow \mathrm{RX} + \mathrm{H_2O}$$ 或者用$\mathrm{PX_3}, \mathrm{PX_5}, \mathrm{SOCl_2}$也可以
5. 不饱和烃加成 [[加成]]
6. 卤原子交换，只适用于制备碘代烃 $$\mathrm{RX} + \mathrm{NaI} \xrightarrow[\text{丙酮}]{} \mathrm{RI} + \mathrm{NaX} \downarrow$$ 原理：NaI溶于丙酮，但是NaCl和NaBr不溶
7. Blanc反应（F-C反应衍生） [[人名反应]] $$\mathrm{C_6H_6} + \mathrm{HCHO} + \mathrm{HCl} \xrightarrow{\mathrm{AlCl_3}} \mathrm{Ph-CH_2Cl}$$
