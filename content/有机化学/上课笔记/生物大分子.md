## 单糖
1. 定义：糖类是多羟基醛或多羟基酮，以及通过水解就 能产生这些醛、酮的物质
2. 分类：
	1. 单糖：不能再被水解成 为更小的糖分子 的糖类
	2. 寡糖：(低聚糖) 由2-9个 单糖分子失水而成 的糖类
	3. 多糖：由十个以上甚至 是几百、几千个 单糖分子失水而 成的糖类
3. 最简单的糖：甘油醛
	1. 甘油醛的羟基在右边：D-(+)-甘油醛 [[立体化学#构型标记]]
	2. 甘油醛的羟基在左边：L-(-)-甘油醛
4. 单糖的D,L标记法：以离醛酮基的最远的手性碳的D/L为主，比如对于Glc [[术语#糖]]：最远的是$C_{5}$，所以$C_{5}$的D/L就是葡萄糖的D/L
5. 差向异构体：含有多个手性碳原子的非对映体，彼此间仅有一个手性碳 原子的构型不同，其余的都相同，彼此称为差向异构体，eg:Glc的$C_{2}到C_{5}$为DLDD和LLDD是一对差向异构体
6. D-糖为主要自然界的糖类
	1. D-Glc=DLDD
	2. D-Gal=DLLD
	3. D-Fru=D(C=O)LDD
	4. D-Rib=DDD
7. Glc的特性：
	1. 仅仅与1equiv. MeOH缩合
	2. 和某些加成试剂不反应，比如$NaHSO_{3}$
	3. 有变旋现象
	4. 表明：已经是半缩醛
8. 环状结构：糖的醛酮基和羟基在分子内亲核加成 [[加成]] 形成分子内半缩醛
	1. Haworth透视式：![[屏幕截图 2026-06-11 103614.png]]
	2. 画法：环碳原子顺时针编号![[屏幕截图 2026-06-11 103752.png]]
	3. 注意：画Haworth式的时候，把$C_{5}$的除了骨架的剩下三个基团旋转，使得-OH在最底下，比如D-Glc的羟甲基转到了左边，倒下来就是在环上方，而$C_{1}$的-OH在下方，所以是α-型
	4. 半缩醛羟基和$-CH2OH$分布在环平面两侧的叫α-型体，分布在同侧的叫β-型体 [[立体选择性]]
	5. 在水溶液中，Glc的α-型体和β-型体能通过开链互变并建立平衡
	6. 与Fischer投影式的关系：L=上 D=下
	7. 环状结构产生翻环等异构：因为e键比a键稳定，所以β-(4e+1e)型比α-(4e+1a)型稳定，含量更高 [[立体选择性]]
	8. 碱性条件下，会形成烯二醇中间体，使得出现差向异构转化，所以对糖的反应要在酸性/中性环境下进行 [[控pH反应]]
### 反应
1. 氧化 [[氧化]]：Tollen和Fehling试剂 [[人名反应]]
	1. 酮不和这两个反应，但是在碱性条件下差向异构能生成醛基，所以重新能反应 [[控pH反应]]
2. 和溴水反应 [[氧化]]：溴水能氧化醛基为羧基，不能氧化酮，所以可以用溴水鉴别醛糖和酮糖，生成的羧酸会分子内酯化形成δ-内酯
3. 和稀硝酸 [[氧化]]：稀硝酸会氧化$C_{1}$醛基和$C_{6}$的$-CH_{2}OH$生成二羧酸；酮糖的$C_{2}和C_{3}$键会断裂，生成小分子二酸
4. $HIO_{4}$氧化 [[氧化]]：会把所有1~5的碳氧化成甲酸，留下末端的$C_{6}$生成甲醛
5. 还原 [[还原]]：催化氢化生成醇
6. 成脎反应：1equiv. 糖和3equiv. 苯肼($\mathrm{Ph-NHNH_2}$)反应，在糖的$C_1,C_2$形成二苯腙（糖脎） $$\mathrm{CHO(CHOH)_4CH_2OH} \xrightarrow[\text{3equiv. PhNHNH}_2]{} \mathrm{CH=NNHPh-C=NNHPh(CHOH)_3CH_2OH} + \mathrm{PhNH_2} + \mathrm{NH_3} + \mathrm{H_2O}$$
	1. $C_{1},C_{2}$不同的糖，生成的糖脎一致，其不影响$C_{3},C_{4},C_{5}$的构型 [[立体选择性]]
	2. 但是，不同糖反应时间不同，所以可以用来鉴别糖
	3. 苯肼只和糖的C-1和C-2成脎后，分子内氢键使其形成较为稳定的六 元环结构，从而糖的其他碳原子不再成脎
7. 处于糖环上相距较近的邻二醇（或1,3-二醇）可与醛或酮生成环状的缩 醛或缩酮 [[醛酮#反应]]
	1. 用于保护羟基
	2. trans.的邻二醇不反应 [[立体选择性]]
	3. 成苷：单糖的半缩醛(酮)羟基可与其它含羟基或活泼氢(如氨基、巯基)的化合 物脱水，生成糖苷（glycoside，也称甙），此反应称为成苷反应
		1. 由于端基效应， 产物以α-型为主 [[立体选择性]]
		2. 具缩醛（酮）结构，在水中不能转化为开链结构
		3. 糖苷无变旋现象，也无还原性
		4. 在碱中较稳定，但在酸作用或酶催化下，可断裂苷 键，生成原来的糖
## 寡糖和多糖
### 二糖
1. 蔗糖：无游离的半缩醛羟基，由α-D-吡喃Glc和β-D-呋喃Fru通过Glc的$C_{1}$和Fru的$C_{2}$通过糖苷键连接，可称作α-1,2-糖苷键或β-2,1-糖苷键
2. 麦芽糖：存在半缩醛羟基，为α-1,4-糖苷键，如果两个Glc均是α-则叫做α-麦芽糖，如果右边的是β-则叫做β-麦芽糖
3. 纤维二糖：纤维素水解的产物，存在半缩醛羟基，纤维二糖的两个都是β-Glc，通过β-1,4-糖苷键连接
4. 乳糖：有半缩醛羟基，β-D-Gal和D-吡喃Glc通过β-1,4-糖苷键连接
5. 环糊精：淀粉水解后的环状低聚糖，由6,7,8个D-吡喃Glc残基通过α-1,4-糖苷键连接，为外部亲水内部疏水的结构
### 多糖
1. 因为M过大，所以整体不具备还原性和变旋现象
2. 直链型通过α-1,4或β-1,4-糖苷键连接，支链通过α-1,6-糖苷键连接
3. 直链淀粉遇$I_{2}$变蓝，支链淀粉变红
4. 纤维素：D-Glc以β-1,4-糖苷键相连
5. 糖原：D-Glc通过α-1,4-糖苷键和α-1,6-糖苷键相连

## 氨基酸
1. 天然氨基酸：约有300多种
2. 蛋白质氨基酸：组成蛋白质的氨基酸，有20(22)种，其它氨基酸为 非蛋白质氨基酸
3. $-NH_{2}$在离羧基的α-C上就叫α-氨基酸，除此之外还有β-氨基酸，γ-氨基酸等
4. 蛋白质氨基酸除了甘氨酸外都为L型 [[立体化学]] 且蛋白质氨基酸都是α-氨基酸（氨基酸的L型：Fischer式的氨基在横向左边，羧基在上

	3. 与亚硝酸反应：Van Slyke测定法 [[人名反应]] $$\mathrm{RCHNH_2-COO^-} \xrightarrow[\mathrm{HNO_2}]{} \mathrm{R-CH(OH)COOH} + \mathrm{N_2} \uparrow$$ [[脱气体]] 放出的氮气是固定的，所以可以定量分析氨基酸
	4. 氨基被酰基取代，保护氨基，或者做比如Sanger测序的反应
	5. 氨基转移：氨基酸诶α-羰基酸反应，实现氨基酸的氨基和酸的羰基O互换，一般是酶催化
	6. 卤化或者酯化羧基，保护羧基
	7. 羧基脱羧，生成$CO_{2}$ [[脱气体]] 和胺类
	8. 茚三酮：和α-氨基酸成蓝紫色化合物，可以鉴定
## 多肽和蛋白质
1. 寡肽：2~10个氨基酸残基；多肽：>10个
2. 命名：从N端开始编号，到C端，假设氨基酸顺序是A氨酸，B氨酸，C氨酸，就叫ABC肽，或者叫：γ-A酰B酰C氨酸
3. 肽键的CO和$-NH_{2}$会共振成-C=N-使得肽键六原子共面，且结构是trans.为主 [[立体选择性]]
4. 活化羧基的活化试剂：DCC（二环己基碳二亚胺，强脱水剂）
5. 固相合成：在不溶的高分子树脂的表面上进行接肽反应称为固相接肽，是一个重复添加氨基酸的过程
6. 二级结构：α-helix, β-pleated sheet, β-turn, random coil [[术语#二级结构 立体化学]]
7. 三级结构：单个多肽链在三维空间中的完整折叠形态，由二级结构元件（α-螺 旋、β-折叠等）的进一步排列与相互作用形成，呈现特定的空间构象
8. 四级结构：四级结构：多个独立折叠的多肽链（亚基）通过非共价相互作用组装形成的复 合体结构，仅存在于多亚基蛋白质中
