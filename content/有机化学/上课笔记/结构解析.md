# 第十二章 有机化合物的波谱分析

## 章节概览

- **12.1** 电磁波与分子的吸收光谱
- **12.2** 红外光谱（IR）
- **12.3** 核磁共振谱（NMR）
- **12.4** 图谱解析示例

---

## 学习目标（重点）

> 1. 有机基团的**特征红外吸收峰**；红外吸收峰与有机物特征基团的关系
> 2. **化学位移**与氢原子核外电子云密度分布；影响化学位移的主要因素；化学位移与有机物结构关系
> 3. **积分曲线**与氢原子数目的关系；**自旋耦合和自旋裂分**
> 4. 用红外光谱和核磁共振氢谱对**有机物的结构解析**

---

## 一、概述——有机物结构表征基本程序

### 1.1 基本流程

```
分离提纯 → 元素定性定量分析 → 分子量测定 → 结构表征
```

| 步骤 | 方法 |
|------|------|
| 分离提纯 | 蒸馏、萃取、升华、重结晶、层析 |
| 元素分析 | 元素分析仪 |
| 官能团分析 | 化学法 + 物理法 |
| 分子量测定 | 凝固点降低法、沸点升高法、质谱（MS） |

### 1.2 化学方法 vs 物理方法

| | 化学方法 | 物理方法（波谱） |
|---|---|---|
| **特点** | 费时、费力、费钱，试剂消耗大 | 样品用量少，准确快速 |
| **例子** | 吗啡结构确定用了 146 年（1806→1952） | UV/IR/NMR/MS |
| **费用** | — | 仪器昂贵（NMR 100-1000 万） |

#### 四大波谱方法

| 方法 | 缩写 | 仪器价格 | 样品用量 | 主要用途 |
|------|------|----------|----------|----------|
| 紫外吸收光谱 | UV | 1-20 万 | 0.01-5 mg | 共轭体系 |
| 红外吸收光谱 | IR | 5-50 万 | 0.1-1 mg | **官能团鉴定** |
| 核磁共振谱 | NMR | 100-1000 万 | 1-5 mg | **结构测定** |
| 质谱 | MS | 50-500 万 | 0.001-0.1 mg | 分子量、碎片 |

## 二、电磁波与分子的吸收光谱（12.1）

### 2.1 电磁波的基本性质

- 光具有**波粒二象性**
- **波动性**：$\nu = c/\lambda$，$\sigma = 1/\lambda$（波数，cm⁻¹）
- **粒子性**：$E = h\nu = hc/\lambda = hc\sigma$
- **波长愈短，能量愈大**

### 2.2 电磁波类型与波谱分析方法

| 波谱方法 | 激发能级 | 波长范围 | 电磁波类型 |
|----------|----------|----------|------------|
| X 射线光谱 | 内层电子 | 0.01-10 nm | X 射线 |
| UV-Vis | n 及 π 电子 | 200-800 nm | 紫外-可见光 |
| IR | 振动与转动 | 0.8-300 μm | 红外线 |
| ESR | 电子自旋 | 0.3-100 mm | 微波 |
| **NMR** | 原子核自旋 | 0.1-1000 m | **无线电波** |

> **核心理解**：不同波长的电磁波激发分子不同能级的跃迁——UV-Vis 激发电子跃迁，IR 激发振动转动跃迁，NMR 激发核自旋跃迁。

## 三、红外吸收光谱 IR（12.2）⭐

### 3.1 基本原理

分子吸收红外光 → **振动和转动能级跃迁** → 红外吸收光谱

**用途**：剖析分子**特征官能团**。

#### （1）分子振动类型

- **伸缩振动**：对称伸缩（$\nu_s$）、反对称伸缩（$\nu_{as}$）
- **弯曲振动**：面内弯曲（剪切、卷曲）、面外弯曲（摇摆、扭曲）

#### （2）振动频率 —— Hooke 定律

$$\nu = \frac{1}{2\pi c}\sqrt{\frac{k}{\mu}}, \quad \mu = \frac{m_1 m_2}{m_1 + m_2}$$

- $k$：化学键**力常数**（N/cm）
- $\mu$：折合质量
- **键的振动频率与力常数成正比，与成键原子质量成反比**

#### （3）红外光谱产生的两个条件

1. 辐射光频率与分子振动频率**相当**（$\Delta E = h\nu$）
2. 分子振动必须引起**偶极矩变化**（极性键有吸收，非极性同核双原子分子无 IR 吸收）

> 化学键极性越强 → 振动时偶极矩变化越大 → 吸收峰越强

### 3.2 红外光谱与分子结构的关系

#### 特征吸收峰（官能团区）：**4000–1300 cm⁻¹**

- 由化学键和官能团的**伸缩振动**产生
- 用于判断化合物**是否具有某种官能团**

#### 指纹区：**1300–400 cm⁻¹**

- 吸收峰密集复杂，像人的指纹
- 用于**区别或确定具体化合物**
- 分子结构的微小差异都能在指纹区得到反映

### 3.3 常见官能团特征频率（必记）⭐⭐⭐

#### 单键伸缩振动

| 化学键 | 波数 (cm⁻¹) | 备注 |
|--------|-------------|------|
| O–H（醇、酚） | 3200–3600 | **宽峰（馒头峰）**，氢键缔合 |
| O–H（羧酸） | 2500–3600 | 更宽 |
| N–H | 3350–3500 | |
| ≡C–H（sp） | 3310–3320 | |
| =C–H（sp²） | 3000–3100 | |
| –C–H（sp³） | 2850–2950 | 饱和 C-H，**3000 为分界线** |
| C–O | 1025–1200 | |

#### 双键伸缩振动

| 化学键 | 波数 (cm⁻¹) |
|--------|-------------|
| C=C | 1620–1680 |
| **C=O** | **1700–1900**（强吸收，特征峰） |
| ┗ 醛和酮 | 1710–1750 |
| ┗ 羧酸 | 1700–1725 |
| ┗ 酸酐 | 1800–1850 & 1740–1790（双峰） |
| ┗ 酰卤 | 1770–1815 |
| ┗ 酯 | 1730–1750 |
| ┗ 酰胺 | 1680–1700 |

#### 三键伸缩振动

| 化学键 | 波数 (cm⁻¹) |
|--------|-------------|
| –C≡C– | 2100–2200 |
| –C≡N | 2240–2280 |

### 3.4 各类有机物 IR 特征

| 化合物类型 | 特征吸收 |
|------------|----------|
| **烷烃** | C–H 伸缩 2850–2960；CH₂ 弯曲 1450–1470；CH₃ 弯曲 1370–1380 |
| **烯烃** | =C–H 伸缩 3000–3100；C=C 伸缩 1620–1680；=C–H 面外弯曲可判取代类型 |
| **炔烃** | ≡C–H 伸缩 ~3315；C≡C 伸缩 ~2130 |
| **芳烃** | Ar–H 伸缩 3000–3100；C=C 骨架 1450–1600（1-4 个峰）；面外弯曲可判邻接 H 数目 |
| **醇** | O–H 伸缩 3200–3400（**馒头峰**）；C–O 伸缩 ~1200 |
| **醛酮** | C=O 伸缩 1700–1750（强）；醛基 C–H ~2717（特征） |

### 3.5 IR 解析步骤

1. 观察官能团区 → 找出官能团吸收 → **确定化合物类型**
2. 观察指纹区 → **确定基团间结合方式**
3. 结合制备过程、其他方法 → **确定可能的构造式**
4. 如果是已知化合物 → 查阅**标准谱图**验证

## 四、核磁共振谱 NMR（12.3）⭐⭐⭐

### 4.1 基本原理

> **核磁共振**：核磁矩不为零的核，在外磁场中自旋能级发生**塞曼分裂**（Zeeman Splitting），原子核吸收特定频率的射频辐射发生共振跃迁。

#### （1）自旋量子数 I

| 条件 | I | 磁性 |
|------|---|------|
| 质子数、中子数均为偶数 | 0 | 非磁性核（¹²C, ¹⁶O） |
| 质子数、中子数不同为偶数 | **≠ 0** | **磁性核**（¹H, ¹³C, ¹⁹F, ³¹P） |

> 只有 **I ≠ 0** 的核才能进行 NMR 观测。

#### （2）能级分裂与共振条件

- 外磁场中磁矩取向数 = $2I + 1$
- ¹H 的 $I = 1/2$，裂分为 2 个能级
- **共振条件**：$h\nu = \Delta E = \gamma \hbar B_0$
- **拉莫尔进动频率**：$\omega_0 = \gamma B_0$，$\nu = \frac{\gamma B_0}{2\pi}$

> 不同核磁旋比 $\gamma$ 不同，相同磁场下共振频率不同。
>
> 例：400 MHz 仪器 → ¹H 400 MHz, ¹³C 100.56 MHz

#### （3）饱和与弛豫

- **饱和**：低能态核全部跃迁到高能态，不再吸收能量
- **弛豫**：高能态核以非辐射形式回到低能态，恢复 Boltzmann 分布
  - **纵向弛豫**（自旋-晶格）：与环境交换能量
  - **横向弛豫**（自旋-自旋）：同类核之间交换能量，不影响总能态分布

#### （4）NMR 氢谱提供的三种关键信息

1. **化学位移** $\delta$ → 质子化学环境
2. **耦合常数** $J$ → 相邻质子关系
3. **积分值** → 质子数目比例

---

### 4.2 化学位移 $\delta$ ⭐

#### 定义

$$\delta = \frac{\nu_{\text{样品}} - \nu_{\text{参考}}}{\nu_{\text{参考}}} \times 10^6 \ (\text{ppm})$$

- 参考标准物：**TMS**（四甲基硅烷，$\delta = 0$）
- 为什么用 TMS？
  - 12 个 H 化学环境完全相同，单一尖峰
  - 屏蔽最强烈（$\sigma$ 最大），不与有机物峰重叠
  - 化学惰性，易溶于有机溶剂，沸点低易回收

#### 屏蔽效应与化学位移的关系

| | 电子云密度↑ | 屏蔽效应↑ | $\delta$↓ | 出峰位置 |
|------|:---:|:---:|:---:|:---:|
| **高场** | 大 | 强 | **小** | 右 |
| **低场** | 小 | 弱 | **大** | 左 |

#### 常见质子化学位移范围（必记）

| 质子类型 | $\delta$ (ppm) |
|----------|:---:|
| 烷烃 C–H（伯、仲、叔） | 0.9–1.8 |
| 与 C=C 相连的 CH₃/CH₂/CH | 1.9–2.6 |
| 与 C=O 相连的 CH₃/CH₂/CH | 2.1–2.5 |
| ≡C–H | 2.5 |
| 与卤素/OR/NR₂ 相连 | 2.7–4.1 |
| 烯烃 =C–H | 4.5–6.5 |
| 芳烃 Ar–H | 6.5–8.5 |
| 醛基 **–CHO** | **9.0–10.0** |
| 羧酸 –COOH | 10–13 |
| 醇/酚 –OH | 0.5–5.0（可变） |

> **等性质子**（化学环境相同的质子）具有大致相同的 $\delta$。

#### 影响化学位移的因素

**（1）电负性（诱导效应）**

- 邻近原子电负性↑ → 氢核周围电子云密度↓ → 屏蔽↓ → **$\delta$↑（低场移动）**
- 吸电子基团越多影响越大，距离越远影响越小

**（2）磁各向异性效应**

| 结构 | 效应 |
|------|------|
| **烯烃 C=C** | 双键平面上的质子：**去屏蔽**，出峰低场 |
| **芳烃** | 苯环平面侧面的质子：**去屏蔽**，$\delta$ 7 左右 |
| **炔烃** | 三键轴上质子：**屏蔽**，出峰较高场 |
| **C=O** | 类似双键效应 |

**（3）氢键效应**

- 形成氢键 → 去屏蔽效应 → **$\delta$ 增大（低场）**
- 例：乙醇 OH 在不同条件下 $\delta$ 为 0.5–5.0 ppm

**（4）其他因素**：范德华效应、温度、溶剂

### 4.3 自旋偶合与自旋裂分 ⭐⭐⭐

#### 基本概念

| 概念 | 定义 |
|------|------|
| **等性质子** | 化学环境相同，$\delta$ 相同 |
| **自旋偶合** | 分子中相邻**不等性质子**之间的自旋相互作用 |
| **自旋裂分** | 自旋偶合使吸收峰分裂成**多重峰**的现象 |
| **偶合常数 $J$** | 裂分峰之间的距离（Hz） |

#### 裂分规律（重点，必记）

1. **质子之间超过 3 根共价键，不产生裂分**
2. **等性质子之间不产生裂分**
3. **n+1 规则**：质子相邻碳上有 **n 个等性质子**时，裂分为 **n+1 重峰**
4. 相邻碳上有磁不等性质子时，裂分数目为各组的乘积
5. **N、O、S 等电负性大的原子上质子一般不参与偶合**（出单峰）

#### 积分曲线

- 吸收峰**面积**与等性质子**数目成正比**
- 积分曲线高度比 = 质子数目比

## 五、图谱解析示例（12.4）

### 5.1 不饱和度计算

$$\Omega = 1 + n_4 + \frac{n_3 - n_1}{2}$$

> 仅含 C、H、O 时：$\Omega = 1 + n_C + \frac{0 - n_H}{2}$

| $\Omega$ | 含义 |
|:---:|------|
| 0 | 饱和 |
| 1 | 1 个双键或 1 个脂环 |
| 2 | 三键 / 2 个双键 / 双键+脂环 |
| 4 | 苯环 |

### 5.2 解析基本思路

1. 计算**不饱和度** $\Omega$，初步判断骨架类型
2. 分析 **IR**：找出特征官能团（C=O、O–H、N–H 等）
3. 分析 **¹H NMR**：
   - 化学位移 → 质子类型
   - 积分面积 → 各类型质子数目比例
   - 裂分情况 → 相邻质子数目（n+1 规则）
4. 拼出可能结构 → 验证所有谱图数据一致性

### 5.3 经典例题

**例**：C₅H₁₀O，IR：1700 cm⁻¹ 强吸收；¹H NMR：$\delta$ 1.02（二重峰），$\delta$ 2.13（单峰），$\delta$ 2.22（七重峰）

- $\Omega = 1 + 5 - 10/2 = 1$ → 有 C=O
- IR 1700 → 羰基
- $\delta$ 2.13（3H, 单峰）→ CH₃–C=O
- $\delta$ 1.02（6H, 二重峰）→ 两个等价 CH₃–CH
- $\delta$ 2.22（1H, 七重峰）→ CH(CH₃)₂
- 结构：**(CH₃)₂CH–CO–CH₃**（3-甲基-2-丁酮）
