## 一、有机化学概述

> **参考资料**：[[有机化学1 绪论-1-合成.pdf]] | [[有机化学1 绪论-2-合成.pdf]]

### 1.1 有机化学的定义

> [!note] 核心定义
> **有机化学 (Organic Chemistry)**：研究**碳氢化合物及其衍生物**的化学。

- 1806年 **J. Berzelius** 首次提出"有机化学"概念。
- 衍生物：碳氢化合物中的氢被其他原子或基团取代后形成的化合物。

**有机化合物的元素组成：**

| 常见元素 | $\mathrm{C}$, $\mathrm{H}$, $\mathrm{N}$, $\mathrm{O}$ |
|----------|------------------------------------------------------|
| 其他元素 | $\mathrm{Si}$, $\mathrm{P}$, $\mathrm{S}$, $\mathrm{F}$, $\mathrm{Cl}$, $\mathrm{Br}$, $\mathrm{I}$ |

---

### 1.2 有机化学发展简史

| 年代 | 事件 |
|------|------|
| 1806 | Berzelius提出"有机化学"概念 |
| 1828 | **F. Wöhler** 用无机物合成了尿素 $\mathrm{NH_4CNO \rightarrow (NH_2)_2CO}$，打破了"生命力论" |
| 1845 | 合成乙酸 $\mathrm{CH_3COOH}$ |
| 1854 | 合成脂肪 |
| 现代 | 合成维生素$\mathrm{B_{12}}$、胰岛素等复杂天然产物 |

> [!important] 里程碑
> Wöhler合成尿素是有机化学发展史上的里程碑，证明有机物可以由无机物合成，打破了"生命力论"的束缚。

**诺贝尔化学奖相关：**
- 2021年：Benjamin List（有机小分子不对称催化）
- 20世纪以来：有机化学领域获得大量诺贝尔奖

## 二、有机化合物的结构理论

### 2.1 原子轨道 (Atomic Orbitals, AO)

**碳原子的电子构型：**
- $\mathrm{C}$: $1s^2 2s^2 2p^2$
- 基态：2个未成对电子（$2p_x^1 2p_y^1$），理论上只能形成2个共价键
- 实际上碳形成 **4个共价键** → 需要激发和杂化

**轨道类型：**
- **$s$轨道**：球形，1个
- **$p$轨道**：哑铃形，3个（$p_x$, $p_y$, $p_z$），互相垂直

---

### 2.2 杂化轨道理论 (Hybridization, Pauling 1931)

> **L. C. Pauling** 于1931年提出杂化轨道理论。

#### (1) $\mathrm{sp^3}$杂化

| 项目 | 内容 |
|------|------|
| **组成** | 1个$s$轨道 + 3个$p$轨道 → 4个等价的$\mathrm{sp^3}$杂化轨道 |
| **$s$成分** | 1/4；**$p$成分**：3/4 |
| **空间构型** | 正四面体 |
| **键角** | $109.5^\circ$ |
| **典型化合物** | 烷烃（如$\mathrm{CH_4}$） |

#### (2) $\mathrm{sp^2}$杂化

| 项目 | 内容 |
|------|------|
| **组成** | 1个$s$轨道 + 2个$p$轨道 → 3个$\mathrm{sp^2}$杂化轨道 + 1个未杂化的$p$轨道 |
| **$s$成分** | 1/3；**$p$成分**：2/3 |
| **空间构型** | 平面三角形 |
| **键角** | $\sim 120^\circ$ |
| **典型化合物** | 烯烃（如$\mathrm{H_2C{=}CH_2}$）、苯、羰基化合物 |

#### (3) $\mathrm{sp}$杂化

| 项目 | 内容 |
|------|------|
| **组成** | 1个$s$轨道 + 1个$p$轨道 → 2个$\mathrm{sp}$杂化轨道 + 2个未杂化的$p$轨道 |
| **$s$成分** | 1/2；**$p$成分**：1/2 |
| **空间构型** | 直线形 |
| **键角** | $180^\circ$ |
| **典型化合物** | 炔烃（如$\mathrm{HC{\equiv}CH}$） |

> [!tip] 规律总结
> $s$成分越多，键角越大，轨道离核越近，电负性越大。

---

### 2.3 化学键的类型

| 键型 | 形成方式 | 特点 |
|------|---------|------|
| $\sigma$键 | 轨道"头对头"重叠 | 电子云沿键轴对称分布，可自由旋转 |
| $\pi$键 | $p$轨道"肩并肩"重叠 | 电子云在键轴上下，不可自由旋转 |

- **单键** = 1个$\sigma$键
- **双键** = 1个$\sigma$键 + 1个$\pi$键
- **叁键** = 1个$\sigma$键 + 2个$\pi$键

---

## 三、键的参数

### 3.1 键长 (Bond Length)

- 单位：$\mathrm{nm}$（$10^{-9}\ \mathrm{m}$）或$\mathrm{\mathring{A}}$（$10^{-10}\ \mathrm{m}$）
- 一般规律：键级越高，键长越短。

| 键型 | 键长 |
|------|------|
| $\mathrm{C{-}C}$单键 | $\sim 0.154\ \mathrm{nm}$ |
| $\mathrm{C{=}C}$双键 | $\sim 0.134\ \mathrm{nm}$ |
| $\mathrm{C{\equiv}C}$叁键 | $\sim 0.120\ \mathrm{nm}$ |
| $\mathrm{C{-}H}$ | $\sim 0.110\ \mathrm{nm}$ |

### 3.2 键角 (Bond Angle)

- 取决于中心原子的**杂化方式**
- $\mathrm{sp^3}$: $\sim 109.5^\circ$ | $\mathrm{sp^2}$: $\sim 120^\circ$ | $\mathrm{sp}$: $180^\circ$

### 3.3 键能 (Bond Energy / Dissociation Energy)

- **定义**：断裂$1\ \mathrm{mol}$化学键所需的能量
- **单位**：$\mathrm{kJ/mol}$
- 键能是**平均值**，如$\mathrm{C{-}H}$键：$\frac{435+444+448+377}{4} \approx 416\ \mathrm{kJ/mol}$

| 键型 | 键能 ($\mathrm{kJ/mol}$) |
|------|--------------------------|
| $\mathrm{C{-}C}$ | $\sim 347$ |
| $\mathrm{C{=}C}$ | $\sim 610$ |
| $\mathrm{C{\equiv}C}$ | $\sim 835$ |

### 3.4 键的极性 (Bond Polarity)

- **非极性共价键**：成键原子电负性相同或相近
- **极性共价键**：成键原子电负性差较大
  - 电负性越大，对成键电子的吸引越强
  - 偶极矩 $\mu = q \times d$
- 电负性顺序：**$\mathrm{F > O > N > Cl > Br > I > S > C > H}$**

**碳卤键极性：** $\mathrm{C{-}F > C{-}Cl > C{-}Br > C{-}I}$

## 四、同分异构现象 (Isomerism)

### 4.1 基本概念

- **同分异构体 (Isomer)**：分子式相同但结构不同的化合物
- 例如：$\mathrm{C_2H_6O}$ → $\mathrm{CH_3CH_2OH}$（乙醇）或$\mathrm{CH_3OCH_3}$（甲醚）

### 4.2 分类

```
同分异构 (Isomerism)
├── 构造异构 (Constitutional Isomerism)
│   ├── 碳架异构
│   ├── 位置异构
│   ├── 官能团异构
│   └── 互变异构
└── 立体异构 (Stereoisomerism)
    ├── 构型异构
    │   ├── 顺反异构 (几何异构)
    │   └── 旋光异构 (对映异构)
    └── 构象异构
```

> [!important] 注意区别
> 构造异构 = 原子连接顺序不同；立体异构 = 原子连接顺序相同但空间排列不同。

## 五、有机化学反应

### 5.1 反应类型

| 反应类型 | 说明 |
|----------|------|
| **取代反应 (Substitution)** | 一个原子或基团被另一个取代 |
| **加成反应 (Addition)** | 在双键/叁键上加成 |
| **消除反应 (Elimination)** | 脱去小分子形成不饱和键 |
| **重排反应 (Rearrangement)** | 分子骨架重排 |
| **氧化/还原反应 (Oxidation/Reduction)** | 电子得失或氧化数变化 |

### 5.2 共价键断裂方式

| 断裂方式 | 特点 | 产物 |
|----------|------|------|
| **均裂 (Homolysis)** | 成键电子对平均分配，各得一个电子 | 自由基 (Free Radical) |
| **异裂 (Heterolysis)** | 成键电子对全部归一方 | 正离子 (Carbocation) + 负离子 (Carbanion) |

- 均裂 → 自由基反应（非极性反应）
- 异裂 → 离子反应（极性反应）

## 六、酸碱理论

### 6.1 Arrhenius酸碱理论 (1884)

- **酸**：在水溶液中电离出$\mathrm{H^+}$的物质
- **碱**：在水溶液中电离出$\mathrm{OH^-}$的物质
- 局限：仅适用于水溶液体系

$$\mathrm{CH_3COOH \rightleftharpoons CH_3COO^- + H^+}$$

### 6.2 Brønsted-Lowry酸碱理论 (1923)

- **酸**：质子给体 (Proton Donor)
- **碱**：质子受体 (Proton Acceptor)

$$\mathrm{B{:} + H{-}A \rightleftharpoons B^+{-}H + A^-}$$

**共轭酸碱对：**
- $\mathrm{CH_3CO_2H + H_2O \rightleftharpoons H_3O^+ + CH_3CO_2^-}$
- $\mathrm{CH_3CO_2H + H_2SO_4 \rightleftharpoons CH_3CO_2H_2^+ + HSO_4^-}$

**$\mathrm{p}K_\mathrm{a}$判断：**
- **同周期**：$\mathrm{CH_3{-}H \sim 49}$ > $\mathrm{H_2N{-}H\ 35}$ > $\mathrm{HO{-}H\ 15.7}$ > $\mathrm{F{-}H\ 3.8}$
  - 酸性随电负性增大而增强
- **同族**：$\mathrm{HO{-}H\ 15.7}$ < $\mathrm{HS{-}H\ 7.0}$ < $\mathrm{HSe{-}H\ 3.77}$
  - 酸性随原子半径增大而增强

> [!tip] 判断技巧
> 碱性强弱由相应共轭酸的$\mathrm{p}K_\mathrm{a}$判断：共轭酸越弱（$\mathrm{p}K_\mathrm{a}$越大），碱越强。

### 6.3 Lewis酸碱理论 (1923)

- **Lewis酸**：电子对受体 (Electron Pair Acceptor)
  - 例：$\mathrm{BF_3}$, $\mathrm{AlCl_3}$, $\mathrm{SnCl_4}$, $\mathrm{ZnCl_2}$, $\mathrm{FeCl_3}$
  - 亲电试剂：$\mathrm{Li^+}$, $\mathrm{Ag^+}$, $\mathrm{Cu^{2+}}$, $\mathrm{R^+}$, $\mathrm{Br^+}$, $\mathrm{NO_2^+}$, $\mathrm{H^+}$
- **Lewis碱**：电子对给体 (Electron Pair Donor)
  - 例：$\mathrm{NH_3}$, $\mathrm{RNH_2}$, $\mathrm{ROH}$, $\mathrm{ROR}$, $\mathrm{RSH}$
  - 亲核试剂：$\mathrm{R^-}$, $\mathrm{OH^-}$, $\mathrm{RO^-}$, $\mathrm{SH^-}$

> [!note] 知识扩展
> Lewis酸碱理论比Brønsted-Lowry理论更广泛，适用于无质子转移的反应。

## 七、电子效应

### 7.1 诱导效应 (Inductive Effect, I)

> 由于电负性不同，$\sigma$键电子向电负性较大的原子偏移。

| 类型 | 方向 | 基团 |
|------|------|------|
| **–I (吸电子)** | 远离碳链 | $\mathrm{-NR_3^+}$, $\mathrm{-NO_2}$, $\mathrm{-SO_2R}$, $\mathrm{-CN}$, $\mathrm{-COOH}$, $\mathrm{-F}$, $\mathrm{-Cl}$, $\mathrm{-Br}$, $\mathrm{-I}$, $\mathrm{-OAr}$, $\mathrm{-COOR}$, $\mathrm{-OR}$, $\mathrm{-COR}$, $\mathrm{-SH}$, $\mathrm{-OH}$ |
| **+I (给电子)** | 指向碳链 | $\mathrm{-O^-}$, $\mathrm{-COO^-}$, $\mathrm{-C(CH_3)_3}$, $\mathrm{-CH(CH_3)_2}$, $\mathrm{-CH_2CH_3}$, $\mathrm{-CH_3}$, $\mathrm{-H}$ |

- –I基团：不饱和取代基 $\mathrm{C{\equiv}CR > C_6H_5 > CH{=}CH_2 > H}$
- +I基团：$\mathrm{(CH_3)_3C{-} > (CH_3)_2CH{-} > CH_3CH_2{-} > CH_3{-} > H}$

### 7.2 场效应 (Field Effect)

- 通过空间（而非化学键）传递的静电效应
- 与诱导效应共同影响分子性质

### 7.3 共轭效应 (Conjugation Effect)

- 存在于共轭体系中（单双键交替、$p$-$\pi$共轭、$\pi$-$\pi$共轭）
- 电子离域使体系更稳定
- 例：苯环体系

### 7.4 超共轭效应 (Hyperconjugation)

- $\mathrm{C{-}H}$ $\sigma$键与相邻不饱和体系（$\pi$键或空$p$轨道）的相互作用
- **$\sigma$-$\pi$超共轭**：烷基$\mathrm{C{-}H}$ $\sigma$键与相邻$\pi$体系的相互作用
- 稳定碳正离子等活性中间体

> [!tip] 区分要点
> 诱导效应沿$\sigma$键传递，随距离减弱；共轭效应沿共轭体系传递，不受距离影响。

## 八、溶剂效应 (Solvent Effect)

### 8.1 溶剂分类

| 分类 | 特点 | 实例 |
|------|------|------|
| **质子性溶剂 (Protic Solvent)** | 含活泼$\mathrm{H}$，能形成氢键 | $\mathrm{H_2O}$, $\mathrm{ROH}$, $\mathrm{RCOOH}$ |
| **非质子性溶剂 (Aprotic Solvent)** | 不含活泼$\mathrm{H}$ | 丙酮、$\mathrm{DMF}$, $\mathrm{DMSO}$, 乙醚 |
| **极性溶剂** | 介电常数$\varepsilon > 15$ | 水、乙腈 |
| **非极性溶剂** | 介电常数小 | 己烷、苯、$\mathrm{CCl_4}$ |

### 8.2 溶剂化作用 (Solvation)

- 溶剂分子将溶质离子或分子包围，降低其化学势
- 影响反应速率和反应方向
