### 反应
1. 碱性 [[酸碱反应]]
	1. $$\mathrm{R-NH_2} + \mathrm{HX} \rightarrow \mathrm{RNH_3^+X^-}$$
	2. 如果连着EDG，使得H越不易电离，使得N的孤电子对更加偏向和体系中质子结合，则碱性更强，除了tert-Bu这种位阻大的，不易与H结合 [[立体选择性]] 所以，连接EWG碱性没有连接EDG大 eg:$R_{2}NH>RNH_{2}>R_{3}N>NH_{3}$ 
	3. 芳香胺的碱性不如脂肪胺，原因同上
2. 烃基化 [[取代]] 本质为$S_{N}$ [[控pH反应]]
	1. $$\mathrm{R-NH_2} + \mathrm{MeI} \xrightarrow[\Delta]{\mathrm{MeOH}} \mathrm{R-NMe_3^+I^-}$$
	2. $$\mathrm{Ar-NH_2} + \mathrm{R-X} \xrightarrow[\mathrm{NaHCO_3}]{} \mathrm{Ar-NHR}$$
	3. $$\mathrm{Ar-NH_2} + \mathrm{MeOH} \xrightarrow[\Delta]{\mathrm{H_2SO_4} \text{或} \mathrm{Al_2O_3}} \mathrm{Ar-NMe_2}$$
	4. $$\mathrm{Ar-NH_2} + \mathrm{Ar-OH} \xrightarrow[\mathrm{ZnCl_2}]{} \mathrm{Ar-NH-Ar}$$
3. 酰基化 [[取代]] [[控pH反应]]
	1. $$\mathrm{R-NH_2} + \mathrm{R'-CO-L} \xrightarrow[\text{碱，比如胺，碳酸钠，吡啶}]{} \mathrm{R'CO-NH-R}$$"
	2. 酰基化后可以降低氨基的致活效果，可以让邻对位只上一个，比如给苯胺用酸酐酰基化，上$Br_{2}$只能上一个
	3. 异氰酸酯的生成：$$\mathrm{R-NH_2} + \mathrm{COCl_2} \rightarrow \mathrm{R-N=C=O}$$ 异氰酸酯水解生成胺和$CO_{2}$，醇解生成R-NH-COOR'，胺解生成R-NH-CONHR'
4. 磺酰化：Hinsberg反应 [[取代]] [[人名反应]] [[控pH反应]]
	1. 伯胺：$$\mathrm{R-NH_2} + \mathrm{Ph-SO_2Cl} \xrightarrow[\mathrm{NaOH}]{} \mathrm{R-NSO_2PhNa}$$"
	2. 仲胺：生成$PhSO_{2}NR_{2}$，无法溶解在NaOH内
	3. 叔胺：不反应
5. 伯胺和亚硝酸反应，生成重氮盐 [[术语#重氮盐]] ，后续取代，芳香烃的称为Sandmeyer反应 [[人名反应]] [[取代]]
	1. 脂肪烃的反应：$$\mathrm{R-NH_2} \xrightarrow[\mathrm{HX}]{\mathrm{NaNO_2}} \mathrm{R-N \equiv NX} \rightarrow \mathrm{R^+} + \mathrm{X^-} + \mathrm{N_2} \uparrow$$" [[脱气体]]
	2. $ArNH_{2}$的反应：$$\mathrm{Ph-NH_2} \xrightarrow[\mathrm{HX}, 0^\circ\text{C}]{\mathrm{NaNO_2}} \mathrm{Ph-N \equiv NX}$$ 或硫酸：$$\mathrm{Ph-NH_2} \xrightarrow[\mathrm{H_2SO_4}, 0^\circ\text{C}]{\mathrm{NaNO_2}} \mathrm{Ph-N \equiv NHSO_4}$$"
	3. 重氮化合物：有$-N^{+}≡N$，偶氮化合物：有$-N=N-$
	4. Sandmeyer反应：
		1. 脱去所有N：$$\mathrm{Ar-N_2Cl} \xrightarrow[\mathrm{H_3PO_2}, \mathrm{H_2O}]{} \mathrm{Ar-H}$$ $$\mathrm{Ar-N_2HSO_4} \xrightarrow[\mathrm{EtOH}]{} \mathrm{Ar-H}$$
		2. 被-OH取代，由苯胺制酚：$$\mathrm{Ph-N_2HSO_4} \xrightarrow[\Delta]{\mathrm{H^+}, \mathrm{H_2O}} \mathrm{PhOH}$$ [[控pH反应]]
		3. 被卤原子取代，变成卤化苯：$$\mathrm{Ph-N_2X} \xrightarrow[\mathrm{CuX}, \mathrm{HX}]{} \mathrm{Ph-X}$$ 可以做一些直接用卤素取代因为定位效应做不了的反应，且此处适用于Cl和Br，不适用于F和I（见下）
		4. Gattermann反应 [[人名反应]]：和Sandmeyer反应类似 $$\mathrm{Ph-N_2X} \xrightarrow[\text{Cu粉}, \Delta]{} \mathrm{Ph-X}$$ [[控温]]
		5. 上I：$$\mathrm{Ph-N_2X} \ (\mathrm{X=Cl或Br}) \xrightarrow[\mathrm{KI}, \Delta]{} \mathrm{Ph-I}$$ [[控温]] 或：$$\mathrm{Ph-N_2HSO_4} \xrightarrow[\mathrm{KI}]{} \mathrm{Ph-I}$$
		6. 上F，又称作Schiemann反应 [[人名反应]]：$$\mathrm{Ph-N_2X} \xrightarrow[\mathrm{HBF_4} \text{或} \mathrm{NaBF_4}]{} \mathrm{Ph-N_2BF_4} \xrightarrow[\Delta]{} \mathrm{Ph-F} + \mathrm{N_2} \uparrow + \mathrm{BF_3} \uparrow$$ [[脱气体]]
		7. 上-CN：$$\mathrm{Ph-N_2HSO_4} \xrightarrow[\Delta]{\mathrm{CuCN}, \mathrm{KCN}} \mathrm{Ph-CN}$$ 可以进一步水解或者还原，引入-COOH或甲胺基
	5. 重氮化合物的还原 [[还原]]：$$\mathrm{Ph-N_2X} \xrightarrow[\mathrm{SnCl_2}, \mathrm{HCl}]{} \mathrm{R-NH-NH_2}$$ $\mathrm{Ph-NHNH_2}$称作苯肼，用$\mathrm{Na_2SO_3}$或$\mathrm{SO_2}$也可以：$$\mathrm{Ph-N_2HSO_4} \xrightarrow[\mathrm{Na_2SO_3}, \mathrm{H_2O}]{} \mathrm{Ph-NH-NH_2}$$
	6. 偶合反应：本质类似$S_{E}$ [[取代]] 要求Ph有EWG使得Ph带部分正电 $$\mathrm{Ar-N_2^+} + \mathrm{Ph-EWG} \ (\mathrm{EWG=OH, NH_2, NHR, NR_2}) \rightarrow \mathrm{Ar-N=N-p-Ph-EWG}$$ 两个芳香环通过偶氮基连起来
		1. 重氮盐和酚偶合，在微碱性环境
		2. 重氮盐和芳香胺偶合，在微酸性环境 [[控pH反应]]
		3. 位阻主要为p然后为o [[立体选择性]]
6. 仲胺和亚硝酸反应，生成N-亚硝基胺 [[取代]]：$$\mathrm{R_1NHR_2} \xrightarrow[\mathrm{HX}]{\mathrm{NaNO_2}} \mathrm{R_1-N(NO)-R_2}$$
7. 氧化 [[氧化]]
	1. $$\mathrm{R_1NHR_2} + \mathrm{H_2O_2} \rightarrow \mathrm{R_1-N(OH)-R_2}$$
	2. $$\mathrm{Ar-NR_1R_2} \xrightarrow[\mathrm{H_2O_2} \text{或} \mathrm{RCO_3H}]{} \mathrm{Ar-N^+O^-R_1R_2}$$"
	3. $$\mathrm{PhNH_2} \xrightarrow[\mathrm{MnO_2}, \text{dil. } \mathrm{H_2SO_4}]{} \text{苯醌}$$
8. $S_{E}$ eg:磺化反应 [[取代]] $$\mathrm{PhNH_2} \xrightarrow[\Delta]{\text{con. } \mathrm{H_2SO_4}} \mathrm{p-NH_3^+-Ph-SO_3^-}$$
9. 制备胺类
	1. 腈的还原 [[还原]]：$$\mathrm{R-CN} \xrightarrow[\mathrm{H_2/Ni} \text{或} \mathrm{LiAlH_4}]{} \mathrm{R-CH_2NH_2}$$
	2. 酰胺用$LiAlH_{4}$还原 [[还原]]
	3. 醛酮和胺反应先生成R=N-R'再催化加氢还原 [[还原]]
	4. Hofmann降解 [[人名反应]]：$$\mathrm{R-CONH_2} \xrightarrow[\mathrm{H_2O}]{\mathrm{X_2}, \mathrm{OH^-}} \mathrm{R-NH_2}$$" [[控pH反应]] 碱性水解，相当于从酰胺出发合成少一个碳的伯胺
	5. Gabriel合成 [[人名反应]]：$$\mathrm{R_1COOCOR_2} \xrightarrow[\Delta]{\mathrm{NH_3}} \mathrm{R_1CONHCOR_2} \xrightarrow[\mathrm{KOH}]{} \mathrm{R_1CONKCOR_2} \xrightarrow[\Delta]{\mathrm{R-X}} \mathrm{R_1CONRCOR_2} \xrightarrow[\Delta]{\mathrm{NaOH}, \mathrm{H_2O}} \mathrm{R-NH_2}$$ R-X不能用Ar-X代替；除了水解，还能用$N_{2}H_{4}$解，同样得到$RNH_{2}$ [[控pH反应]] [[控温]] ![[屏幕截图 2026-06-06 175944.png]]
	6. 制取芳香胺：还原 [[还原]] [[控pH反应]] [[控温]]
		1. $$\mathrm{Ph-NO_2} \xrightarrow[\text{2. } \mathrm{OH^-}]{\text{1. } \mathrm{Fe}, \mathrm{HCl}} \mathrm{Ph-NH_2}$$
		2. N的还原 [[术语#芳香化合物的缩写]]：$$\mathrm{N-NO_2} \xrightarrow[\Delta]{\mathrm{H_2}, \mathrm{Ni}, \mathrm{EtOH}} \mathrm{N-NH_2}$$
		3. 酚胺：$$\mathrm{p-HO-Ph-NO_2} \xrightarrow[\mathrm{H_2}, \mathrm{Ni}]{} \mathrm{p-HO-Ph-NH_2}$$
		4. 硝基化合物选择性还原： [[还原]]
			1. 还原硝基不还原醛酮：用$SnCl_{2}$：$$\mathrm{m-NO_2-Ph-CHO} \xrightarrow[\mathrm{SnCl_2}, \text{con. } \mathrm{HCl}]{} \mathrm{m-NH_2-Ph-CHO}$$"
			2. 用NaSH只还原一个硝基：$$\mathrm{m-NO_2-Ph-NO_2} \xrightarrow[\Delta]{\mathrm{NaSH}, \mathrm{MeOH}} \mathrm{m-NH_2-Ph-NO_2}$$
			3. 用$Na_{2}S$选择性还原：$$\mathrm{o-HO-Ph-NO_2} \xrightarrow[\Delta]{\mathrm{Na_2S}, \mathrm{NH_4Cl}} \mathrm{o-HO-Ph-NH_2}$$" 受EDG和EWG影响，EDG活化o>p，EWG活化m
10. 季铵盐
	1. 制备方法：$$\mathrm{R_3N} + \mathrm{RX} \rightarrow \mathrm{R_4N^+X^-}$$ 加热到熔点分解，反应为上述反应的逆反应 [[控温]] 常用MeI，易于反应
	2. 季铵碱：$$\mathrm{R_4N^+X^-} + \mathrm{KOH} \rightarrow \mathrm{R_4N^+OH^-} + \mathrm{KX}$$ 或用湿的$\mathrm{Ag_2O}$：$$\mathrm{R_4N^+X^-} \xrightarrow[\text{湿} \mathrm{Ag_2O}]{} \mathrm{R_4N^+OH^-} + \mathrm{AgX} \downarrow$$
	3. Hofmann消除 [[人名反应]]：季铵碱加热脱去β-H生成烯烃 [[控温]] $$\mathrm{R-CH_2N^+Me_3} \xrightarrow[\Delta]{} \mathrm{R=CH_2}$$ 规则：一般消去位阻小的β-H（酸性强的H），与Zaitsev规则相反 [[立体选择性]]，类似E1cb机理（单分子共轭碱消除）；如果β-C上有EWG，优先消除这个C的β-H，进一步验证了优先消去酸性强的β-H
	4. Hofmann和Zaitsev规则的区别：![[屏幕截图 2026-06-06 190112.png]]                 [[术语#决定因素]]
	5. Hofmann消除的立体选择性 [[立体选择性]]：消除氢与季胺碱处于反式共平面位置
	6. 所以，常用路径为：胺+MeI再+湿氧化银再加热，最终消去
