### 1.3 反应

#### (1) 碱性 [[酸碱反应]]

$$\mathrm{R{-}NH_2} + \mathrm{HX} \rightarrow \mathrm{RNH_3^+X^-}$$

> [!important] 影响因素
> - **EDG增强碱性**：使N的孤电子对更易与质子结合
> - **EWG减弱碱性**：吸电子效应使N的孤电子对不易与质子结合
> - **空间位阻**：如tert-Bu位阻大，不易与H结合 [[立体选择性]]

> [!note] 碱性顺序
> $$\mathrm{R_2NH > RNH_2 > R_3N > NH_3}$$

> 芳香胺的碱性不如脂肪胺。

#### (2) 烃基化 [[取代]]

> 本质为$S_N$反应 [[控pH反应]]

| 反应式 | 条件 |
|--------|------|
| $$\mathrm{R{-}NH_2} + \mathrm{MeI} \xrightarrow[\Delta]{\mathrm{MeOH}} \mathrm{R{-}NMe_3^+I^-}$$ | 甲基化 |
| $$\mathrm{Ar{-}NH_2} + \mathrm{R{-}X} \xrightarrow{\mathrm{NaHCO_3}} \mathrm{Ar{-}NHR}$$ | - |
| $$\mathrm{Ar{-}NH_2} + \mathrm{MeOH} \xrightarrow[\Delta]{\mathrm{H_2SO_4} \text{或} \mathrm{Al_2O_3}} \mathrm{Ar{-}NMe_2}$$ | 高温催化 |
| $$\mathrm{Ar{-}NH_2} + \mathrm{Ar{-}OH} \xrightarrow{\mathrm{ZnCl_2}} \mathrm{Ar{-}NH{-}Ar}$$ | - |

#### (3) 酰基化 [[取代]] [[控pH反应]]

$$\mathrm{R{-}NH_2} + \mathrm{R'{-}CO{-}L} \xrightarrow[\text{碱（胺、碳酸钠、吡啶）}]{} \mathrm{R'CO{-}NH{-}R}$$

> [!important] 应用
> 酰基化后可以降低氨基的致活效果，可以让邻对位只上一个取代基。例如给苯胺用酸酐酰基化，再上$\mathrm{Br_2}$只能上一个。

##### (a) 异氰酸酯的生成

$$\mathrm{R{-}NH_2} + \mathrm{COCl_2} \rightarrow \mathrm{R{-}N{=}C{=}O}$$

> - 异氰酸酯水解生成胺和$\mathrm{CO_2}$
> - 醇解生成$\mathrm{R{-}NH{-}COOR'}$
> - 胺解生成$\mathrm{R{-}NH{-}CONHR'}$

#### (4) 磺酰化：Hinsberg反应 [[取代]] [[人名反应]] [[控pH反应]]

> [!important] 鉴别伯仲叔胺
> 
> | 胺类型 | 反应产物 | 是否溶于NaOH |
> |--------|----------|--------------|
> | **伯胺** | $$\mathrm{R{-}NSO_2PhNa}$$ | 是 |
> | **仲胺** | $$\mathrm{PhSO_2NR_2}$$ | 否 |
> | **叔胺** | 不反应 | - |

#### (5) 伯胺和亚硝酸反应——重氮化反应 [[术语#重氮盐]]

> [!note] 反应类型
> 芳香烃的重氮盐取代反应称为Sandmeyer反应 [[人名反应]] [[取代]]

##### (a) 脂肪烃的反应 [[脱气体]]

$$\mathrm{R{-}NH_2} \xrightarrow[\mathrm{HX}]{\mathrm{NaNO_2}} \mathrm{R{-}N{\equiv}NX} \rightarrow \mathrm{R^+} + \mathrm{X^-} + \mathrm{N_2} \uparrow$$

##### (b) 芳香胺的反应

$$\mathrm{Ph{-}NH_2} \xrightarrow[\mathrm{HX}, 0^\circ\text{C}]{\mathrm{NaNO_2}} \mathrm{Ph{-}N{\equiv}NX}$$

或用硫酸：

$$\mathrm{Ph{-}NH_2} \xrightarrow[\mathrm{H_2SO_4}, 0^\circ\text{C}]{\mathrm{NaNO_2}} \mathrm{Ph{-}N{\equiv}NHSO_4}$$

> [!note] 重要概念
> - **重氮化合物**：有$\mathrm{-N^+{\equiv}N}$
> - **偶氮化合物**：有$\mathrm{-N=N-}$

##### (c) Sandmeyer反应及相关取代

| 反应类型 | 反应式 | 条件 |
|----------|--------|------|
| **脱氮** | $$\mathrm{Ar{-}N_2Cl} \xrightarrow[\mathrm{H_3PO_2}, \mathrm{H_2O}]{} \mathrm{Ar{-}H}$$ | - |
| **脱氮** | $$\mathrm{Ar{-}N_2HSO_4} \xrightarrow{\mathrm{EtOH}} \mathrm{Ar{-}H}$$ | - |
| **被-OH取代** | $$\mathrm{Ph{-}N_2HSO_4} \xrightarrow[\Delta]{\mathrm{H^+}, \mathrm{H_2O}} \mathrm{PhOH}$$ | [[控pH反应]] |
| **被卤原子取代（Cl/Br）** | $$\mathrm{Ph{-}N_2X} \xrightarrow[\mathrm{CuX}, \mathrm{HX}]{} \mathrm{Ph{-}X}$$ | - |
| **Gattermann反应** [[人名反应]] | $$\mathrm{Ph{-}N_2X} \xrightarrow[\text{Cu粉}, \Delta]{} \mathrm{Ph{-}X}$$ | [[控温]] |
| **被I取代** | $$\mathrm{Ph{-}N_2X} \ (\mathrm{X=Cl或Br}) \xrightarrow[\mathrm{KI}, \Delta]{} \mathrm{Ph{-}I}$$ | [[控温]] |
| **被I取代** | $$\mathrm{Ph{-}N_2HSO_4} \xrightarrow{\mathrm{KI}} \mathrm{Ph{-}I}$$ | - |
| **被F取代（Schiemann反应）** [[人名反应]] | $$\mathrm{Ph{-}N_2X} \xrightarrow[\mathrm{HBF_4} \text{或} \mathrm{NaBF_4}]{} \mathrm{Ph{-}N_2BF_4} \xrightarrow[\Delta]{} \mathrm{Ph{-}F} + \mathrm{N_2} \uparrow + \mathrm{BF_3} \uparrow$$ | [[脱气体]] |
| **被-CN取代** | $$\mathrm{Ph{-}N_2HSO_4} \xrightarrow[\Delta]{\mathrm{CuCN}, \mathrm{KCN}} \mathrm{Ph{-}CN}$$ | 可进一步水解或还原，引入-COOH或甲胺基 |

##### (d) 重氮化合物的还原 [[还原]]

$$\mathrm{Ph{-}N_2X} \xrightarrow[\mathrm{SnCl_2}, \mathrm{HCl}]{} \mathrm{R{-}NH{-}NH_2}$$

> $\mathrm{Ph{-}NHNH_2}$称作苯肼。

也可用$\mathrm{Na_2SO_3}$或$\mathrm{SO_2}$还原：

$$\mathrm{Ph{-}N_2HSO_4} \xrightarrow[\mathrm{Na_2SO_3}, \mathrm{H_2O}]{} \mathrm{Ph{-}NH{-}NH_2}$$

##### (e) 偶合反应

> [!important] 概念
> 本质类似$S_E$反应 [[取代]]，要求Ph有EWG使得Ph带部分正电。

$$\mathrm{Ar{-}N_2^+} + \mathrm{Ph{-}EWG} \ (\mathrm{EWG=OH, NH_2, NHR, NR_2}) \rightarrow \mathrm{Ar{-}N=N{-}p{-}Ph{-}EWG}$$

> [!note] 反应条件
> - 重氮盐和酚偶合：在微碱性环境
> - 重氮盐和芳香胺偶合：在微酸性环境 [[控pH反应]]
> - 位阻主要为p，其次为o [[立体选择性]]

#### (6) 仲胺和亚硝酸反应

$$\mathrm{R_1NHR_2} \xrightarrow[\mathrm{HX}]{\mathrm{NaNO_2}} \mathrm{R_1{-}N(NO){-}R_2}$$

> 生成N-亚硝基胺 [[取代]]

#### (7) 氧化反应 [[氧化]]

| 反应式 | 条件 |
|--------|------|
| $$\mathrm{R_1NHR_2} + \mathrm{H_2O_2} \rightarrow \mathrm{R_1{-}N(OH){-}R_2}$$ | - |
| $$\mathrm{Ar{-}NR_1R_2} \xrightarrow[\mathrm{H_2O_2} \text{或} \mathrm{RCO_3H}]{} \mathrm{Ar{-}N^+O^-R_1R_2}$$ | - |
| $$\mathrm{PhNH_2} \xrightarrow[\mathrm{MnO_2}, \text{dil. } \mathrm{H_2SO_4}]{} \text{苯醌}$$ | - |

#### (8) 亲电取代（磺化）[[取代]]

$$\mathrm{PhNH_2} \xrightarrow[\Delta]{\text{con. } \mathrm{H_2SO_4}} \mathrm{p{-}NH_3^+{-}Ph{-}SO_3^-}$$

| **腈的还原** [[还原]] | $$\mathrm{R{-}CN} \xrightarrow[\mathrm{H_2/Ni} \text{或} \mathrm{LiAlH_4}]{} \mathrm{R{-}CH_2NH_2}$$ | - |

### 1.5 季铵盐与季铵碱

#### (1) 季铵盐的制备 [[控温]]

$$\mathrm{R_3N} + \mathrm{RX} \rightarrow \mathrm{R_4N^+X^-}$$

> - 加热到熔点分解，反应为逆反应
> - 常用MeI，易于反应

#### (2) 季铵碱的制备

$$\mathrm{R_4N^+X^-} + \mathrm{KOH} \rightarrow \mathrm{R_4N^+OH^-} + \mathrm{KX}$$

或用湿的$\mathrm{Ag_2O}$：

$$\mathrm{R_4N^+X^-} \xrightarrow[\text{湿} \mathrm{Ag_2O}]{} \mathrm{R_4N^+OH^-} + \mathrm{AgX} \downarrow$$

#### (3) Hofmann消除 [[人名反应]] [[控温]]

$$\mathrm{R{-}CH_2N^+Me_3} \xrightarrow[\Delta]{} \mathrm{R{=}CH_2}$$

> [!important] Hofmann规则
> 一般消去位阻小的$\beta$-H（酸性强的$\mathrm{H}$），与Zaitsev规则相反 [[立体选择性]]。

> [!note] 机理
> 类似E1cb机理（单分子共轭碱消除）。如果$\beta$-C上有EWG，优先消除这个$\mathrm{C}$的$\beta$-H。

> [!note] 立体选择性
> 消除氢与季胺碱处于反式共平面位置。

> Hofmann和Zaitsev规则的区别见[[屏幕截图 2026-06-06 190112.png]] [[术语#决定因素]]

> [!tip] 常用路径
> 胺 + MeI → 季铵盐 + 湿氧化银 → 季铵碱 + 加热 → 消除生成烯烃

## 知识检查

> [!question] AI 认为需要修正的内容：
> 1. "碱性顺序：$\mathrm{R_2NH > RNH_2 > R_3N > NH_3}$" → 在水溶液中，脂肪胺的碱性顺序受溶剂化效应和空间位阻共同影响，通常为：$\mathrm{R_2NH > RNH_2 > R_3N > NH_3}$（对甲基胺），但对乙基胺等可能略有不同。
> 2. "重氮盐和酚偶合，在微碱性环境" → 更准确地说，是在弱碱性条件下（pH 8-10），此时酚以酚氧负离子形式存在，亲核性更强。
> 3. "重氮盐和芳香胺偶合，在微酸性环境" → 在弱酸性条件下（pH 5-7），此时胺以游离碱形式存在，亲核性较强，同时重氮盐稳定性较好。
> 4. "Hofmann消除机理类似E1cb" → 更准确地说，Hofmann消除通常是E2机理，但由于季铵基的强吸电子效应和较大空间位阻，过渡态具有部分E1cb特征（反式共平面消除）。
