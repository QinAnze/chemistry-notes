## 醛酮
### 命名 [[命名]]
1. 简单醛：同醇的命名，芳香醛把-Ar看作取代基，比如苯甲醛
2. 简单酮：XX酮，两个X是酮羰基左右的基团，如果一个是-Ar一个是-R，则命名为酰基苯
3. 系统命名：选主链，从靠近羰基一侧开始编号，写取代基，标羰基位置，比如4-甲基戊-2-酮，醛>酮>醇>卤素>硝基>不饱和烃，环上的如果羰基碳在环内则为环X酮，如果在环外，把换作为取代基

### 反应
1. 羰基的亲核加成 [[加成]]
	1. 原理：$$\mathrm{-CO-} + \mathrm{Nu} \xrightarrow[\text{水解}]{} \mathrm{-C(OH)(Nu)-}$$
	2. 与格氏试剂$$\mathrm{R_1-CO-R_2} + \mathrm{R-MgX} \xrightarrow[\mathrm{H_2O}]{\mathrm{H^+}} \mathrm{R_1-C(OH)(R)-R_2}$$ [[控pH反应]] 立体性：酮的赤式和苏式遵循Cram规则 [[立体选择性]] [[术语#赤式和苏式 立体化学]]
	3. Cram规则 [[人名反应]]：与格氏试剂加成，羰基处于M和S两个基团中间，Nu从S基团一侧进攻羰基碳（位阻小）[[立体选择性]] [[术语#L/M/S/H 立体化学]]
	4. 活性顺序：醛>酮，-R>-Ar，原理：EWG>EDG，还有空间位阻影响，比如脂肪酮>芳香酮
	5. Li试剂$$\mathrm{R_1-CO-R_2} + \mathrm{R-Li} \xrightarrow[\mathrm{H_2O}]{\mathrm{H^+}} \mathrm{R_1-C(OH)(R)-R_2}$$ [[控pH反应]] 如果格氏试剂位阻太大，可以用Li试剂代替格氏试剂，也需要水解才能变成醇
	6. 炔基钠$$\mathrm{R_1-CO-R_2} \xrightarrow[\text{水解}]{\mathrm{R-C \equiv CNa}} \mathrm{R_1-(OH)(C \equiv C-R)-R_2}$$ 可以制备炔醇，后续同样要酸性水解 [[控pH反应]]
	7. Reformasky反应 [[人名反应]] 见羧酸衍生物 [[羧酸及其衍生物]] $$\mathrm{R_1COR_2} + \mathrm{Br-CH_2-COOR} \xrightarrow[\text{2. H}_2\text{O, H}^+]{\text{1. Zn}} \mathrm{R_1(OH)(R_2)CH_2COOR}$$ [[控pH反应]] 特点：RZnBr活性不如RMgBr，不会和酯基反应，后续羟基可以在酸性条件下消除，生成$\alpha,\beta-不饱和酸$
	8. 与HCN反应：生成$\alpha-羟基腈$，后续如果酸性水解生成$\alpha-羟基酸$，如果浓硫酸则消除为$\alpha,\beta-不饱和羧酸$ [[控pH反应]]
	9. 与$\mathrm{NaHSO_3}$反应$$\mathrm{R_1-CO-R_2} + \mathrm{NaHSO_3} \rightarrow \mathrm{R_1-(OH)(SO_3Na)-R_2} \downarrow$$ 为白色沉淀，且反应可逆，在酸碱环境下$\Delta$均可以还原回去，所以可以用来提纯
	10. 与水反应：生成偕二醇$$\mathrm{R_1-CO-R_2} + \mathrm{H_2O} \rightarrow \mathrm{R_1-C(OH)_2-R_2}$$ 如果羰基和EWG连接，则可形成稳定的偕二醇，否则会脱水重新生成醛酮
	11. 与醇的加成$$\mathrm{R_1-CO-R_2} + \mathrm{ROH} \xrightarrow[\mathrm{H^+}(\text{无水酸})]{} \mathrm{R_1-C(OR)_2-R_2}$$ [[控pH反应]] 最终生成缩醛酮，注意酸一定是无水酸，比如氯化氢，$\text{con. H}_2\text{SO}_4$，TsOH等 [[术语#Ts]] 在稀酸溶液中，会重新分解为原来的醛酮
	12. 醇加成的例子：末端醇酮的自身加成成环状半缩醛酮
	13. 与二醇的用处：成环状缩酮保护醛酮，见下图![[屏幕截图 2026-05-30 122303.png]]
	14. 与氨和衍生物（胺类，肼类）的反应$$\mathrm{R_1-CO-R_2} + \mathrm{H_2N-R} \xrightarrow[\mathrm{H^+}]{} \mathrm{R_1R_2C=N-R}$$ 用弱酸催化（eg:$\mathrm{NH_4Cl}$），用强酸会和N结合，失去Nu [[控pH反应]] 机理：![[屏幕截图 2026-05-30 124928.png]]

2. $\alpha-H反应$
	1. 酸性：被碱拔H，生成$\mathrm{R-CO-CH^-R}$并且能烯醇互变，如果连接EWG则$\alpha-H酸性增强$，所以$\mathrm{R-CO-CH_2-CO-R'}$这类结构的都有活泼的$\alpha-H$，比如各种羧酸衍生物含有二羰基结构的类型
	2. 卤化反应 $$\mathrm{Ph-CO-CH_3} \xrightarrow[\text{乙醚，低温}]{\mathrm{Br_2, AlCl_3}} \mathrm{Ph-CO-CH_2Br}$$ 机理：![[屏幕截图 2026-05-30 195913.png]]               酸催化下可以控制在一卤代阶段，因为电子云密度增强，使得O的电子云密度下降 [[控pH反应]] [[控温]]
	3. 卤仿反应 $$\mathrm{R-CO-CH_3} \xrightarrow[\text{碱}]{\mathrm{X_2}} \mathrm{R-CO-CX_3} \xrightarrow[\text{碱}]{} \mathrm{R-COO^-} + \mathrm{HCX_3}$$ 碱催化下，进攻位阻小的$\alpha-H$，且难以控制在一卤代物，而是完全变成$\mathrm{CX_n}$ [[立体选择性]] [[控pH反应]] $\mathrm{CHX_3}$为卤仿，比如用$\mathrm{I_2}$和NaOH反应，生成碘仿亮黄色沉淀，要注意的是，卤素有氧化性，所以有$\mathrm{CH_3C(OH)-}$时会氧化成$\mathrm{CH_3CO-}$，也可进行卤仿反应，卤仿反应可用于减短碳链
3. 羟醛缩合 [[加成]] 又称为Aldol缩合 [[人名反应]]
	1. 反应式 $$\mathrm{R_1-CHO} + \mathrm{H-R_2-CHO} \xrightarrow[\text{碱或酸}]{} \mathrm{R_1-CH(OH)-R_2-CHO} \xrightarrow[\Delta]{} \mathrm{R_1-CH=R_2-CHO}$$ 在稀碱或稀酸的催化下，含有α-氢原子的醛或酮结合在一起，生成 β- 羟基醛或酮的反应，所以反应称为羟醛缩合反应 [[控pH反应]] [[控温]]
	2. 机理：![[屏幕截图 2026-05-30 201921.png]]
	3. 特点：一个是醛，后续变成C(OH)，一个是有$\alpha-H$的醛酮，后续脱去$\alpha-H$，保留羰基，后续羟基脱水也会变成不饱和双键
	4. 如果用甲醛，那可以引入$\mathrm{-CH_2OH}$
	5. Claisen-Schmidt缩合 [[人名反应]]：芳醛与含 α-氢的醛或酮在碱性条件下反应脱水生成 α,β-不饱和醛或酮 $$\mathrm{Ph-CHO} + \mathrm{CH_3CHO} \xrightarrow[\text{碱}]{} \mathrm{Ph-CH=CH-CHO}$$ [[控pH反应]] 同Aldol缩合
	6. Perkin反应 [[人名反应]]：芳醛与脂肪族酸酐在其碱金属盐催化下加热缩合 $$\mathrm{Ph-CHO} + \mathrm{(CH_3CO)_2O} \xrightarrow[\text{碱, }\Delta]{} \mathrm{Ph-CH=CHCOOH}$$ [[控pH反应]] [[控温]] 原理同Aldol缩合，特点：酸酐必须有 α-H，羧酸盐与之对应，且不适用于脂肪醛，其易发生自身缩合反应
	7. Mannich反应 [[人名反应]]：含有$\alpha$-活泼氢的化合物（如酮）与醛及胺之间发生的三组分缩合反应， 使用甲醛时可在酮的α位引入一个胺甲基，称为胺甲基化反应，原理是Aldol缩合加上胺取代羟基 $$\mathrm{R_1COCH_2R_2} + \mathrm{HCHO} + \mathrm{HNR(R')} \xrightarrow[\mathrm{H^+}]{} \mathrm{R_1COCH(R_2-CH_2-NR(R'))}$$ [[控pH反应]]
4. 氧化 [[氧化]]
	1. 温和氧化剂：Tollen试剂，Fehling试剂 [[人名反应]]
		1. Tollen试剂：$\mathrm{[Ag(NH_3)_2]OH}$，即银氨溶液，氧化醛基
		2. Fehling试剂：$\mathrm{CuSO_4}$，即铜氨溶液，仅氧化脂肪醛
	2. $\mathrm{Ag_2O}$：不氧化双键，氧化醛基
	3. 强氧化剂：$\mathrm{KMnO_4}$和$\mathrm{K_2Cr_2O_7}$等
		1. 醛基在-Ar上，条件不能过于剧烈
		2. 实例：$$\mathrm{R-CHO} \xrightarrow[\mathrm{H_2SO_4}]{\mathrm{KMnO_4}} \mathrm{R-COOH}$$ $$\mathrm{Ph-CHO} \xrightarrow[\text{CrO}_3, \mathrm{H^+} \text{或冷稀KMnO}_4]{} \mathrm{Ph-COOH}$$ [[控温]] [[控pH反应]]
		3. 氧化酮用强氧化剂，包括$\text{con. HNO}_3$ $$\mathrm{R_1-CH_2-CO-CH_2-R_2} \xrightarrow[\text{强氧化剂}]{} \mathrm{R_1-COOH} + \mathrm{R_2CH_2-COOH}$$ 如果氧化环酮，可以制取二元羧酸
		4. Baeyer-Villiger氧化重排 [[人名反应]]
			1. 酮被过氧酸氧化成酯 $$\mathrm{R-CO-R'} \xrightarrow[\mathrm{R''CO_3H}]{} \mathrm{R-COO-R'}$$
			2. 过氧酸可用$\mathrm{CH_3CO_3H}, \mathrm{CF_3CO_3H}, \mathrm{PhCO_3H}$
			3. 醛发生此反应生成羧酸
			4. R'为迁移基团，会在羰基和R'中间插个O
			5. 连接基团多的先迁移，H>3'C>环己基>2'C>-Bn>-Ph>1'C>-Me（注意几个芳基的位置）
			6. 迁移基团手性不变 [[立体选择性]]
5. 还原 [[还原]]
	1. 催化加氢：生成醇，同时还原碳碳双键和三键
	2. $\mathrm{LiAlH_4}$和$\mathrm{NaBH_4}$再水解，生成醇，但是不还原碳碳双键三键，还原性$\mathrm{LiAlH_4}>\mathrm{NaBH_4}$，前者还可以还原羧酸及其衍生物和$\mathrm{-NO_2}$，后者只能还原醛酮
	3. Meerwein-Ponndorf还原 [[人名反应]]：$$\mathrm{R_1-CO-R_2} + \mathrm{CH_3CH(OH)CH_3} \xrightarrow[((\mathrm{CH_3})_2\mathrm{CHO})_3\mathrm{Al}]{} \mathrm{R_1-CH(OH)-R_2} + \mathrm{CH_3COCH_3}$$ 异丙醇作为还原剂，$((\mathrm{CH_3})_2\mathrm{CHO})_3\mathrm{Al}$起到催化剂(路易斯酸催化)的作用，相当于把酮羰基和异丙醇的羟基互换，逆反应为Oppenauer氧化 [[人名反应]] 异丙醇过量则平衡右移，走M-P还原，如果丙酮过量，则走Oppenauer氧化
	4. 金属的酮双分子还原：
		1. 反应：$$\mathrm{2R_1-CO-R_2} \xrightarrow[\text{2. 水解}]{\text{1. Mg，苯}} \mathrm{R_1R_2C(OH)-C(OH)R_1R_2}$$ 相当于两分子酮羰基还原成羟基，并且两个羰基碳成键
		2. 还原剂：钠、镁、铝或镁汞齐、铝汞齐等
		3. 非质子性溶剂中反应，再水解后产物：邻二叔醇（频哪醇）
	5. Clemmensen还原 [[人名反应]]：$$\mathrm{R_1-CO-R_2} \xrightarrow[\text{con. HCl}]{\text{Zn-Hg}} \mathrm{R_1-CH_2-R_2}$$ 把醛酮还原成了烷基
	6. Wolff-Kishner-Huang M.L.还原法 [[人名反应]] [[脱气体]]
		1. W-K还原：$$\mathrm{R_1-CO-R_2} \xrightarrow[\text{强碱}]{\mathrm{N_2H_4}} \mathrm{R_1-CH_2-R_2} + \mathrm{N_2}$$ [[控pH反应]]
		2. 黄鸣龙还原：$$\mathrm{R_1-CO-R_2} \xrightarrow[\text{乙二醇或三甘醇, }\Delta]{\mathrm{N_2H_4·H_2O, 碱}} \mathrm{R_1-CH_2-R_2} + \mathrm{N_2}$$ 优点：安全
	7. Cannizzaro反应 [[人名反应]]
		1. 原理同双分子还原，不过是碱催化下的反应
		2. 不过产物太杂，所以常用的是无α-H的醛+甲醛的：$$\mathrm{R-CHO} \xrightarrow[\text{con. NaOH}]{\mathrm{HCHO}} \mathrm{R-CH_2OH} + \mathrm{HCOONa}$$ [[控pH反应]] 原理是两个醛基，一个被还原成醇，一个被氧化成羧酸
