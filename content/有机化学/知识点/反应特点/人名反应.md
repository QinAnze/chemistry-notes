# 常见有机化学人名反应

## 1. 取代反应

| 反应名称 | 核心用途 |
|---------|---------|
| **傅-克烷基化反应 (Friedel-Crafts Alkylation)** | 在芳香环上引入烷基 |
| **傅-克酰基化反应 (Friedel-Crafts Acylation)** | 在芳香环上引入酰基，是合成芳酮的关键方法 |
| **卤代反应 (Halogenation)** | 在芳香环或烷烃上引入卤素原子 |
| **硝化反应 (Nitration)** | 向有机化合物中引入硝基 (-NO₂) |
| **磺化反应 (Sulfonation)** | 向有机化合物中引入磺酸基 (-SO₃H) |

## 2. 加成反应

| 反应名称 | 核心用途 |
|---------|---------|
| **马尔科夫尼科夫加成 (Markovnikov Addition)** | 不对称烯烃与极性试剂加成时，氢加在含氢较多的碳上 |
| **反马尔科夫尼科夫加成 (Anti-Markovnikov Addition)** | 在过氧化物存在下，溴化氢与烯烃加成时氢加在含氢较少的碳上 |
| **硼氢化-氧化反应 (Hydroboration-Oxidation)** | 烯烃经硼烷加成后氧化，得到反马氏规则的醇 |
| **催化氢化 (Catalytic Hydrogenation)** | 在催化剂作用下将不饱和键还原为饱和键 |
| **狄尔斯-阿尔德反应 (Diels-Alder Reaction)** | 共轭二烯与亲双烯体发生 [4+2] 环加成，构建六元环 |

## 3. 消除反应

| 反应名称 | 核心用途 |
|---------|---------|
| **E1 消除反应** | 单分子消除，生成取代较多的烯烃（查依采夫产物） |
| **E2 消除反应** | 双分子协同消除，遵循查依采夫规则 |
| **霍夫曼消除 (Hofmann Elimination)** | 季铵碱热分解，主要生成取代较少的烯烃（霍夫曼产物） |

## 4. 氧化还原反应

| 反应名称 | 核心用途 |
|---------|---------|
| **琼斯氧化 (Jones Oxidation)** | 用 CrO₃/H₂SO₄ 将伯醇氧化为羧酸、仲醇氧化为酮 |
| **斯文氧化 (Swern Oxidation)** | 用 DMSO/(COCl)₂/Et₃N 温和地将醇氧化为醛或酮 |
| **戴斯-马丁氧化 (Dess-Martin Oxidation)** | 用高价碘试剂将醇氧化为醛或酮，条件温和 |
| **PCC 氧化 (Pyridinium Chlorochromate)** | 将伯醇氧化为醛而不进一步氧化为羧酸 |
| **沃尔夫-凯惜纳还原 (Wolff-Kishner Reduction)** | 在强碱性条件下将羰基还原为亚甲基 |
| **克莱门森还原 (Clemmensen Reduction)** | 用 Zn-Hg/HCl 将羰基还原为亚甲基 |
| **硼氢化钠还原 (NaBH₄ Reduction)** | 选择性还原醛、酮为醇，不还原酯和酰胺 |
| **氢化铝锂还原 (LiAlH₄ Reduction)** | 强还原剂，可还原醛、酮、酯、羧酸、酰胺为醇或胺 |
| **催化氢化 (Catalytic Hydrogenation)** | 用 H₂/Pd 或 H₂/Pt 还原烯烃、炔烃、硝基等 |
| **伯奇还原 (Birch Reduction)** | 用碱金属/液氨将芳香环部分还原为 1,4-环己二烯 |

## 5. 重排反应

| 反应名称 | 核心用途 |
|---------|---------|
| **瓦格纳-梅尔外因重排 (Wagner-Meerwein Rearrangement)** | 碳正离子中间体发生烷基迁移，形成更稳定的碳正离子 |
| **频哪醇重排 (Pinacol Rearrangement)** | 邻二醇在酸催化下重排为酮或醛 |
| **贝克曼重排 (Beckmann Rearrangement)** | 肟在酸催化下重排为酰胺 |
| **霍夫曼重排 (Hofmann Rearrangement)** | 酰胺与溴/碱反应，失去一个碳原子生成少一个碳的胺 |
| **柯提斯重排 (Curtius Rearrangement)** | 酰基叠氮热分解，失去 CO₂ 生成少一个碳的胺 |
| **施密特反应 (Schmidt Reaction)** | 羧酸或酮与叠氮酸反应，分别生成胺或酰胺 |
| **克莱森重排 (Claisen Rearrangement)** | 烯丙基芳基醚或烯丙基乙烯基醚发生 [3,3]-σ 迁移 |
| **科普重排 (Cope Rearrangement)** | 1,5-二烯发生 [3,3]-σ 迁移 |

## 6. 缩合反应

| 反应名称 | 核心用途 |
|---------|---------|
| **羟醛缩合 (Aldol Condensation)** | 两分子醛或酮缩合生成 β-羟基羰基化合物或 α,β-不饱和羰基化合物 |
| **克莱森缩合 (Claisen Condensation)** | 两分子酯缩合生成 β-酮酯 |
| **狄克曼缩合 (Dieckmann Condensation)** | 分子内克莱森缩合，合成五元或六元环 β-酮酯 |
| **克脑文盖尔缩合 (Knoevenagel Condensation)** | 醛或酮与活泼亚甲基化合物缩合生成 α,β-不饱和化合物 |
| **迈克尔加成 (Michael Addition)** | 亲核试剂对 α,β-不饱和羰基化合物的 1,4-共轭加成 |
| **曼尼希反应 (Mannich Reaction)** | 醛、胺和含活泼氢化合物三组分缩合生成 β-氨基羰基化合物 |
| **罗宾逊成环 (Robinson Annulation)** | 迈克尔加成后接分子内羟醛缩合，构建稠环体系 |
| **珀金反应 (Perkin Reaction)** | 芳香醛与酸酐缩合生成 α,β-不饱和芳香酸 |
| **维蒂希反应 (Wittig Reaction)** | 磷叶立德与醛或酮反应，将羰基转化为烯烃 |
| **霍纳尔-沃兹沃思-埃蒙斯反应 (HWE Reaction)** | 用磷酸酯稳定的碳负离子与醛酮反应生成烯烃，E 选择性高 |

## 7. 人名命名的偶联反应

| 反应名称 | 核心用途 |
|---------|---------|
| **武兹反应 (Wurtz Reaction)** | 两分子卤代烷与钠反应生成对称烷烃 |
| **武兹-菲蒂希反应 (Wurtz-Fittig Reaction)** | 卤代烷与卤代芳烃偶联生成烷基芳烃 |
| **乌尔曼反应 (Ullmann Reaction)** | 铜催化的芳基卤代物偶联生成联芳烃 |
| **格氏反应 (Grignard Reaction)** | 格氏试剂与羰基化合物反应生成醇 |
| **铃木偶联 (Suzuki Coupling)** | 钯催化的硼酸与卤代芳烃交叉偶联 |
| **根岸偶联 (Negishi Coupling)** | 钯催化的有机锌试剂与卤代物交叉偶联 |
| **斯蒂尔偶联 (Stille Coupling)** | 钯催化的有机锡试剂与卤代物交叉偶联 |
| **赫克反应 (Heck Reaction)** | 钯催化的烯烃与卤代芳烃偶联生成取代烯烃 |
| **薗头偶联 (Sonogashira Coupling)** | 钯/铜共催化的末端炔烃与卤代芳烃偶联 |
| **布赫瓦尔德-哈特维希偶联 (Buchwald-Hartwig Coupling)** | 钯催化的芳基卤代物与胺偶联生成芳胺 |

## 8. 碳正离子/碳负离子相关反应

| 反应名称 | 核心用途 |
|---------|---------|
| **SN1 反应 (单分子亲核取代)** | 通过碳正离子中间体进行亲核取代 |
| **SN2 反应 (双分子亲核取代)** | 协同的亲核取代，构型翻转 |
| **格氏试剂制备 (Grignard Reagent Formation)** | 卤代烷与镁反应制备强亲核试剂 |
| **有机锂试剂制备** | 卤代烷与锂反应制备超强亲核试剂 |

## 9. 芳香亲电/亲核取代

| 反应名称 | 核心用途 |
|---------|---------|
| **亲电芳香取代 (EAS)** | 亲电试剂进攻富电子芳香环 |
| **亲核芳香取代 (SNAr)** | 强吸电子基活化的芳环发生亲核取代 |
| **本炔机理 (Benzyne Mechanism)** | 强碱条件下通过苯炔中间体进行亲核取代 |

## 10. 其他重要人名反应

| 反应名称 | 核心用途 |
|---------|---------|
| **傅瑞斯重排 (Fries Rearrangement)** | 酚酯在 Lewis 酸催化下重排为邻/对位酰基酚 |
| **瑞默-蒂曼反应 (Reimer-Tiemann Reaction)** | 酚与氯仿/碱反应生成邻位甲酰基酚 |
| **科尔贝电解 (Kolbe Electrolysis)** | 羧酸盐电解脱羧偶联生成烷烃 |
| **科尔贝-施密特反应 (Kolbe-Schmitt Reaction)** | 酚钠与 CO₂ 反应生成水杨酸 |
| **威廉姆森醚合成 (Williamson Ether Synthesis)** | 卤代烷与醇钠反应合成醚 |
| **加布里埃尔合成 (Gabriel Synthesis)** | 用邻苯二甲酰亚胺合成伯胺 |
| **霍夫曼降解 (Hofmann Degradation)** | 酰胺降解为少一个碳的胺 |
| **夏普莱斯不对称环氧化 (Sharpless Epoxidation)** | 用手性钛催化剂对烯丙醇进行对映选择性环氧化 |
| **夏普莱斯不对称双羟基化 (Sharpless Dihydroxylation)** | 用手性配体对烯烃进行对映选择性双羟基化 |
| **野依不对称氢化 (Noyori Asymmetric Hydrogenation)** | 用手性钌催化剂进行对映选择性氢化 |
