# 有机化学中放出气体的反应

## 1. 生成 CO₂ 的反应
- **羧酸与碳酸氢钠/碳酸钠**：`R-COOH + NaHCO₃ → R-COONa + H₂O + CO₂↑`
- **脱羧反应**：`R-COOH → R-H + CO₂↑`（加热，或β-酮酸/丙二酸型易脱羧）

## 2. 生成 N₂ 的反应
- **重氮盐分解**：`Ar-N₂⁺Cl⁻ → Ar-Cl + N₂↑`（Sandmeyer反应等）
- **偶氮化合物分解**
- **胺与亚硝酸**：`R-NH₂ + HNO₂ → R-OH + N₂↑ + H₂O`（脂肪伯胺）

## 3. 生成 SO₂ 的反应
- **磺酰氯分解/反应**：`R-SO₂Cl` 相关反应
- **亚硫酸盐与酸**：无机，但有机物磺化逆反应可涉及

## 4. 生成 HX（卤化氢）的反应
- **卤代烃消除**：`R-CH₂-CHX-R' → R-CH=CH-R' + HX`（NaOH/醇，加热）
- **醇与HX取代**：`R-OH + HX → R-X + H₂O`（可逆，浓酸条件下）
- **酰卤生成**：`R-COOH + SOCl₂ → R-COCl + SO₂↑ + HCl↑`

## 5. 生成 NH₃ 的反应
- **酰胺水解**：`R-CONH₂ + H₂O → R-COOH + NH₃↑`（强酸/强碱，加热）
- **胺的霍夫曼消除**：`R-NH₂` 相关（但主要生成烯烃和胺）

## 6. 生成 H₂O（脱水）的反应
- **醇分子内脱水**：`R-CH₂-CH₂-OH → R-CH=CH₂ + H₂O`（浓H₂SO₄/Al₂O₃，加热）
- **醇分子间脱水**：`2 R-OH → R-O-R + H₂O`（醚的生成）
- **羧酸与醇酯化**：`R-COOH + R'-OH ⇌ R-COOR' + H₂O`

## 7. 其他
- **过氧化物分解**：`R-O-O-R → 2 R-O·`（自由基，可能伴随O₂）
- **臭氧分解**：烯烃 `R₂C=CR₂ + O₃ →` 分解产物（含O₂）

---
*记忆要点：脱羧出CO₂、消除出HX、脱水成烯/醚、重氮出N₂、酰胺水解出NH₃*
