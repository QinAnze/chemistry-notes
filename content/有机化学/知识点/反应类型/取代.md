# 取代反应 (Substitution Reaction)

## 定义

取代反应是指**有机化合物分子中的某个原子或原子团被其他原子或原子团所替代的反应**。

一般通式：
$$R-L + Nu^- \rightarrow R-Nu + L^-$$

其中：
- $R$ = 有机基团
- $L$ = 离去基团 (Leaving Group)
- $Nu^-$ = 亲核试剂 (Nucleophile)

---

## 分类

取代反应根据反应机理可分为两大类：

### 1. 亲核取代反应 (Nucleophilic Substitution, $S_N$)

亲核试剂进攻带正电或部分正电的碳原子，取代离去基团。

#### 1.1 $S_N1$ 反应（单分子亲核取代）

**机理特点：**
- 反应分两步进行
- 第一步：离去基团离去，形成碳正离子中间体（决速步）
- 第二步：亲核试剂进攻碳正离子
- 反应速率只与底物浓度有关：$v = k[RL]$

**特征：**
- 碳正离子中间体 → 可能发生重排
- 产物外消旋化（平面碳正离子被两面进攻）
- 三级卤代烷 > 二级 > 一级 > 甲基
- 极性质子溶剂有利

#### 1.2 $S_N2$ 反应（双分子亲核取代）

**机理特点：**
- 一步协同反应，无中间体
- 亲核试剂从离去基团的背面进攻
- 反应速率与底物和亲核试剂浓度都有关：$v = k[RL][Nu^-]$

**特征：**
- 构型翻转（Walden 翻转）
- 空间位阻影响大
- 甲基 > 一级 > 二级 > 三级
- 极性非质子溶剂有利

| 特征 | $S_N1$ | $S_N2$ |
|------|--------|--------|
| 动力学 | 一级 | 二级 |
| 中间体 | 碳正离子 | 无（过渡态） |
| 立体化学 | 外消旋化 | 构型翻转 |
| 底物活性 | 3° > 2° > 1° > CH₃ | CH₃ > 1° > 2° > 3° |
| 重排 | 可能发生 | 不发生 |
| 溶剂 | 质子溶剂有利 | 非质子溶剂有利 |

---

### 2. 亲电取代反应 (Electrophilic Substitution, $S_E$)

亲电试剂进攻富电子体系（如芳香环），取代氢原子。

#### 2.1 芳香亲电取代反应

**反应通式：**
$$Ar-H + E^+ \rightarrow Ar-E + H^+$$

**常见类型：**

| 反应名称 | 亲电试剂 ($E^+$) | 试剂 | 产物 |
|---------|-----------------|------|------|
| **卤代反应** | $X^+$ (X = Cl, Br) | $X_2/FeX_3$ | 卤代芳烃 |
| **硝化反应** | $NO_2^+$ | 浓 $HNO_3$/浓 $H_2SO_4$ | 硝基化合物 |
| **磺化反应** | $SO_3$ | 浓 $H_2SO_4$ | 芳基磺酸 |
| **傅-克烷基化** | $R^+$ | $RX/AlCl_3$ | 烷基苯 |
| **傅-克酰基化** | $RCO^+$ | $RCOCl/AlCl_3$ | 芳基酮 |

**反应机理（以卤代为例）：**
1. 亲电试剂生成：$Br_2 + FeBr_3 \rightarrow Br^+ + FeBr_4^-$
2. 亲电进攻：$Br^+$ 进攻苯环 π 电子云，形成 σ 络合物（Wheland 中间体）
3. 去质子恢复芳香性：失去 $H^+$，$FeBr_4^-$ 作为碱接受质子

**定位效应：**
- **邻对位定位基**（活化）：$-OH, -NH_2, -OR, -R, -X$
- **间位定位基**（钝化）：$-NO_2, -CN, -COOH, -CHO, -COR, -SO_3H$
- **卤素**特殊：钝化但邻对位定位

---

### 3. 自由基取代反应 (Free Radical Substitution)

通过自由基中间体进行的取代反应。

**典型反应：烷烃的卤代**

$$CH_4 + Cl_2 \xrightarrow{h\nu} CH_3Cl + HCl$$

**反应机理（链式反应）：**

1. **链引发**：$Cl_2 \xrightarrow{h\nu} 2Cl·$
2. **链增长**：
   - $Cl· + CH_4 \rightarrow HCl + ·CH_3$
   - $·CH_3 + Cl_2 \rightarrow CH_3Cl + Cl·$
3. **链终止**：自由基相互结合

**选择性：**
- 溴代选择性 > 氯代选择性
- 反应活性：$3°H > 2°H > 1°H$

---

## 常见取代反应实例

### 醇的取代

```
R-OH + HX → R-X + H₂O          (SN1或SN2)
R-OH + SOCl₂ → R-Cl + SO₂ + HCl  (SN2，构型翻转)
R-OH + PBr₃ → R-Br + H₃PO₃      (SN2)
```

### 卤代烃的水解

```
R-X + H₂O → R-OH + HX           (SN1或SN2)
R-X + NaOH → R-OH + NaX         (SN2为主)
```

### 威廉姆森醚合成

```
R-X + R'-ONa → R-O-R' + NaX     (SN2)
```

### 胺的制备（Gabriel合成）

```
邻苯二甲酰亚胺钾 + R-X → N-烷基邻苯二甲酰亚胺 → R-NH₂
```

---

## 总结

| 反应类型 | 进攻试剂 | 被进攻对象 | 典型底物 |
|---------|---------|-----------|---------|
| 亲核取代 ($S_N$) | 亲核试剂 ($Nu^-$) | 缺电子碳 | 卤代烃、醇衍生物 |
| 亲电取代 ($S_E$) | 亲电试剂 ($E^+$) | 富电子体系 | 芳香化合物 |
| 自由基取代 | 自由基 ($R·$) | 各类C-H键 | 烷烃 |
