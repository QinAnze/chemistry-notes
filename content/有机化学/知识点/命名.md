# 有机化合物命名规则（2017 IUPAC 新版）

## 一、基本命名原则

### 1. 母体选择（Parent Hydride）
- **最长碳链原则**：选择含最多取代基的最长连续碳链作为主链
- **优先官能团**：按优先顺序确定母体后缀（suffix）
- **最低位次原则（Lowest Set of Locants）**：编号使取代基位次之和最小
- **多重母体规则**：当主链同时含官能团与不饱和键时，母体名应同时体现二者，如 hex-2-en-1-ol

> [!important] 母体选择四步法
> ① 选定主官能团（决定后缀）→ ② 选含主官能团的最长碳链（可为非直链）→ ③ 使主官能团位次最低进行编号 → ④ 其余基团作前缀按字母顺序排列。

### 2. 编号规则
- 从靠近优先官能团的一端开始编号
- 若两端距离相同，按字母顺序使先出现的取代基编号较小
- 新规则明确使用 **1,1,2,3** 而非 **1,2,2,3** 的编号方式（即优先使第一个不同位次最小）
- 对于烯炔共存体系：使双键、三键位次均最低；若仍相同，双键位次优先

> [!example] 编号对比
> 2-甲基-3-己烯 正确写法为 **hex-3-ene**（位次 3）；若误从另一端编号，会得到 hex-4-ene，违反最低位次原则。

### 3. 2017 与传统命名的衔接
- 2017 IUPAC 全面推行 **取代基命名法（Substitutive Nomenclature）**
- 中文化学命名长期沿用 "官能团类别法 + 取代基法" 混合体系
- 中国化学会 2022 版《有机化学命名原则》已与 2013/2017 IUPAC 对接，但在俗名、冠首母体上保留中文习惯

## 二、官能团优先顺序（从高到低）

| 优先级 | 官能团 | 后缀 | 作为前缀 |
|--------|--------|------|----------|
| 1 | 羧酸 (-COOH) | -oic acid | carboxy- |
| 2 | 磺酸 (-SO₃H) | -sulfonic acid | sulfo- |
| 3 | 酯 (-COOR) | -oate | (alkoxycarbonyl-) |
| 4 | 酰卤 (-COX) | -oyl halide | halocarbonyl- |
| 5 | 酰胺 (-CONH₂) | -amide | carbamoyl- |
| 6 | 腈 (-CN) | -nitrile | cyano- |
| 7 | 醛 (-CHO) | -al | formyl- / oxo- |
| 8 | 酮 (>C=O) | -one | oxo- |
| 9 | 醇 (-OH) | -ol | hydroxy- |
| 10 | 酚 (-OH 连芳环) | -ol | hydroxy- |
| 11 | 硫醇 (-SH) | -thiol | mercapto- / sulfanyl- |
| 12 | 胺 (-NH₂) | -amine | amino- |
| 13 | 醚 (-O-) | -ether（或烷氧基作前缀） | (alkoxy-) |
| 14 | 烯烃 (C=C) | -ene | — |
| 15 | 炔烃 (C≡C) | -yne | — |

> [!note] 中文常用前缀对照
> 羧基、磺酸基、酯基（烷氧羰基）、卤甲酰基、氨甲酰基、氰基、醛基（甲酰基）、氧代（羰基）、羟基、巯基、氨基、烷氧基。

> [!warning] 易错：主链含官能团优先
> 主链必须包含最高优先级官能团，即使此链并非最长。例：含 -COOH 的化合物，即使有一条更长的纯烃链，主链也必须选含 -COOH 的链。

### 常见烷基前缀

| 中文 | 英文 | 缩写 | 结构 |
|------|------|------|------|
| 甲基 | methyl- | Me | -CH₃ |
| 乙基 | ethyl- | Et | -CH₂CH₃ |
| 丙基 | propyl- | Pr | -CH₂CH₂CH₃ |
| 异丙基 | isopropyl- / propan-2-yl- | i-Pr | -CH(CH₃)₂ |
| 丁基 | butyl- | Bu | -CH₂CH₂CH₂CH₃ |
| 异丁基 | isobutyl- / 2-methylpropyl- | i-Bu | -CH₂CH(CH₃)₂ |
| 仲丁基 | sec-butyl- / butan-2-yl- | s-Bu | -CH(CH₃)CH₂CH₃ |
| 叔丁基 | tert-butyl- / 1,1-dimethylethyl- | t-Bu | -C(CH₃)₃ |
| 新戊基 | neopentyl- / 2,2-dimethylpropyl- | — | -CH₂C(CH₃)₃ |
| 苯基 | phenyl- | Ph | -C₆H₅ |
| 苄基（苯甲基） | benzyl- | Bn | -CH₂C₆H₅ |
| 烯丙基 | allyl- / prop-2-en-1-yl | — | -CH₂CH=CH₂ |
| 乙酰基 | acetyl- / ethanoyl | Ac | -COCH₃ |

> [!warning] 易错：1-甲基乙基 vs 异丙基
> "异丙基" 是保留俗名前缀，2017 系统命名为 **propan-2-yl-**；"1-甲基乙基" 是其展开式描述，二者均指同一基团 $-CH(CH_3)_2$。考试中视要求选用，系统命名优先用 propan-2-yl-。

### 含杂原子前缀
- 卤素：fluoro-, chloro-, bromo-, iodo-（中文：氟、氯、溴、碘代）
- 硝基：nitro- $-NO_2$
- 亚硝基：nitroso- $-NO$
- 氨基：amino- $-NH_2$
- 羟基：hydroxy-（当醇不作为母体时）
- 羰基/氧代：oxo-（>C=O 作取代基时）
- 醛基/甲酰基：formyl-（-CHO 作侧链时）
- 羧基：carboxy-（-COOH 作侧链时）
- 酯基（烷氧羰基）：alkoxycarbonyl- $-COOR$
- 氰基：cyano- $-CN$

## 四、脂环与芳烃命名

### 1. 脂环烃（Alicyclic Hydrocarbons）
- 单环：冠以 cyclo-，如 cyclohexane（环己烷）、cyclohexene（环己烯）
- 编号从靠近官能团或不饱和键的碳开始，使不饱和键与取代基位次最低
- 环作为取代基时称 cycloalkyl-（环烷基），如 cyclohexylmethanol

### 2. 芳烃（Aromatic Hydrocarbons）
- 保留俗名母体：benzene（苯）、toluene（甲苯）、xylene（二甲苯）、cumene（异丙苯）
- 二取代苯相对位置：
  - 邻位 ortho- (o-)，1,2-
  - 间位 meta- (m-)，1,3-
  - 对位 para- (p-)，1,4-
- 三取代苯相对位置（中文习称）：

| 中文 | 英文 | 位次关系 |
|------|------|----------|
| 连 | vicinal (vic-) | 1,2,3- |
| 均 | symmetrical (sym-) | 1,3,5- |
| 偏 | asymmetrical (asym-) | 1,2,4- |

> [!tip] 取代基作主链判断
> 当苯环上连有含碳主链时：烷基 $\leq 6$ 碳常以苯为母体（如乙苯 ethylbenzene）；烷基 $>6$ 碳或含官能团时以链为母体、苯作苯基前缀（如 2-苯基辛烷 2-phenyloctane）。

### 3. 稠环芳烃（Fused Aromatic Rings）
- 保留俗名：naphthalene（萘）、anthracene（蒽）、phenanthrene（菲）
- 编号按特定环路规则：萘 1–8 沿两环外缘编号，9、10 为稠合位
- 稠环命名通式：[a,b] 前缀 + 母体，如 benzo[a]pyrene（苯并[a]芘）

### 4. 杂环（Heterocycles）
- 含杂原子环按 Hantzsch-Widman 体系命名：如 oxirane（环氧乙烷）、aziridine（氮丙啶）
- 保留俗名：pyridine（吡啶）、furan（呋喃）、thiophene（噻吩）、pyrrole（吡咯）、imidazole（咪唑）
- 编号自杂原子起，使其余杂原子位次最低；多杂原子按 O>S>N 优先

## 五、桥环与螺环编号

### 1. 桥环烃（Bridged Bicyclic）
- 命名格式：bicyclo[a.b.c.d]alkane，$a \geq b \geq c$ 为三条桥碳数（不含桥头碳）
- 编号：从第一桥头碳出发，沿最长桥→次长桥→最短桥回到第二桥头碳
- 例：bicyclo[2.2.1]heptane（降冰片烷，norbornane），桥头碳为 C1、C4

> [!example] 桥环编号示范
> bicyclo[2.2.1]heptane：两桥头碳记 1、4；最长桥（2 碳）2,3；次长桥（2 碳）5,6；最短桥（1 碳）7。

### 2. 螺环烃（Spiro Compounds）
- 命名格式：spiro[a.b]alkane，$a \leq b$ 为两环各除螺原子外的碳数
- 编号：从小环紧邻螺原子的碳起步，绕小环→螺原子→大环
- 例：spiro[2.4]heptane（螺[2.4]庚烷）
- 多螺原子：dispiro、trispiro 依次标注

## 六、立体化学命名

### 1. R/S 构型（Cahn-Ingold-Prelog 规则）
- 按原子序数排序基团优先级（同序数时逐层向外比较）
- 最低优先级基团远离观察者
- 顺时针为 R（rectus），逆时针为 S（sinister）
- 多手性中心：逐个标注，如 (2R,3S)-2,3-dihydroxybutanedioic acid

> [!important] CIP 优先级判定要点
> ① 比较直接相连原子序数；② 相同则比较次级原子集合（按序数降序排列后逐位比较）；③ 重键按重复原子处理（如 C=O 视为连两个 O）；④ 同位素按质量数大者优先（D > H）。

### 2. E/Z 构型（烯烃）
- 优先基团在同侧为 Z（zusammen，同）
- 优先基团在异侧为 E（entgegen，反）
- 仅当双键两端碳各有两个不同基团时才需 E/Z 标注

### 3. 顺反 cis/trans 与顺式/反式
- **cis/trans**：用于环系或两端各有相同基团的双键，同侧为 cis，异侧为 trans
- **顺式/反式**：中文通用，环上两取代基同侧为顺式，异侧为反式
- 严格场合应优先用 E/Z 与 R/S，cis/trans 仅在双键两端有相同基团时明确无歧义

### 4. 其他立体描述符
- **D/L**：糖类、氨基酸（基于甘油醛构型，D/L 与 R/S 不直接对应）
- **α/β**：环上取代基相对于参考平面的位置（如糖的端基异构、甾族化合物）
- **endo/exo**：桥环中取代基相对于最长桥的位置
- **syn/anti**：相对于参考平面的同侧/异侧

### 5. 旋光异构（Optical Isomerism）
- 手性碳（不对称碳）：连四个不同基团
- 对映异构体：互为镜像，(+)/(−) 表示旋光方向
- 内消旋体（meso）：含手性中心但分子整体对称，无旋光性
- 外消旋体（racemic mixture, rac-）：等量对映体混合

> [!summary] 立体描述符速查
> 绝对构型：R/S、D/L；相对构型：E/Z、cis/trans、顺式/反式、syn/anti、endo/exo、α/β；旋光方向：(+)、(−)、(±)；混合状态：rac-、meso-。

## 八、2017 新版主要变化

### 1. 命名方式统一
- **取代基命名法（Substitutive Nomenclature）** 成为唯一推荐方式
- 废除旧的 **官能团类别命名法（Functional Class Nomenclature）**
  - 例：旧版 "ethyl methyl ether" → 新版 "methoxyethane"
  - 例：旧版 "acetone" → 新版 "propan-2-one"

### 2. 位次编号规范
- 位次数字与名称之间用 **连字符（hyphen）** 连接
- 多个位次之间用 **逗号** 分隔
- 示例：1,2-dichloroethane，hexan-2-ol

### 3. 多官能团化合物
- 使用 **multiplicative nomenclature** 处理对称结构
- 示例：1,3-dichloropropane-2-ol

### 4. 环系命名
- 保留俗名的环系减少
- 推荐系统命名：cyclohexane, bicyclo[2.2.1]heptane 等

### 5. 中国化学会命名与 IUPAC 对照

| 项目 | IUPAC（2013/2017） | 中国化学会（2022） |
|------|--------------------|--------------------|
| 醚 | methoxyethane | 甲氧基乙烷 |
| 酮 | propan-2-one | 2-丙酮（保留"丙酮"） |
| 位次位置 | hexan-2-ol | 2-己醇（位次前置） |
| 取代基排序 | 按字母顺序 | 按复杂程度/笔画习惯 |
| 醛 | ethanal | 乙醛 |
| 双键位次 | but-2-ene | 2-丁烯 |

> [!note] 中英命名差异要点
> 中文习惯将官能团位次置于母体名之前（如 2-己醇而非己-2-醇）；保留俗名更普遍；取代基顺序中文常按简单→复杂排列，而非字母序。

## 九、常见化合物命名示例

| 结构 | 2017 IUPAC 名称 | 中文名称 |
|------|----------------|----------|
| CH₃CH₂OH | ethanol | 乙醇 |
| CH₃CH(OH)CH₃ | propan-2-ol | 2-丙醇/异丙醇 |
| CH₃COCH₃ | propan-2-one | 丙酮 |
| CH₃CH₂COOH | propanoic acid | 丙酸 |
| CH₃CH=CHCH₃ | but-2-ene | 2-丁烯 |
| CH₃C≡CH | prop-1-yne | 丙炔 |
| C₆H₅OH | phenol | 苯酚 |
| CH₃OCH₂CH₃ | methoxyethane | 甲氧基乙烷 |
| HOOCCH₂CH(NH₂)COOH | aspartic acid | 天冬氨酸 |
| BrCH₂CH₂Br | 1,2-dibromoethane | 1,2-二溴乙烷 |
| (CH₃)₃CCOOH | 2,2-dimethylpropanoic acid | 2,2-二甲基丙酸 |

### 多官能团命名实例分析

> [!example] 多官能团实例
> 结构 $HOCH_2CH_2COCH_3$：含 -OH 与 C=O，酮优先级高于醇，故母体为 butan-2-one，羟基作前缀，命名为 **4-hydroxybutan-2-one**（4-羟基-2-丁酮）。

> [!warning] 烯炔编号易错
> 含双键与三键的链编号时，先使两者位次组合最低；若双键、三键位次组合相同（如 1,3 与 1,3），则双键优先取低位次。例：5-氯戊-1-烯-4-炔（5-chloropent-1-en-4-yne），不可写成戊-4-烯-1-炔。
