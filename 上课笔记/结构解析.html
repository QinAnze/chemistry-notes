1.3 四谱联用基本思路

有机物结构表征的完整流程通常把四大波谱配合使用，相互印证：

四谱分工

  • UV：判断是否含共轭体系、芳香环（提供 π 电子信息）
  • IR：鉴定官能团（C=O、O–H、N–H、C≡N 等）
  • NMR：确定氢/碳的化学环境、数目与连接关系（最核心）
  • MS：测定分子量与分子式，提供碎片信息（辅助推断骨架）
步骤谱图获得信息
1MS分子量 → 推测分子式 → 计算 Ω
2IR官能团种类（C=O、O–H、N–H、C≡C 等）
3UV共轭体系、芳香性
4¹H/¹³C NMR氢碳数目、化学环境、连接方式
5综合拼接结构 → 验证所有谱图一致性

解析顺序口诀

“先分子式（MS）看骨架，再 IR 找官能团，UV 辨共轭，NMR 定连接，最后综合验证。”

二、电磁波与分子的吸收光谱（12.1）

2.1 电磁波的基本性质

  • 光具有波粒二象性
  • 波动性：ν=c/λ，σ=1/λ（波数，cm⁻¹）
  • 粒子性：E=hν=hc/λ=hcσ
  • 波长愈短，能量愈大

2.2 电磁波类型与波谱分析方法

波谱方法激发能级波长范围电磁波类型
X 射线光谱内层电子0.01-10 nmX 射线
UV-Visn 及 π 电子200-800 nm紫外-可见光
IR振动与转动0.8-300 μm红外线
ESR电子自旋0.3-100 mm微波
NMR原子核自旋0.1-1000 m无线电波

核心理解：不同波长的电磁波激发分子不同能级的跃迁——UV-Vis 激发电子跃迁，IR 激发振动转动跃迁，NMR 激发核自旋跃迁。

2.3 吸收光谱的产生条件

分子吸收特定波长电磁波后，从基态跃迁到激发态，产生吸收光谱。产生吸收需满足：

  1. 辐射能量等于两能级差：ΔE=hν
  2. 跃迁符合选律（如振动需引起偶极矩变化；电子跃迁需轨道重叠）

吸收强度由 Lambert-Beer 定律描述：

A=logII0​​=εcl

  • A：吸光度；ε：摩尔吸光系数（L·mol⁻¹·cm⁻¹）；c：浓度；l：光程
  • ε>104 为强吸收，ε<102 为弱吸收

三、紫外-可见吸收光谱 UV-Vis（12.2）

3.1 基本原理

分子吸收紫外-可见光后，价电子发生能级跃迁，产生紫外吸收光谱。常用波长范围 200–800 nm，纵坐标为吸光度 A 或摩尔吸光系数 ε。

（1）电子跃迁类型

跃迁类型能量波长范围特点
σ→σ∗高<150 nm（远紫外）饱和烃，远紫外区不可测
n→σ∗较高150–250 nm含 O/N/X/S 的饱和化合物
π→π∗中200–800 nm烯、炔、醛酮、芳烃（需共轭才进可见区）
n→π∗低250–600 nm含 C=O、C=N、N=O 等，弱吸收（ε 10–100）

关键跃迁

  • π→π∗：共轭烯烃特征，强吸收（ε>104）
  • n→π∗：羰基等杂原子双键，弱吸收但特征

（2）生色团与助色团

概念定义例
生色团（chromophore）产生 π→π∗ 或 n→π∗ 吸收的基团C=C、C≡C、C=O、NO₂、N=N、苯环
助色团（auxochrome）含孤对电子的饱和基团，与生色团相连使吸收红移并增色–OH、–OR、–NH₂、–X、–SH

3.2 影响吸收带的因素

（1）共轭效应——红移与增色

红移（bathochromic shift，红移）

共轭体系延长 → 最高占有分子轨道（HOMO）与最低未占轨道（LUMO）能级差减小 → 吸收波长变长（红移），吸收强度增大（增色效应）。

化合物λmax​ (nm)ε
乙烯17115 000
1,3-丁二烯21721 000
β-胡萝卜素（11 个共轭双键）452152 000

（2） Woodward 规则（概念）

Woodward-Fieser 规则用于估算共轭二烯及烯酮的 λmax​：以母体基础值加上各取代基的增量。

Woodward 规则要点

  • 同环二烯基础值 253 nm，异环二烯 214 nm
  • 每增加一个共轭双键 +30 nm
  • 烷基取代 +5 nm，环外双键 +5 nm，–OR +6 nm，–SR +30 nm，–Cl +5 nm
  • 仅适用于共轭体系的 π→π* 吸收，不适用于芳香族

（3）溶剂效应

  • 极性溶剂使 n→π∗ 蓝移（基态更稳定）
  • 极性溶剂使 π→π∗ 红移（激发态更稳定）

3.3 芳香族紫外吸收

苯的紫外光谱有三个特征吸收带：

吸收带λmax​ (nm)ε来源
E₁ 带~18447 000π→π*（远紫外）
E₂ 带~2047 400π→π*
B 带（精细结构）256230π→π*（振动精细结构，特征）

B 带

B 带是苯及其衍生物的特征吸收带，伴有振动精细结构。取代基使 B 带红移，可用于鉴别芳香化合物。

3.4 UV 在生物大分子中的应用

生物工程联系：紫外吸收法测生物分子浓度

  • 蛋白质：芳香族氨基酸 Trp/Tyr/Phe 在 280 nm 附近有吸收，是测定蛋白质浓度的常用方法（A280​）
  • 核酸：嘌呤/嘧啶碱基共轭体系在 260 nm 有强吸收（A260​）
  • 纯度判断：纯 DNA 的 A260​/A280​≈1.8，纯 RNA ≈2.0；蛋白质污染使比值下降
  • 胡萝卜素类：长共轭体系吸收可见光，呈现颜色

四、红外吸收光谱 IR（12.3）⭐

4.1 基本原理

分子吸收红外光 → 振动和转动能级跃迁 → 红外吸收光谱

用途：剖析分子特征官能团。

（1）分子振动类型

  • 伸缩振动（stretching）：对称伸缩（νs​）、反对称伸缩（νas​）
  • 弯曲振动（bending）：面内弯曲（剪切、卷曲）、面外弯曲（摇摆、扭曲）

（2）振动频率 —— Hooke 定律

ν=2πc1​μk​​,μ=m1​+m2​m1​m2​​

  • k：化学键力常数（N/cm）
  • μ：折合质量
  • 键的振动频率与力常数成正比，与成键原子质量成反比

三大规律

  1. 键越强（k 大）→ 频率越高（如三键 > 双键 > 单键）
  2. 原子越轻 → 频率越高（X–H 键在最高频 3000 cm⁻¹ 附近）
  3. 共轭使双键力常数下降 → 吸收红移

（3）红外光谱产生的两个条件

  1. 辐射光频率与分子振动频率相当（ΔE=hν）
  2. 分子振动必须引起偶极矩变化（极性键有吸收，非极性同核双原子分子无 IR 吸收）

化学键极性越强 → 振动时偶极矩变化越大 → 吸收峰越强

4.2 红外光谱与分子结构的关系

特征吸收峰（官能团区）：4000–1300 cm⁻¹

  • 由化学键和官能团的伸缩振动产生
  • 用于判断化合物是否具有某种官能团

指纹区：1300–400 cm⁻¹

  • 吸收峰密集复杂，像人的指纹
  • 用于区别或确定具体化合物
  • 分子结构的微小差异都能在指纹区得到反映

官能团区 vs 指纹区

  • 官能团区（4000–1300 cm⁻¹）：定性判断官能团种类
  • 指纹区（1300–400 cm⁻¹）：与标准谱图比对，确证具体化合物

  • 对位双取代：860–800 cm⁻¹

4.5 氢键对 IR 的影响

氢键的影响

形成氢键 → X–H 键力常数 k 减小 → 吸收峰向低频（低波数）移动并变宽变强。

  • 游离 O–H（气相/稀溶液）：~3600 cm⁻¹ 尖峰
  • 缔合 O–H（液态/浓溶液）：3200–3400 cm⁻¹ 宽峰
  • 羧酸二聚体：O–H 移至 2500–3300 cm⁻¹ 极宽

4.6 酰胺 I / II 带——生物工程联系

蛋白质和多肽的 IR 中，酰胺键（–CO–NH–）有两类特征吸收：

吸收带波数 (cm⁻¹)归属
酰胺 I 带1600–1700（典型 1650）C=O 伸缩（为主）+ N–H 弯曲
酰胺 II 带1480–1575（典型 1540）N–H 弯曲 + C–N 伸缩
酰胺 III 带1200–1400C–N 伸缩 + N–H 弯曲

应用

酰胺 I 带的精细位置可反映蛋白质二级结构：

  • α-螺旋：~1650–1658 cm⁻¹
  • β-折叠：~1620–1640 cm⁻¹
  • 无规卷曲：~1640–1648 cm⁻¹ 用 FTIR（傅里叶变换红外）分析酰胺 I 带是研究蛋白质构象的常用手段。

4.7 IR 解析步骤

  1. 观察官能团区 → 找出官能团吸收 → 确定化合物类型
  2. 观察指纹区 → 确定基团间结合方式
  3. 结合制备过程、其他方法 → 确定可能的构造式
  4. 如果是已知化合物 → 查阅标准谱图验证

五、核磁共振谱 NMR（12.4）⭐⭐⭐

5.1 基本原理

核磁共振（Nuclear Magnetic Resonance）：核磁矩不为零的核，在外磁场中自旋能级发生塞曼分裂（Zeeman Splitting），原子核吸收特定频率的射频辐射发生共振跃迁。

（1）自旋量子数 I

条件I磁性
质子数、中子数均为偶数0非磁性核（¹²C, ¹⁶O）
质子数、中子数不同为偶数≠ 0磁性核（¹H, ¹³C, ¹⁹F, ³¹P）

只有 I ≠ 0 的核才能进行 NMR 观测。有机化学中最常用的是 ¹H NMR（氢谱） 和 ¹³C NMR（碳谱）。

（2）能级分裂与共振条件

  • 外磁场中磁矩取向数 = 2I+1
  • ¹H 的 I=1/2，裂分为 2 个能级
  • 共振条件：hν=ΔE=γℏB0​
  • 拉莫尔进动频率：ω0​=γB0​，ν=2πγB0​​

不同核磁旋比 γ 不同，相同磁场下共振频率不同。

例：400 MHz 仪器 → ¹H 400 MHz, ¹³C 100.56 MHz

（3）饱和与弛豫

  • 饱和：低能态核全部跃迁到高能态，不再吸收能量
  • 弛豫：高能态核以非辐射形式回到低能态，恢复 Boltzmann 分布
    • 纵向弛豫（自旋-晶格，T1​）：与环境交换能量
    • 横向弛豫（自旋-自旋，T2​）：同类核之间交换能量，不影响总能态分布

（4）NMR 氢谱提供的三种关键信息

  1. 化学位移 δ → 质子化学环境
  2. 耦合常数 J → 相邻质子关系
  3. 积分值 → 质子数目比例

5.2 化学位移 δ ⭐

定义

δ=ν参考​ν样品​−ν参考​​×106 (ppm)

  • 参考标准物：TMS（四甲基硅烷，δ=0）
  • 为什么用 TMS？
    • 12 个 H 化学环境完全相同，单一尖峰
    • 屏蔽最强烈（σ 最大），不与有机物峰重叠
    • 化学惰性，易溶于有机溶剂，沸点低易回收

电子云密度↑屏蔽效应↑δ↓出峰位置
高场大强小右
低场小弱大左

等性质子（化学环境相同的质子）具有大致相同的 δ。

影响化学位移的因素

（1）电负性（诱导效应）

  • 邻近原子电负性↑ → 氢核周围电子云密度↓ → 屏蔽↓ → δ↑（低场移动）
  • 吸电子基团越多影响越大，距离越远影响越小

例：CH₃–F（δ 4.3）> CH₃–OH（δ 3.4）> CH₃–NH₂（δ 2.2）> CH₄（δ 0.2）

（2）磁各向异性效应

结构效应
烯烃 C=C双键平面上的质子：去屏蔽，出峰低场
芳烃苯环平面侧面的质子：去屏蔽（环电流效应），δ 7 左右
炔烃三键轴上质子：屏蔽，出峰较高场 (δ ~2.5)
C=O类似双键效应，醛基质子位于去屏蔽区

（3）氢键效应

  • 形成氢键 → 去屏蔽效应 → δ 增大（低场）
  • 例：乙醇 OH 在不同条件下 δ 为 0.5–5.0 ppm

（4）其他因素：范德华效应、温度、溶剂、浓度

5.3 自旋偶合与自旋裂分 ⭐⭐⭐

基本概念

概念定义
等性质子化学环境相同，δ 相同
自旋偶合分子中相邻不等性质子之间的自旋相互作用
自旋裂分自旋偶合使吸收峰分裂成多重峰的现象
偶合常数 J裂分峰之间的距离（Hz），与磁场强度无关

裂分规律（重点，必记）

  1. 质子之间超过 3 根共价键，不产生裂分
  2. 等性质子之间不产生裂分
  3. n+1 规则：质子相邻碳上有 n 个等性质子时，裂分为 n+1 重峰
  4. 相邻碳上有磁不等性质子时，裂分数目为各组的乘积
  5. N、O、S 等电负性大的原子上质子一般不参与偶合（出单峰）

常见多重峰

n峰数名称峰高比（二项式）
01单峰(s)1
12二重峰(d)1:1
23三重峰(t)1:2:1
34四重峰(q)1:3:3:1
67七重峰1:6:15:20:15:6:1

偶合常数 J

  • 一般范围 0–20 Hz
  • 同碳偶合 2J、邻碳偶合 3J（最常见，6–8 Hz）、远程偶合（>3 键，小）
  • 顺式烯氢 J≈10 Hz，反式 J≈17 Hz（可判断烯烃构型）

积分曲线

  • 吸收峰面积与等性质子数目成正比
  • 积分曲线高度比 = 质子数目比

5.4 ¹³C NMR 与 DEPT ⭐

¹³C NMR 特点

  • ¹³C 天然丰度仅 1.1%，灵敏度低，需脉冲傅里叶变换（PFT）累加
  • 化学位移范围宽（0–220 ppm），几乎每个碳一个峰，重叠少
  • 一般不作偶合裂分（¹³C–¹H 偶合常数大，通常去偶谱为单峰）
  • 不积分（信号强度不严格与碳数成正比）

C–O / C–N / C–X50–90

羰基 C=O160–220
┗ 酮/醛190–220
┗ 羧酸/酯/酰胺160–180

DEPT 谱（无畸变极化转移增强）

DEPT（Distortionless Enhancement by Polarization Transfer）通过改变脉冲角度区分碳的类型：

DEPT 实验CH₃CH₂CH季碳
DEPT-45+++无
DEPT-90无无+无
DEPT-135+−+无

DEPT 用法

  • DEPT-135 中：CH₃、CH 向上，CH₂ 向下，季碳不出峰
  • 结合常规 ¹³C 谱（含全部碳）即可确定各类碳数目

5.6 NMR 在生物分子与代谢组学中的应用

生物工程联系

  • 代谢组学：¹H NMR 是代谢物指纹分析的标准方法（尿液、血清、组织提取物），一锅检测数百种代谢物
  • 蛋白质结构：同位素标记（¹⁵N/¹³C）后用多维 NMR 解析溶液中蛋白质三维结构（<40 kDa）
  • 核酸结构：通过 NOE 约束推断 DNA/RNA 折叠
  • 水分与食品分析：固体 NMR 测定生物大分子流动相态

六、质谱 MS（12.5）⭐

6.1 基本原理

质谱（Mass Spectrometry）：样品分子在离子源中被电离成气相离子，按质荷比 m/z 分离并检测，得到质谱图。

  • 横坐标：质荷比 m/z
  • 纵坐标：相对丰度（以最强峰为基峰，定为 100%）
  • 质谱不属于吸收光谱，是破坏性分析
进样 → 离子源（电离） → 质量分析器（按 m/z 分离） → 检测器 → 质谱图

6.2 分子离子峰与同位素峰

（1）分子离子峰 M+⋅

  • 分子失去一个电子形成的离子峰
  • 其 m/z 等于分子量（用最高质量端确定）
  • 分子离子峰越稳定，越易观察到（芳香环、共轭体系稳定）

M+ 的判断

  • M+⋅ 必须是谱图中最高质量端的合理峰（除同位素外）
  • 符合氮规则：含偶数 N（含 0 个）的分子，M 为偶数；含奇数 N，M 为奇数

（2）同位素峰

常见同位素丰度：

同位素对丰度比
¹²C / ¹³C100 : 1.1（每个碳）
¹H / ²H100 : 0.015
¹⁶O / ¹⁸O100 : 0.2
³⁵Cl / ³⁷Cl3 : 1（M : M+2 特征双峰）
⁷⁹Br / ⁸¹Br1 : 1（M : M+2 等高双峰）

同位素推断分子式

  • M+2 峰强异常 → 含 S、Cl、Br
  • Cl 含 1 个：M:M+2 ≈ 3:1；Br 含 1 个：M:M+2 ≈ 1:1
  • (M+1)/M 比值 → 估算碳原子数：nC​≈(M+1)/M÷1.1%

6.3 碎片裂解规律

分子离子获得多余能量后沿化学键断裂产生碎片离子。常见裂解方式：

裂解方式描述特征
α-裂解羰基/烯丙基旁的 σ 键断裂醛酮、酯的特征碎片
McLafferty 重排γ-H 通过六元环过渡迁移到不饱和键，并断裂含 γ-H 的羰基化合物，特征偶数质量碎片
烯丙位裂解烯丙基稳定正离子烯烃特征
苄基裂解形成苄基正离子（m/z 91）烷基苯特征
σ-断裂链状分子单键断裂饱和烃

解析口诀

  • 烷基苯必有 m/z 91（草鎓离子 C7​H7+​）
  • 醛酮常有 McLafferty 重排偶数质量峰
  • 高质量端碎片往往保留分子主要骨架信息

6.4 高分辨质谱 HRMS 测分子式

HRMS 测分子式

高分辨质谱（如 FT-ICR、Orbitrap、TOF）可精确测定 m/z 至小数点后 4–6 位，根据精确分子量直接确定分子式（不同分子式精确质量略有差异）。

  • 精确质量：¹²C = 12.0000, ¹H = 1.00783, ¹⁶O = 15.9949, ¹⁴N = 14.0031
  • 例：CO（27.9949） vs N₂（28.0061）vs C₂H₄（28.0313）——整数质量都是 28，精确质量可区分

6.5 电离方式

电离方式适用特点
EI（电子轰击）小分子，挥发性样品标准 70 eV，碎片丰富，便于谱库检索；但分子离子峰常弱
CI（化学电离）小分子软电离，[M+H]⁺ 明显，碎片少
FAB极性大分子快原子轰击
MALDI（基质辅助激光解吸）蛋白质、多肽、聚合物软电离，[M+H]⁺ / [M+Na]⁺ 主导，配 TOF 分析
ESI（电喷雾电离）蛋白质、核酸、代谢物产生多电荷离子，可测大分子量；常用 LC-MS 联用

6.6 MS 在蛋白组学与代谢组学中的应用

生物工程联系

  • 蛋白组学：ESI-LC-MS/MS 分析酶解肽段，通过肽质量指纹（PMF）和串联质谱鉴定蛋白质
  • 代谢组学：LC-HRMS 或 GC-MS 同时检测数百–数千种代谢物，用于疾病标志物筛选
  • 分子量测定：ESI 产生多电荷离子 (M+nH)n+，通过相邻峰电荷差反推蛋白分子量
  • 同位素标记定量：SILAC、iTRAQ、TMT 等质谱定量方法

七、综合解析步骤与实例思路（12.6）

7.1 不饱和度计算

Ω=1+n4​+2n3​−n1​​

  • n4​：四价原子数（C、Si）
  • n3​：三价原子数（N、P）
  • n1​：一价原子数（H、X）

仅含 C、H、O 时：Ω=1+nC​−2nH​​（O 不影响 Ω）

含卤素 X 时：X 视为 H 计算；含 N 时：Ω=1+nC​+2nN​−nH​−nX​​

Ω含义
0饱和链状
11 个双键或 1 个脂环
2三键 / 2 个双键 / 双键+脂环
4苯环（1 环 + 3 双键）

7.2 综合解析基本思路

综合解析五步法

  1. 确定分子式：元素分析 + MS（分子量）→ 推分子式 → 计算 Ω
  2. IR 找官能团：C=O、O–H、N–H、C≡C、C=C 等
  3. UV 辨共轭：是否含共轭体系或芳香环
  4. NMR 定骨架：¹H（位移/积分/裂分）、¹³C（碳数和类型）、DEPT
  5. 拼接验证：拼出结构 → 检查所有谱图数据是否一致 → 与标准谱比对

7.3 经典例题

例 1：C₅H₁₀O，IR：1700 cm⁻¹ 强吸收；¹H NMR：δ 1.02（二重峰），δ 2.13（单峰），δ 2.22（七重峰）

  • Ω=1+5−10/2=1 → 有 C=O
  • IR 1700 → 羰基
  • δ 2.13（3H, 单峰）→ CH₃–C=O
  • δ 1.02（6H, 二重峰）→ 两个等价 CH₃–CH
  • δ 2.22（1H, 七重峰）→ CH(CH₃)₂
  • 结构：(CH₃)₂CH–CO–CH₃（3-甲基-2-丁酮）

7.4 综合解析实例

例 2：某液体，分子式 C₈H₈O₂，IR：1680 cm⁻¹（强）、3200–2500 cm⁻¹（宽）；¹H NMR：δ 7.2–8.0（4H, m）、δ 9.8（1H, s）、δ 4.0（2H, s，可被 D₂O 交换后消失）

  • Ω=1+8−8/2=5 → 高度不饱和，可能含苯环（4）+ 一个双键（1）
  • IR 1680 → 缔合 C=O（羧酸或酰胺）；3200–2500 宽峰 → 羧酸 O–H（缔合）
  • ¹H NMR δ 9.8（1H, s）→ –COOH
  • δ 4.0（2H, s，可交换）→ –OH（酚）
  • δ 7.2–8.0（4H, m）→ 二取代苯环
  • 综合推断：水杨酸（邻羟基苯甲酸）
  • 验证：IR 苯环 C=C 1600/1580、缔合 C=O 1680、O–H 宽峰；¹H 苯环 4H + COOH + 酚 OH 一致

检验要点

  • 不饱和度 Ω=5 与“苯环 + C=O”吻合
  • COOH 氢与酚 OH 氢均可被 D₂O 交换
  • 取代位置（邻位）需结合 IR 芳环面外弯曲或耦合模式最终确认

易错点

易错点

  • 不饱和度公式：仅含 C/H/O 时 Ω=1+nC​−nH​/2，O 不参与计算
  • 卤素：把 X 当作 H 计算（X 也是一价）
  • 氮规则：含奇数个 N，分子离子峰为奇数质量
  • n+1 规则只针对相邻碳上等性质子；超过 3 键不裂分；等性质子间不裂分
  • OH/NH 因快速交换常出单峰，不参与偶合
  • 3000 cm⁻¹是 sp² 与 sp³ C–H 的分界线，不是 C=C 吸收
  • 缔合 O–H 是宽峰，不要与 N–H 混淆
  • DEPT-135 中 CH₂ 向下，CH₃/CH 向上，季碳不出峰
