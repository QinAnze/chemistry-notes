一、羧酸

参考资料：有机化学9-1羧酸及其衍生物9-1 20260518.pdf | 有机化学9-2羧酸及其衍生物9-2 20260521.pdf | 有机化学9-2羧酸及其衍生物9-2 20260526.pdf

1.1 羧酸的分类与命名 命名

分类

  1. 按羧基数目：一元酸（monocarboxylic acid）、二元酸（dicarboxylic acid）、多元酸
  2. 按烃基结构：脂肪酸（fatty acid，饱和/不饱和）、芳香酸（aromatic acid）、脂环酸
  3. 按官能团关系：常见有羟基酸、酮酸、氨基酸等取代羧酸

命名规则

  1. 系统命名（IUPAC）：选择含羧基的最长碳链为主链，称”某酸”，后缀 -oic acid；从羧基碳开始编号，取代基位次可用希腊字母 α、β、γ、δ 等表示，最末端碳原子可用 ω 表示
  2. 不饱和脂肪酸：选择同时含有羧基和不饱和碳原子在内的最长碳链为主链，称为”某烯酸”（-enoic acid）或”某炔酸”，编号从羧基碳开始，需标明双键构型（Z/E）
  3. 脂环族羧酸：将环作为取代基，看做甲酸衍生物，称”环某烷甲酸”
  4. 多元羧酸：选择含有尽可能最多羧基在内的最长碳链为主链，在”酸”前面标明羧基数目
  5. 芳香酸：以苯甲酸（benzoic acid）为母体，编号从羧基所连碳开始，邻/间/对位或用数字标注

常见俗名

  • 一元脂肪酸：甲酸 formic acid、乙酸 acetic acid、丙酸 propionic acid、丁酸 butyric acid
  • 二元酸：草酸 oxalic acid（乙二酸）、丙二酸 malonic acid、琥珀酸 succinic acid（丁二酸）、己二酸 adipic acid
  • 芳香/取代酸：苯甲酸 benzoic acid、水杨酸 salicylic acid、肉桂酸 cinnamic acid、阿司匹林 aspirin（乙酰水杨酸）

1.2 羧酸的结构 共轭效应

结构要点

  • 羧基碳为 sp2 杂化，羰基氧与羟基氧均参与形成 p-π 共轭体系，使 C=O 与 C−O 键长趋于平均化（双键变长、单键变短）
  • 共轭使羰基亲电性降低（与醛酮相比不易发生亲核加成），同时使 O-H 键极化增强，氢易以 H+ 形式电离，故显酸性
  • 羧基的活泼部位：羰基可还原、−OH 可被取代、α-H 可取代、可发生脱羧

结构对比：羧酸 vs 醛酮

特征醛/酮羧酸
羰基亲电性强（易亲核加成）弱（−OH 给电子共轭分散正电）
典型反应亲核加成−OH 取代（加成-消除）、酸性
α-H 活泼性较活泼较弱（需 HVZ 条件）
共轭体系局部 p-π整体 −COOH 三中心共轭

羧酸根负离子 RCOO− 中两个 C-O 键完全等长（约 0.127 nm），负电荷均分于两氧，共振稳定，是羧酸具酸性的热力学根源。

1.3 物理性质

氢键二聚体

羧酸分子间通过两个氢键形成**二聚体（dimer）**缔合，使其沸点显著高于同碳数的醇。 HCOOH 沸点 100.7∘C（对比 C2​H5​OH 78.5∘C、n-C3​H9​OH 97.4∘C、CH3​COOH 117.9∘C）。

性质规律
沸点同碳数：羧酸 > 醇 > 醛酮 > 醚；因二聚缔合，沸点异常高
水溶性低级羧酸（≤4 碳）与水互溶；随碳链增长疏水性增强，溶解度骤降
状态/气味低级酸有刺激气味；≥10 碳的脂肪酸为蜡状固体，无味
二元酸易形成分子内/间氢键，水溶性较好；沸点随碳数升高

1.4 酸性 酸碱反应

酸性顺序

RCOOH>H2​CO3​>PhOH>H2​O>HC≡CH>R−NH2​>R−H 羧酸 pKa​ 约 45；酚约 10；水 15.74；醇 1619；炔约 25；氨约 34；烷烃约 50。

影响因素 基团分类

  • 吸电子基（EWG，−I/−C） 连接 −COOH 使 H+ 电离增加，酸性增强；给电子基（EDG，+I/+C） 减弱酸性
  • 取代基位置：诱导效应随距离衰减，α 位影响最大；共轭效应需共平面传递
  • 苯甲酸邻位效应：邻位取代基（无论给/吸电子）常使酸性异常增强，因邻位基团扭曲羧基平面、破坏共轭，并稳定羧酸根
  • 二元酸：第一电离 pKa1​ 较小（另一羧基吸电子），第二电离 pKa2​ 较大（羧酸根给电子排斥），如草酸 pKa1​=1.29、pKa2​=4.28

取代基对 pKa​ 的影响（诱导效应）

卤素 −I 强弱 F>Cl>Br>I，酸性随之递减。给电子烷基使酸性减弱：CH3​COOH（4.75）< HCOOH（3.75）。

共轭与取代位置效应（取代苯甲酸 pKa​）

  • 对位 −NO2​（−I,−C）3.42；间位 −NO2​（仅 −I）3.49；对位 −OH（+C>−I）4.57 ——对位给电子共轭使酸性减弱
  • 邻位 −NO2​ 2.21（最强，邻位效应）；间位 −OH 4.05；对位 −OH 4.57；邻位 −Cl 2.92

Fischer 酯化机理（Fischer esterification） 立体选择性

  • 酰氧键断裂（acyl-oxygen cleavage）：大多数情况下羧酸脱 −OH、醇脱 −H，机理为质子化羰基 → 醇氧亲核加成 → 质子转移 → 失水 → 去质子。手性醇的 C−O 键不断裂，构型保留。见屏幕截图 2026-05-28 210647.png
  • 烷氧键断裂（alkyl-oxygen cleavage）：叔醇、苄醇、烯丙醇时醇脱 −OH，按 SN​1 进行，可能发生消旋化和重排。见屏幕截图 2026-05-28 212957.png
  • 酯化为可逆反应，加大醇或酸的量、及时分出水可推动正向进行
  • 判断断键方式的实验：用同位素标记醇 R′18OH 进行酯化，若产物酯含 18O 而水无 18O，则为酰氧键断裂（醇脱 H）。多数伯仲醇符合此规律

酯化立体化学规律

  • 伯、仲醇：酰氧键断裂，手性醇构型保留（C−O 不断）
  • 叔醇、苄醇、烯丙醇：烷氧键断裂，经碳正离子 SN​1，可能消旋化/重排
  • 空间因素：羧酸与叔醇酯化困难（位阻），常用酰氯或酸酐代替

(3) 脱羧反应 脱气体

脱羧条件

当 R 为吸电子基（EWG，如 −NO2​、−CO−、−CN、−COOH、C=C）时，β-位存在不饱和键的羧酸易脱羧。β-酮酸、丙二酸、硝基乙酸等尤为容易。

R−COOHNaOHΔ​R−H+CO2​

脱羧机理

  • β-羰基酸脱羧：经过六元环状过渡态，羰基氧与羧基氢形成氢键，C−C 键断裂同时质子转移，无需催化剂即可在温和条件脱羧——这是乙酰乙酸乙酯酮式分解、TCA 循环中草酰琥珀酸脱羧的化学基础
  • 一般羧酸脱羧：需高温、强碱；羧酸根负离子在过渡态形成碳负离子，故 EWG 稳定碳负离子时易脱羧
  • 生物体内脱羧：由脱羧酶（含辅酶焦磷酸硫胺素 TPP / 磷酸吡哆醛 PLP）催化，温和条件下进行
(a) Kolbe 合成法 人名反应

电解羧酸盐溶液（阳极氧化脱羧偶联）： 2R−COO−电解​R−R+2CO2​+2e− 两个羧酸根在阳极失去电子、脱羧生成烷基自由基，再偶联生成烷烃 R−R（单羧酸盐得偶联烷烃）。

(b) Hunsdiecker 反应 人名反应

银盐与溴反应得少一个碳的溴代烃：R−COOAg+Br2​→R−Br+CO2​+AgBr 经自由基机理，得少一个碳的溴代烷，是由羧酸降解制备卤代烃的方法。

(c) 二元酸的受热反应
碳链长度反应类型产物
乙二酸、丙二酸脱羧一元酸 + CO2​
丁二酸、戊二酸脱水五/六元环状酸酐
己二酸、庚二酸既脱羧又脱水环酮（五/六元环）

Blanc 规律 人名反应

二元酸受热反应倾向于生成五元环或六元环化合物（成环热力学有利）。

(4) α-H 反应

(a) Hell-Volhard-Zelinsky 反应（HVZ）人名反应 取代

RCH2​COOHBr2​/Cl2​P​RCHXCOOH

HVZ 机理与意义

  • 磷催化下先生成酰卤（PCl3​/PCl5​ 或 PBr3​ 与羧酸作用），酰卤的 α-H 比羧酸更活泼，更易烯醇化卤代；卤代后酰卤再与羧酸交换（Schutzenberger 平衡）
  • 实际催化循环：P+X2​→PX3​，PX3​+3RCOOH→3RCOX+H3​PO3​
  • 可通过控制卤素用量实现单卤代；羧酸本身直接卤代困难（羧基 −OH 的 p-π 共轭降低 α-H 活泼性）
  • 卤代产物是重要的合成中间体；卤素的 −I 效应进一步活化 α-H，故易多卤代

后续反应

  • 继续酯化，加 Zn，得到 Reformatsky 试剂 人名反应：R−CH(ZnX)−COOR′，可与醛酮亲核加成合成 β-羟基酸酯
  • 卤素可被其他基团取代：NaCN 加氰基（→羧酸）、NH3​ 加氨基（→氨基酸）、NaOH 加羟基（→羟基酸）取代

羟基酸脱水规律

成环倾向遵循 Blanc 规律：五元、六元环最稳定。γ-、δ-内酯极易形成且稳定，是天然产物（如维生素 C、香豆素内酯）常见骨架。β-羟基酸因无法形成稳定四元环，转而消除脱水。

二、羧酸衍生物

2.1 命名规则 命名

命名规则

  1. 酰卤：将羧酸的”酸”换成”酰氯/酰溴”，如乙酸→乙酰氯 acetyl chloride
  2. 酰胺：将羧酸的”酸”换成”酰胺”；N 上有取代基则为”N-某酰某胺”；成环叫内酰胺（lactam，如 ε-己内酰胺）
  3. 腈：与同碳数羧酸相似，将”酸”字换成”腈”字（如丙烯腈 acrylonitrile）
  4. 酸酐：同酸为”X酸酐”；不同酸为”X酸X酐”（如乙酸苯甲酸酐）
  5. 酯：酸名 + 醇烃基名 + 酯（如乙酸乙酯 ethyl acetate）
  6. 官能团优先级：羧酸 > 磺酸 > 酸酐 > 酯类 > 酰卤 > 酰胺 > 腈 > 醛酮 > 醇 > 酚 > 胺 > 醚

(4) 酰胺的反应

(a) 酸碱性

酰胺同时具有弱酸性和弱碱性（氮上孤对电子与羰基共轭，碱性大幅降低）。

  • 碱性：CH3​CONH2​+HCl→CH3​CONH2​⋅HCl（弱，可被水解破坏）
  • 酸性：2CH3​CONH2​+HgO→(CH3​CONH)2​Hg+H2​O（氮上氢具弱酸性）

酰胺酸碱性解释

酰胺氮上孤对电子与羰基 π 体系共轭，向羰基离域，使氮上电子云密度大幅降低，故：

  • 碱性极弱（共轭酸 pKa​≈−0.5，远弱于胺 pKa​∼10）
  • 氮上氢具弱酸性（pKa​∼17），可被强碱夺取形成酰胺负离子
  • 酰胺的 C−N 键因共轭具有部分双键性质，旋转受阻，存在顺反异构——这正是肽键呈平面反式构象的化学根源

(c) Hofmann 降解 人名反应

生物工程联系：硫酯与高能键

乙酰辅酶 A（acetyl-CoA）是硫酯（thioester，R−CO−S−CoA）。硫酯键水解自由能高（接近 ATP），是代谢中的高能键：乙酰 CoA 进入 TCA 循环、脂肪酸合成/β-氧化、氨基酸代谢均以硫酯为活化形式转运二碳单位。这正对应”硫酯比氧酯更活泼、更易被亲核进攻”的化学原理（离去基 RS− 较稳定）。

(7) 酮-烯醇互变异构 共轭效应

要点

  • 烯醇互变异构（keto-enol tautomerization）在羧酸衍生物里均可出现，但普通单羰基化合物中烯醇式比例极小（酮式更稳定，能差 45~60 kJ/mol）
  • β-二羰基化合物（R−CO−CH2​−CO−R′）：中间亚甲基被两个羰基双重致活，烯醇式因形成分子内氢键和 π-π 共轭而显著稳定，比例大增
  • 如果 R、R’ 为 EDG 则烯醇式不稳定；如果一个基团是 EWG 则易在该侧形成烯醇
  • 验证反应：CH3​C(OH)=CH−COEtPCl5​​CH3​C(Cl)=CH−COEt（证明烯醇式羟基存在）

酮-烯醇平衡的实验证据

现象说明
与 NaBH4​ 反应酮式羰基被还原
与 H2​N−OH、HCN 反应酮式羰基的亲核加成
与 Na 反应放出 H2​烯醇式羟基的活泼氢
与 PCl5​ / AcCl 反应烯醇式羟基被取代（生成氯代/酯化产物）

平衡因 β-二羰基化合物可同时表现酮式（羰基反应）和烯醇式（羟基反应）两套化学性质，是判断互变异构存在的经典依据。

(8) Claisen 缩合 人名反应

RCH2​−CO−OR′+H−CH(R)COOR′碱​RCH2​CO−CH(R)COOR′+HOR′

特点

  • 原理：2 当量酯，一个脱 OR′−、一个脱 α-H；碱拔氢后 α-C− 进攻另一分子酯羰基，加成-消除得 β-酮酯
  • 碱用量：催化量则反应可逆；大于化学计量（如 NaOEt，每分子酯 1 当量）才能使平衡右移完全——因产物 β-酮酯酸性较强会被碱夺氢形成稳定烯醇盐，需酸化才游离出产物
  • 碱需与酯的烷氧基匹配（EtO 酯用 NaOEt），避免酯交换
  • 常用强碱：NaH,NaNH2​,LDA,Ph3​CNa,t-BuOK（弱亲核性强碱）
  • 作用：合成 1,3-二羰基化合物；类似羟醛缩合 控pH反应

Claisen 缩合机理（以乙酸乙酯制乙酰乙酸乙酯为例）

  1. 夺氢：CH3​COOEt+EtO−⇌−CH2​COOEt+EtOH（平衡偏左，因酯 α-H 弱酸）
  2. 加成：−CH2​COOEt 进攻另一分子 CH3​COOEt 的羰基碳，形成四面体中间体
  3. 消除：EtO− 离去，生成 β-酮酯 CH3​COCH2​COOEt
  4. 不可逆夺氢：产物 β-酮酯 α-H 受双重致活（pKa​≈11），被 EtO− 夺氢形成稳定烯醇盐——这一步使总反应推向正向
  5. 酸化：加酸游离出乙酰乙酸乙酯 关键：步骤 1 平衡不利，靠步骤 4 不可逆夺氢拉动整体反应，故必须用等摩尔以上碱。

交叉 Claisen 与 Dieckmann 缩合

  • 交叉酯缩合：一分子无 α-H 的酯（如苯甲酸乙酯、甲酸酯、草酸酯）与一分子有 α-H 的酯缩合，可避免自身缩合副产物
  • Dieckmann 缩合 人名反应：端二酯的分子内 Claisen 缩合，生成五/六元环 β-酮酯

与生物工程的联系（整合）

关键联系

  • 肽键即酰胺键：蛋白质一级结构由氨基酸的 α-氨基与 α-羧基缩合形成酰胺（肽键），其部分双键性质（共轭）决定了平面的反式构象
  • 脂质与酯：甘油三酯（triglyceride，三脂肪酸甘油酯）、磷脂（phospholipid，甘油二酯 + 磷酸胆碱/乙醇胺）核心均为酯键；油脂皂化值（saponification value）定义为完全皂化 1 g 油脂所需 KOH 毫克数，反映脂肪酸平均相对分子质量
  • 硫酯/乙酰 CoA 高能键：硫酯比氧酯更易水解（RS− 是较好的离去基，且 C−S 键弱、共轭差），是代谢中转移乙酰基的高能载体；硫酯水解放能接近 ATP
  • β-酮酸脱羧：乙酰乙酸脱羧生成丙酮，是酮体代谢（糖尿病酮症）化学基础；TCA 循环中草酰琥珀酸的脱羧、异柠檬酸→α-酮戊二酸的脱羧均遵循 β-羰基酸脱羧规律
  • 三羧酸循环（TCA / Krebs）：草酰乙酸 + 乙酰 CoA → 柠檬酸（柠檬酸合酶催化的缩合反应，类似 Claisen 加成→水解），全程以羧酸/酰基转化为骨架
  • 氨基酸代谢：氨基酸的羧基经脱羧生成生物胺（如组氨酸→组胺、谷氨酸→GABA、5-羟色氨酸→5-羟色胺、多巴→多巴胺）；转氨基反应将 α-氨基转移给 α-酮戊二酸
  • 脂肪酸 β-氧化：脂酰 CoA 经脱氢→水化→脱氢→硫解四步循环，每次切下乙酰 CoA，硫酯酶催化的硫解即 β-酮硫酯的断裂

化学原理→代谢规律的对应

  • 酰胺稳定（肽键低能）→ 蛋白质需蛋白酶特异性催化才水解，避免体内随意断裂
  • 硫酯活泼（乙酰 CoA 高能）→ 活化乙酰基，驱动 Claisen 型缩合（柠檬酸合成）
  • 磷酯可皂化/酯交换 → 磷脂代谢、膜脂重塑的基础
  • β-酮酸易脱羧 → 酮体生成、TCA 中间体脱羧的化学本质
  • 羧酸根共振稳定 → 氨基酸两性、脂肪酸离子化（肥皂去污）的根源

  1. “Kolbe 合成法：电解羧酸盐溶液得 CH3​−R−R−CH3​” → 更准确：两个羧酸根在阳极氧化脱羧生成烷基自由基后偶联，单羧酸盐产物应为 R−R（已修正）。
  2. “酰胺水解生成 NH3​” → 伯酰胺生成 NH3​，仲酰胺生成 RNH2​，叔酰胺生成 R2​NH（已补充）。
  3. “氰基水解生成 NH4+​” → 机理上先生成酰胺，再水解为羧酸 + NH4+​（酸性）或 NH3​（碱性）（已补充）。
  4. “Curtius/Schmidt 重排最终产物都是 R−NH2​” → 正确，中间体均为异氰酸酯 R−N=C=O，与 Hofmann 一致（已统一）。
  5. 补充：Rosenmund 还原需”催化剂中毒”（喹啉-硫使 Pd 失活）才能停在醛；酯化在叔醇时按 SN​1 烷氧键断裂，可重排/消旋——均为易错点。

易错点汇总

常见易错点

  1. 还原剂选择：NaBH4​ 只还原醛酮、酰氯，不能还原羧酸/酯/酰胺/腈；还原羧酸必须用 LiAlH4​ 或 B2​H6​
  2. 酯化断键判断：伯/仲醇为酰氧键断裂（构型保留），叔醇为烷氧键断裂（SN​1，消旋/重排）——区分依据是醇的性质
  3. 水解可逆性：碱催化皂化不可逆（生成羧酸根），酸催化水解可逆（Fischer 逆反应）——故油脂皂化用碱
  4. 活性顺序方向：酰氯 > 酸酐 > 酯 > 酰胺，不要把酸酐与酰氯颠倒；羧酸本身因羧酸根离去性中等，活性介于酸酐与酯之间
  5. 格氏试剂与羧酸：格氏试剂会被羧酸的活泼氢破坏生成 RH，不能直接加成羧酸制酮——应先将羧酸转为酰氯/腈再反应
  6. Hofmann 降解碳数：产物少一个碳；迁移步骤构型保持（不是翻转，不是消旋）
  7. HVZ 卤代：磷催化先生成酰卤再卤代；不能直接卤代羧酸（羧基本身 α-H 不够活泼）
  8. 二元酸受热规律：乙/丙二酸脱羧，丁/戊二酸脱水成酐，己/庚二酸脱羧脱水成环酮——切忌混淆碳数与产物
  9. Claisen 缩合碱用量：需大于化学计量的碱（非催化量），否则产物以烯醇盐形式存在无法游离——区别于羟醛缩合
  10. 丙二酸酯 vs 乙酰乙酸乙酯合成法：前者产物为取代乙酸（再水解脱羧），后者产物为取代丙酮（再酮式分解）——切勿调换
