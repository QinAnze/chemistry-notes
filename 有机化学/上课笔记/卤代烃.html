卤代烃 §

命名 命名 §

  1. 烃为母体，卤原子为取代基，与烷烃命名相近
  2. 选连有取代基(含卤原子)最多的最长链为主链
  3. 主链编号：从靠近取代基一端开始编号，使取代基位次最小； 有选择余地时，使英文首字母排前的取代基编号较小

反应 §

  1. 亲核取代 取代
    1. 原理：R-X中−R为δ+而−X为δ−，所以C容易受负离子或带有孤对电子的中性分子的进攻，此类分子为Nu Nu和E 此时-X为L L
    2. 反应式：R−X+Nu→R−Nu+X−
    3. 常见反应：
      1. 水解：RX+H2​O→ROH+HX，在碱性条件可以中和HX，使得反应正向进行 控pH反应 反应活性：RI>RBr>RCl
      2. 醇解：Williamson反应 人名反应：合成醚 RX+R′ONa→ROR′+NaX 适用于伯卤代烷烃，仲的效果差，叔走消除
      3. 氨解：NH3​RX​RNH2​RX​R2​NHRX​RNRX​R4​N+X− 最终生成季铵盐
      4. 氰解：RX+NaCN醇​RCN水解​RCOOH 用于增长碳链
      5. 和炔基钠反应：RX+R′C≡CNa→R′C≡C−R” 适用于伯卤代烷烃
      6. 与硝酸银反应：鉴别卤代烷 RX+AgNO3​→R−ONO3​+AgX↓ 可以根据沉淀颜色鉴别卤代烷，反应活性：叔>仲>伯，伯需要Δ 控温，RI>RBr>RCl，不适用于Ar-X
    4. 机理 亲核取代 取代 立体选择性 立体选择性 控pH反应
      1. SN​1
        1. 单分子亲核取代，反应速率仅与R-Br浓度有关，反应两步进行
        2. 机理为-X离去，出现R+碳正离子，经历碳正离子重排，Nu再进攻碳正离子，形成R-Nu键
        3. 决速步骤是R-X的断裂出现碳正离子
        4. 立体化学上，进攻碳正离子可以从任意方向进攻，导致最后R和S型的量几乎等量，外消旋化 立体选择性 内消旋 立体化学
        5. 碳正离子稳定性决速，所以3’C>2’C>1’C
      2. SN​2
        1. 双分子亲核取代，反应速率与R-Br和Nu浓度均有关，反应一步协同进行
        2. 机理是Nu和R-X形成中间过渡态Nu-R-X后X离去，形成R-Nu和X−，是一个协同的步骤，即R-Nu形成和R-X断裂同时发生
        3. 决速步骤是中间协同这一步
        4. 立体化学上，Nu从背面进攻，会导致构型翻转，称为Walden转化 人名反应 立体选择性 注意：骨架构型发生改变，R/S符号 有可能不变
    5. 影响因素
      1. -R的影响 决定因素
        1. -R对SN​1的影响：碳正离子稳定性，所以3’C>2’C>1’C
        2. -R对SN​2的影响：空间位阻，位阻越大，Nu越不易进攻，所以1’C>2’C>3’C
        3. 特殊的：
          1. 烯烃：C=C-X的C-X难以断裂，比如氯乙烯，不活泼，不走SN​反应
          2. 烯丙型：更稳定，更易于进行，走SN​1有重排，走消除得共轭二烯烃
          3. 苄基：更易反应
          4. 桥环得卤原子在桥头碳上，自身位阻大，且难离去，所以SN​都难以进行
      2. Nu的影响
        1. 概念：Nu性：亲核试剂对带有正电性碳原子的亲和力或者置换离去基团的能力
        2. Nu性大小，对SN​1无影响，对SN​2影响显著，Nu性增大，利于SN​2
        3. Nu性大小比较：
          1. 反应中心相同，碱性越强，亲核性越强 eg:RO−>HO−>ArO−>RCOO−
          2. 负离子>共轭酸
          3. 同主族与溶剂极性有关：
            1. 极性质子性溶剂：Nu性与碱性相反，比如Nu性（羧酸，醇，水中）：I−>Br−>Cl−
            2. 极性非质子性溶剂：Nu性与碱性相同，比如DMSO（二甲亚砜），乙腈，丙酮
            3. 非极性溶剂：Nu难溶，不适用
          4. 体积越小，位阻越小，Nu性越强 立体选择性
      3. 溶剂极性的影响
        1. 非极性：Nu难溶，不使用
        2. 极性质子性：利于SN​1，利于RX的解离，且和Nu形成HB，不利于SN​2进行
        3. 极性非质子性：利于SN​2进行，使负离子裸露，活性高
      4. L的影响
        1. L即X，C-X的强度决定了X的L性
        2. 一般来说，L能力：R−SO3​Ph>RI>RBr>RCl>ROOC−CH3​
        3. L能力强，对于SN​1更有利
        4. 判断离去能力：看离去基团的碱性，基团碱性强，离去倾向小，所以：RO−,HO−,H2​N−不是好的离去基团
  2. 消除 消除
    1. X为EWG，所以R-X的β-H有酸性，可以在NaOH/醇中消去HX得到碳碳双键三键
    2. Zaitsev规则 人名反应：主要消含H少的C上的H原子
    3. 邻二卤代物和偕二卤代物在强碱，乙醇，Δ环境下会脱两分子H形成炔烃，如果不加热，停留在烯卤代烃 控温 控pH反应
    4. 邻二卤代物在Zn下的反应：R1​R2​CX−CXR3​R4​ZnEtOH, Δ​R1​R2​C=CR3​R4​+ZnX2​
    5. 机理 消除 消除 立体选择性
      1. E1
        1. 单分子消除，同样经历碳正离子阶段，第一步为α-C脱去X形成碳正离子并重排，第二步为β-C被强碱拔H，α-C与β-C形成双键
        2. 第一步是慢反应，决速步骤
        3. E1和SN​1第一步一样，所以会竞争，需要根据反应条件判断，eg:和水溶液，有氢氧根，走SN​1，和乙醇，有EtO−，走E1
      2. E2
        1. 双分子消除，一步完成，同样有过渡态
        2. 立体条件 立体选择性：β-H和L必须反式共平面，此时能量更低，对于Nu的进攻更有利
        3. 立体专一性：赤式以cis产物为主，苏式以trans产物为主 立体选择性
        4. E2和SN​2也类似，如果Nu进攻β-H走E2，如果Nu进攻α-C走SN​2
    6. 影响因素
      1. -R
        1. 碳正离子稳定性影响E1
        2. R的β-H越多，更容易走E2，所以活性：3’C>2’C>1’C
      2. L
        1. 好的离去基团利于SN​1和E1，反之利于SN​2和E2
        2. 活性RI>RBr>RCl
      3. 溶剂：极性溶剂利于E1，不利于E2
      4. 温度：高温有利于消除 控温
    7. 如果分子上有双键三键，那消烯丙位的H，这样形成共轭烯炔烃，更加稳定
    8. 大位阻试剂：适合末端碳碳双键和三键的形成，位阻小易进攻 立体选择性
    9. Hofmann规则 人名反应：动力学控制，适用于强碱弱碱化物（如氨、肼、肼衍生物）和环状氨碱（如四甲基氢氧化铵），和扎伊采夫规则的对比：
  3. 消除和取代的竞争：
    1. RX结构影响：3’C-X易消除，1’C-X易取代
    2. Nu和碱性影响：Nu性强的利于SN​2，碱性强的利于E2
    3. 碱的浓度越大越利于E 控pH反应
    4. 大位阻碱：利于消除，因为易于靠近β-H
    5. 高温消除，低温取代 控温
