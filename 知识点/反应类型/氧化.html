氧化反应

Jun 27, 2026, 28 min read

氧化反应 (Oxidation Reaction)

氧化反应（Oxidation Reaction）是通过增加氧原子、减少氢原子或失去电子来提高有机分子氧化态的官能团转化反应，与还原反应互为逆过程。氧化反应在合成中用于构建 C=O、C−O、C=C 断裂及官能团鉴别，是生物体内能量代谢（呼吸链、脱氢酶催化）的化学基础。

一、氧化反应的基本概念

定义（Definition）

氧化：有机物失去电子，或增加氧原子、减少氢原子的过程。

氧化态（oxidation state）变化主线：

  • −CH3​→−CH2​OH→−CHO→−COOH（碳氧化态 −3→−1→+1→+3）
  • 烷烃 → 醇 → 醛 → 羧酸（氧化态逐级升高）

广义氧化还包括：脱氢（−CH2​−CH2​−→−CH=CH−）、加氧（环氧化、邻二醇化）、断链（臭氧分解、KMnO₄ 强氧化）以及含杂原子底物的氧化（硫醚→亚砜→砜、胺→亚胺→氮氧化物）。

氧化剂的分类

按氧化能力与选择性可分为三类：

类别代表试剂典型应用
温和选择性氧化剂PCC、Swern、Dess-Martin、Ag2​O、MnO2​、mCPBA、OsO4​醇→醛、烯丙醇→烯丙醛、烯→环氧、烯→顺式邻二醇
强氧化剂KMnO4​/H+、K2​Cr2​O7​/H+、Jones、HNO3​醇→羧酸、烯/炔断链、芳烃侧链→羧基
鉴别型氧化剂Tollens、Fehling、Benedict醛 vs 酮、脂肪醛 vs 芳香醛的鉴别

选择氧化剂的核心原则：控制氧化深度与保护不饱和键。

二、烯烃的氧化

1. 环氧化（Epoxidation）

过氧酸环氧化

过氧酸（peracid）将烯烃氧化为环氧化物（epoxide），氧原子以 顺式（syn） 方式加到双键两侧，保留烯烃原有构型： R2​C=CR2​+R′CO3​H→环氧化物+R′CO2​H

  • 常用过氧酸：mCPBA（间氯过氧苯甲酸，meta-chloroperoxybenzoic acid）、过氧乙酸 CH3​CO3​H、三氟过氧乙酸 CF3​CO3​H
  • 顺式加成：顺式烯烃得顺式环氧，反式烯烃得反式环氧
  • 电子效应：烯烃上 EDG（给电子基）加速反应；过氧酸对富电子烯烃选择性更高
  • mCPBA 可同时氧化酮为酯（即 Baeyer-Villiger 氧化，见第六节）

Sharpless 不对称环氧化

烯丙醇在 Ti(OiPr)4​ + 手性酒石酸二乙酯（DET）+ t-BuOOH 体系下发生对映选择性环氧化，是手性环氧醇合成的里程碑（Sharpless 因此获 2001 年诺贝尔化学奖）。详见 醇酚醚。

其他环氧化路线

  • 卤代醇法：CH2​=CH2​Cl2​, H2​O​HOCH2​CH2​Cl碱​环氧乙烷（分子内 SN​2 关环）
  • 工业银催化氧化：CH2​=CH2​O2​, Ag, Δ​环氧乙烷（乙烯直接环氧化，主流工业路线）

2. 邻二醇化（Dihydroxylation）

顺式邻二醇

OsO4​（四氧化锇）或稀冷 KMnO4​ 经 顺式（syn）双羟基化 得邻二醇（1,2-diol）： RCH=CHR′OsO4​, H2​O2​​RCH(OH)-CH(OH)R′ RCH=CHR′KMnO4​稀、冷、中性​RCH(OH)-CH(OH)R′

  • 机理：金属氧化物对双键发生 [3+2] 环加成，形成五元环状锇酸酯/锰酸酯中间体，水解后顺式开环得邻二醇
  • OsO4​ 选择性高但剧毒、昂贵，常用催化量 OsO4​ + 共氧化剂（NMO、H2​O2​，Upjohn 法；K3​Fe(CN)6​，Sharpless AD 法）
  • KMnO4​ 必须严格控制稀、冷、中性；碱性与加热条件下会过度氧化甚至断链

反式邻二醇的获得

顺式邻二醇由 OsO4​/KMnO4​ 直接得到；反式邻二醇须经 环氧开环 路线：烯烃先用 mCPBA 环氧化，再在酸性或碱性下水解开环，得反式邻二醇（详见 醇酚醚 环氧化物开环）。

3. 臭氧分解（Ozonolysis）

臭氧氧化断链

臭氧（O3​）与烯烃加成生成臭氧化物（ozonide），后处理决定产物： R2​C=CR2​+O3​→臭氧化物后处理​R2​C=O+O=CR2​

后处理方式产物
还原性（Zn/H2​O、(CH3​)2​S、PPh3​）醛 / 酮
氧化性（H2​O2​）羧酸 / 酮（醛被进一步氧化为酸）
NaBH₄醇（将醛酮还原为醇）

应用：制备醛酮、推断烯烃结构。

由断链产物反推烯烃结构

臭氧分解（还原后处理）是推断烯烃结构的经典方法，反推思路：

  1. 写出所有羰基产物（醛 RCHO、酮 RCOR′）
  2. 把每个 C=O 的 氧换为双键，两两对接即得原烯烃的 C=C
  3. 产物中含 HCHO（甲醛）→ 原双键端为 =CH2​
  4. 产物中含 RCOOH（用 H2​O2​ 后处理）→ 原双键端为 =RCH
  5. 产物中含 CO2​ → 原双键端为 =CH2​ 且用氧化性后处理

例：某烯烃经 O3​、Zn/H2​O 得 CH3​CHO 与 OHCCH2​CHO，反推原结构为 CH3​CH=CHCH2​CHO。

4. KMnO₄ 强氧化断链

KMnO₄ 氧化烯烃的条件—产物对应

条件产物
稀、冷、中性顺式邻二醇
稀、冷、碱性邻二醇（可能继续氧化）
浓、热、酸性完全断裂，得羧酸 / 酮

强氧化断链规律（按双键碳上取代基分类）：

  • R2​C= → 酮 R2​C=O
  • RCH= → 醛 RCHO（酸性条件下继续氧化为羧酸 RCOOH）
  • H2​C= → CO2​+H2​O（端基 =CH2​ 完全氧化）

5. Wacker 氧化

Wacker 氧化（Wacker Oxidation）

乙烯在 PdCl2​ / CuCl2​ / O2​ / H2​O 体系中被氧化为乙醛，端基烯烃得甲基酮（Markovnikov 取向）： CH2​=CH2​+21​O2​PdCl2​, CuCl2​​CH3​CHO RCH=CH2​+21​O2​PdCl2​, CuCl2​​RCOCH3​

  • 机理：Pd2+ 与烯烃配位，水亲核进攻后 β-消除得酮，Pd0 被 Cu2+/O2​ 再氧化实现催化循环
  • 工业意义：乙烯 → 乙醛的 Wacker 法曾是主流路线
  • 选择性：端基烯烃遵循马氏规则，得甲基酮

三、炔烃的氧化

炔烃（Alkyne）的氧化

炔键（C≡C）被氧化时一般在三键处断链：

炔烃类型试剂产物
端基炔 RC≡CH热浓 KMnO4​RCOOH+CO2​
端基炔 RC≡CHO3​ 后水解RCOOH+HCOOH（HCOOH 可再氧化为 CO2​）
非端基炔 RC≡CR′热浓 KMnO4​RCOOH+R′COOH
非端基炔 RC≡CR′O3​，还原后处理RCHO+R′CHO（或氧化为羧酸）

应用：通过断链产物反推炔烃结构，方法与烯烃臭氧分解类似。端基炔因含 =CH 端，氧化产物含 CO2​（或甲酸）是判断端基的重要线索。

四、醇的氧化

醇的氧化发生在 α-C—H 键：α-H 先断裂形成 C=O，故氧化能力取决于 α-H 的存在与氧化剂强弱。

1. 伯醇的氧化

伯醇氧化路径

R-CH2​OH温和氧化​R-CHO强氧化​R-COOH

醛比醇更易氧化，欲停留在醛阶段须用温和氧化剂或及时蒸出醛。

1.1 氧化为醛（温和条件）

氧化剂组成 / 体系特点适用范围
PCC（Pyridinium Chlorochromate，氯铬酸吡啶盐）CrO3​⋅HCl⋅ 吡啶Corey 试剂；酸性；不过度氧化为羧酸伯醇→醛，不氧化双键
Swern 氧化DMSO / (COCl)2​ / Et3​N，低温无铬无毒；温和；避免烯丙位重排伯醇→醛，敏感底物适用
Dess-Martin 试剂（DMP， periodinane）高价碘试剂中性、室温、高选择性伯醇→醛
Sarrett 试剂CrO3​⋅ 吡啶络合物不氧化双键、三键伯醇→醛，碱性环境
MnO2​活性 MnO2​专氧化烯丙醇 / 苄醇HO-R-CH=CH-CH2​OH→HO-R-CH=CH-CHO

PCC（Pyridinium Chlorochromate）

  • 即氯铬酸吡啶盐，Corey 试剂，常用以替代 Sarrett 试剂
  • 在 酸性 条件下使用
  • 特点：氧化伯醇 → 醛，不过度氧化为羧酸；不氧化孤立双键

1.2 氧化为羧酸（强氧化）

氧化剂条件特点
KMnO4​（酸性）酸性、加热强氧化剂，伯醇→羧酸
K2​Cr2​O7​/H+ / Jones 试剂CrO3​/H2​SO4​，丙酮溶剂伯醇→羧酸，仲醇→酮
CrO3​/H2​SO4​（Jones）酸性水溶液同时氧化醛为酸

2. 仲醇的氧化

仲醇氧化为酮

R-CH(OH)-R′氧化​R-CO-R′

氧化剂特点
Jones 试剂（CrO3​/H2​SO4​）酸性条件，常用
KMnO4​（酸性）强氧化剂
Swern 氧化温和，无重金属
Dess-Martin 试剂高价碘，温和
PCC选择性高
Oppenauer 氧化异丙醇铝催化，丙酮为氧化剂，专一氧化仲醇，不影响双键

酮一般不再继续氧化（无 α-H 难断裂 C−C），故仲醇氧化产物较稳定。

3. 叔醇的抗氧化性

叔醇不易氧化

  • 叔醇（3° alcohol）没有 α-H，无法形成 C=O，在常规条件下不被氧化
  • 强氧化条件下先发生 脱水（E1）生成烯烃，再由烯烃被氧化断链
  • 因此叔醇在鉴别（Lucas 试剂）与氧化反应中表现特殊：可脱水但不易直接氧化

4. 催化脱氢（Dehydrogenation）

醇的催化脱氢

伯醇、仲醇在 Cu、Ag、Ni 等催化剂下高温脱氢生成醛或酮： RCH2​OHCu, Δ​RCHO+H2​ R2​CHOHCu, Δ​R2​CO+H2​

  • 脱氢不引起 C=C 断裂，工业制甲醛即甲醇脱氢法
  • 叔醇无 α-H，不能脱氢
  • 脱氢为吸热可逆反应，通入空气（氧化脱氢）移除 H2​ 可推动反应正向
  • 这是 醛酮 制备章节中”醇催化脱氢”的氧化侧体现

5. 邻二醇的氧化断键

邻二醇（1,2-diol）的断键氧化

1,2-二醇可被 高碘酸（HIO4​） 或 四乙酸铅（Pb(OAc)4​） 定量断链，每条 C−C 键断裂生成两个羰基化合物： R1​R2​C(OH)-C(OH)R3​R4​HIO4​​R1​COR2​+R3​COR4​

  • 定量关系：每消耗 1 mol HIO4​ 断裂 1 条 1,2-二醇 C−C 键
  • α-羟基醛 / α-羟基酮同样可被 HIO4​ 氧化；邻三醇中间碳被氧化两次
  • 应用：测定糖的碳骨架与糖苷键连接位置（Periodate oxidation，见 醇酚醚）
  • Pb(OAc)4​ 多用于有机溶剂体系，机理为环状铅酸酯分解

频哪醇（pinacol）的相关反应

邻二醇除断键氧化外，在酸性条件下可发生 频哪醇重排（pinacol rearrangement） 生成酮（非氧化，属重排反应，详见 醇酚醚），注意与 HIO4​ 断键区分。

五、醛的氧化

醛氧化为羧酸

R-CHO氧化​R-COOH

醛的羰基 H 极易被氧化，故醛是比醇更强的还原剂；酮无此 H 因此抗氧化性强。

1. 一般氧化剂

氧化剂反应特点
KMnO4​/H+R-CHO→R-COOH强氧化剂，芳醛条件勿过剧烈
K2​Cr2​O7​/H+R-CHO→R-COOH酸性条件
CrO3​/H+Ph-CHO→Ph-COOH氧化芳醛为芳酸
Ag2​OR-CHO→R-COOH不氧化双键，用于不饱和醛→不饱和酸

2. 鉴别型氧化剂

醛的鉴别氧化

Tollens 试剂（银氨溶液，ammoniacal silver nitrate）： [Ag(NH3​)2​]OH

  • 氧化醛基为羧酸盐，Ag+ 被还原为 银镜（silver mirror）
  • 脂肪醛、芳香醛均反应；酮不反应
  • 不氧化双键，可区分不饱和醛

Fehling 试剂（铜-酒石酸钾钠碱性溶液）： CuSO4​+酒石酸钾钠+NaOH

  • 仅氧化 脂肪醛，Cu2+→Cu2​O 砖红色沉淀
  • 芳香醛不反应（苯环共轭降低羰基活性）
  • 酮不反应

Benedict 试剂：与 Fehling 类似，用柠檬酸代替酒石酸，更稳定，用于糖类鉴别。

芳香醛不被 Fehling 氧化的原因

芳香醛（如苯甲醛 PhCHO）的羰基与苯环共轭，羰基碳正电性被分散，且芳醛较难形成水合物中间体，故不能被 Fehling 试剂（弱氧化性、需水合机理）氧化；但 Tollens 试剂氧化性略强、机理不同，仍可氧化芳醛。这是区分脂肪醛与芳香醛的关键。

六、酮的氧化 —— Baeyer-Villiger 氧化

酮的氧化特点

酮一般 不易氧化（无羰基 H，α-C 较稳定）；强氧化条件下可发生 C−C 断裂得到混合羧酸，但合成意义有限。最重要的酮氧化反应是 Baeyer-Villiger 氧化。

Baeyer-Villiger 氧化（酮→酯）

R-CO-R′+R′′CO3​H→R-CO-O-R′ (酯)

  • 过氧酸常用：CH3​CO3​H、mCPBA、CF3​CO3​H、PhCO3​H
  • 在羰基 C 与迁移基团之间 插入一个氧，酮变为酯；环酮 → 内酯
  • 醛发生类似反应生成羧酸（迁移基团为 H）

迁移基团顺序（Migration Aptitude）

Baeyer-Villiger 反应中，哪个基团迁移决定产物结构。迁移能力顺序： H>3∘C>环己基>2∘C>−Bn>−Ph>1∘C>−Me

  • 迁移基团若具手性，迁移后 构型保持（stereospecific，详见 立体选择性）
  • 不对称酮的产物由迁移顺序决定；对称酮则产物唯一
  • 应用：环己酮 → 己内酯；苯乙酮 → 乙酸苯酯（Ph 迁移）

Oppenauer 氧化 vs Baeyer-Villiger

Oppenauer 氧化（仲醇→酮，异丙醇铝/丙酮体系）是醇的氧化而非酮的氧化，注意区分；其逆反应为 Meerwein-Ponndorf-Verley 还原（详见 还原、醛酮）。

七、芳环侧链氧化与酚的氧化

1. 芳烃侧链氧化

芳烃侧链氧化

苯环侧链 只要有 α-H，均被强氧化剂氧化为羧基： Ph-CH2​RKMnO4​/H+, Δ​Ph-COOH Ph-CH(R)R′KMnO4​/H+​Ph-COOH

特点：

  • 侧链无论长短、是否支链，只要有 α-H 一律变为 −COOH（烷基苯 → 苯甲酸）
  • 无 α-H 的侧链不氧化：叔丁基苯（PhC(CH3​)3​）不被氧化
  • 氧化剂：KMnO4​/H+、K2​Cr2​O7​/H+、稀 HNO3​
  • 应用：制备芳香羧酸（甲苯→苯甲酸、乙苯→苯甲酸、对二甲苯→对苯二甲酸）

2. 酚氧化为醌

酚（Phenol）氧化为醌

酚极易被氧化，产物为 醌（quinone）。对苯二酚（氢醌，hydroquinone）氧化为对苯醌是经典的可逆氧化还原对： HO-Ph-OH氧化还原​O=Ph=O

  • 醌式结构为共轭环己二烯二酮，有颜色（酚类放置变红即因氧化成醌）
  • 醌虽含芳环骨架但 无芳香性，化学性质类似 α,β-不饱和酮（易 1,4-加成）
  • 显影剂（米吐尔、对苯二酚）即利用氢醌→醌的氧化还原
  • 苯酚氧化条件温和（空气、Fe3+、K3​Fe(CN)6​ 均可）

醌/氢醌与生物氧化还原

对苯二酚-对苯醌是经典可逆电子载体。生物体内 辅酶 Q（泛醌，ubiquinone） 以醌/氢醌结构在呼吸链中传递电子与质子；维生素 K（萘醌类）参与凝血因子羧化；黑色素、漆酶催化的漆酚氧化聚合都经酚→醌途径。

八、其他氧化反应

1. 硫醚的氧化

硫醚（Sulfide）→ 亚砜 → 砜

R-S-R′1 eq. H2​O2​​R-S(=O)-R′ (亚砜 sulfoxide)过量氧化剂​R-S(=O)2​-R′ (砜 sulfone)

  • 温和氧化剂（H2​O2​、mCPBA、NaIO4​）控制用量得 亚砜（如 DMSO）
  • 过量氧化剂 / 强条件得 砜
  • 亚砜的 S 为手性中心（孤对电子 + 两个不同烃基时），具手性
  • 生物体内 甲硫氨酸（Met）的硫醚可被氧化为甲硫氨酸亚砜，是蛋白质氧化损伤的标志

2. 胺的氧化

胺（Amine）的氧化

  • 伯胺 / 仲胺 可被 H2​O2​ 或 mCPBA 氧化为 羟胺（hydroxylamine） 或 氮氧化物（N-oxide）
  • 叔胺 与 H2​O2​ 反应生成 氧化胺（amine N-oxide）：R3​N→R3​N+-O−
  • Cope 消除反应即以氧化胺为中间体（详见 消除）
  • 苯胺易被氧化为偶氮苯 / 苯醌，是染料化学的基础

3. 脱氢芳构化

脱氢芳构化（Dehydrogenative Aromatization）

环己烯 / 环己二烯 / 环己酮类在高温下用 Pd/C、S、Se 或 DDQ（二氯二氰基苯醌）脱氢生成芳烃： 环己烯Pd/C, Δ​苯+H2​ 环己酮S 或 Se, Δ​苯酚

  • 本质是连续脱氢使分子达到芳香稳定构型
  • 在甾体、生物碱合成中用于构建芳环
  • DDQ 是温和的脱氢试剂，常用于将二氢芳环转为芳环

十、断链反推思路小结

断链氧化反推结构三步法

  1. 写出所有氧化产物：醛 / 酮（O3​+还原）、羧酸 / 酮（O3​+氧化或 KMnO₄）、CO2​（端 =CH2​ 强氧化）
  2. 将每个 C=O 的氧替换为双键，按分子片段两两对接，恢复原 C=C / C≡C
  3. 核对称性：环烯烃断链得二羰基化合物（如环己烯→己二醛 / 己二酸）

关键判据：

  • 产物含 HCHO → 原双键端为 =CH2​
  • 产物含 CO2​ → 原双键端为 =CH2​ 且强氧化
  • 产物含 RCOOH + R′COOH → 对称/非对称烯烃均可能，需结合分子式
  • 产物含酮 RCOR′ → 原双键碳为 R2​C=

十二、易错点

常见错误

  1. PCC / Sarrett / Swern / DMP 停在醛：伯醇用温和氧化剂得醛，不过度氧化为酸；用 KMnO4​、Jones 则得羧酸。
  2. Ag2​O、MnO2​、PCC 不氧化双键：欲氧化醇/醛而保留 C=C 必须选这些；KMnO4​、Jones、K2​Cr2​O7​ 会氧化双键。
  3. Tollens 鉴别醛与酮：脂肪醛、芳香醛均反应；Fehling 仅氧化脂肪醛，芳香醛不反应 —— 这是区分脂肪醛与芳香醛的关键。
  4. 芳烃侧链氧化条件：必须有 α-H；叔丁基苯（无 α-H）不被氧化；不论侧链长短一律变为 −COOH。
  5. Baeyer-Villiger 迁移：迁移基团在 C 与 R 之间插入 O，酮→酯（非酮→酸）；醛发生 B-V 反应才得羧酸；迁移基团构型保持。
  6. 臭氧分解后处理：Zn/H2​O（还原）得醛酮，H2​O2​（氧化）得羧酸/酮 —— 切勿混淆。
  7. 顺式 vs 反式邻二醇：OsO4​、稀冷 KMnO4​ 得 顺式；经环氧开环路线得 反式。
  8. 叔醇不氧化：无 α-H；强条件先脱水成烯再断链，不能直接得醛/酮。
  9. HIO4​ 仅断 1,2-二醇：孤立二醇（1,3-二醇等）不断键；每条 1,2-二醇键消耗 1 mol HIO4​。
  10. 炔烃氧化端基判据：端基炔氧化得 CO2​（强氧化）或 HCOOH，非端基炔只得两分子羧酸。

十三、与生物工程的联系

生物氧化：脱氢酶与呼吸链

生物体内的氧化以 酶催化脱氢 为主，化学本质与醇/醛氧化对应，但条件温和、选择性极高，依赖辅酶作为电子载体。

(1) NAD⁺ / NADH 与醇脱氢

NAD+ 是脱氢酶的辅酶，吡啶环为电子受体。NAD+ 把醇的 α-H 以氢负（H−）形式夺走，醇被氧化为醛/酮，NAD+ 被还原为 NADH —— 等价于生物版的”MnO2​ / Oppenauer 氧化”。 CH3​CH2​OH+NAD+ADH​CH3​CHO+NADH+H+ CH3​CHO+NAD+ALDH​CH3​COOH+NADH+H+

典型实例：

  • 乙醇 → 乙醛（醇脱氢酶 ADH）；乙醛 → 乙酸（醛脱氢酶 ALDH）
  • 丙酮酸 → 乙酰辅酶 A + CO2​（丙酮酸脱氢酶复合体，连接糖酵解与 TCA）
  • 乳酸 → 丙酮酸（乳酸脱氢酶 LDH，糖异生方向）

戒酒药 双硫仑（Disulfiram） 抑制 ALDH 使乙醛蓄积，正是阻断”醛→酸”氧化的药理应用。

(2) 呼吸链与氧化磷酸化

线粒体电子传递链（ETC）的本质是分步氧化还原：

  • NADH 把电子传给复合体 I，沿 CoQ（泛醌，即醌/氢醌循环）→ 细胞色素（Fe2+/Fe3+ 循环）→ O2​ 逐步传递
  • 泛醌（辅酶 Q）的醌/氢醌可逆氧化还原 是呼吸链的核心电子载体，对应化学上的酚→醌氧化
  • 最终 O2​ 被还原为 H2​O，质子梯度驱动 ATP 合成 —— 这是生物体获得能量的”温和燃烧”

(3) 酶促氧化与单加氧酶

  • 细胞色素 P450（CYP450） 是肝药酶核心，催化烷基 羟基化（−CH3​→−CH2​OH，相当于生物版的”烷→醇”氧化）、烯烃 环氧化（如苯并 [a] 芘致癌代谢生成二醇环氧化物，见 醇酚醚）
  • 单加氧酶 利用 O2​ 与 NADPH 在底物中插入一个氧原子，对应 mCPBA 式的环氧化 / 羟基化
  • 漆酶 / 多酚氧化酶 催化酚 → 醌氧化聚合（漆酚固化、水果褐变、黑色素合成）

(4) 视黄醛氧化与视觉循环

  • 视黄醛（retinal） 由视黄醇（维生素 A，retinol）经 醇脱氢酶 氧化而来（醇→醛氧化）
  • 视黄醛的醛基与视蛋白赖氨酸侧链形成 Schiff 碱构成视紫红质（详见 醛酮）
  • 吸收光子后 11-顺 → 全-反异构化触发视觉信号；全-反视黄醛再经脱氢酶还原为视黄醇循环使用
  • 这一过程串联了 醇氧化（视黄醇→视黄醛） 与 亚胺形成（视黄醛+赖氨酸） 两类化学

(5) 抗氧化与自由基清除

  • 维生素 C（抗坏血酸） 的烯二醇结构易失去 2 个电子氧化为脱氢抗坏血酸（二酮内酯），构成可逆氧化还原对，是机体水相抗氧化剂
  • 维生素 E（生育酚） 的酚羟基捕获过氧自由基，中断脂质过氧化链反应（酚→酚氧自由基→醌）
  • 这些过程对应化学上的酚/烯醇氧化，体现了”温和可逆氧化还原”在生物抗氧化中的核心地位

考试重点提醒

  1. ⭐ 氧化剂的选择性：

    • 伯醇→醛：PCC、Swern、Dess-Martin、Sarrett（不氧化双键）
    • 伯醇→羧酸：KMnO4​、Jones、K2​Cr2​O7​
    • 醛→羧酸且保留双键：Ag2​O

  2. ⭐ 醛的鉴别：

    • Tollens → 银镜（脂肪醛 + 芳香醛均反应）
    • Fehling → 砖红沉淀（仅脂肪醛，芳香醛不反应）

  3. ⭐ 烯烃氧化断裂：

    • O3​ + Zn/H2​O → 醛/酮；O3​ + H2​O2​ → 羧酸/酮
    • KMnO4​（酸性、热） → 羧酸 / 酮 / CO2​
    • 由断链产物反推烯烃结构

  4. ⭐ Baeyer-Villiger 氧化：酮→酯，迁移顺序 H>3∘>2∘>Ph>1∘>Me，构型保持

  5. ⭐ 芳烃侧链氧化：必须有 α-H，叔丁基苯不氧化，烷基苯一律 → PhCOOH

  6. ⭐ 邻二醇断键：HIO4​、Pb(OAc)4​，定量断 1,2-二醇，用于糖结构测定

  7. ⭐ 生物氧化对应：ADH/ALDH 对应醇→醛→酸氧化；NAD⁺ 为电子载体；呼吸链泛醌循环对应酚→醌氧化
