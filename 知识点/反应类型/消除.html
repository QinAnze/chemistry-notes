消除反应

消除反应（elimination reaction）是有机分子中脱去两个原子或基团、同时形成新的 π 键（双键或三键）的反应。从分子中脱去的两个基团可位于同一碳（α-消除）、相邻碳（β-消除）或更远位置，其中以 β-消除最为常见，是合成烯烃、炔烃以及若干小分子（H2​O、HX、CO2​、N2​）的基本途径，也与取代、加成、氧化还原等反应存在广泛竞争。

参考资料：卤代烃 | 醇酚醚 | 羧酸及其衍生物 | 有机化学6-1卤代烃 全部 20260427.pdf | 有机化学7-1醇酚醚7-1 20260430.pdf | 有机化学9-1羧酸及其衍生物9-1 20260518.pdf

一、消除反应的分类

按消除基团相对位置分类

  1. α-消除（1,1-消除）：同一碳上脱去两个基团，生成卡宾（carbene, :CR2​）或氮烯（nitrene, :NR）。
  2. β-消除（1,2-消除）：相邻两碳各脱去一个基团，生成 C=C 或 C≡C，最常见。
  3. γ-消除（1,3-消除）等：相隔更远的消除，常伴随成环（如形成环丙烷）。
  4. 热消除（pyrolytic elimination）：不需外加酸碱、经加热进行的分子内消除，如 Cope 消除、Chugaev 消除，常为顺式消除。

按机理分类

  • E1（单分子消除）：碳正离子中间体，与 SN​1 同决速步
  • E2（双分子消除）：一步协同，过渡态反式共平面
  • E1cb（共轭碱单分子消除）：先拔氢生成碳负离子，再离去基团离去
  • Ei（分子内消除，热消除）：经环状过渡态，顺式消除

二、β-消除的三种机理

2.1 E1 机理（单分子消除，unimolecular elimination）

Tip E1 特点

速率 v=k[RX]，仅与底物浓度有关；反应分两步，决速步是碳正离子生成，与 SN​1 共享决速步与中间体。

步骤描述
第一步（慢，决速步）α-C 上离去基团 L 异裂离去，生成平面型碳正离子 R+（可能重排）
第二步（快）碱 B 拔取 β-H，电子对转移形成 C=C 双键

Note E1 关键要点

  • 底物活性：由碳正离子稳定性决定，3∘>2∘>1∘；苄基/烯丙基 > 叔烷基
  • 碱：弱碱即可（醇、水亦可），因碱不参与决速步
  • 重排：常发生碳正离子重排（甲基迁移、负氢迁移）以生成更稳定碳正离子
  • 立体化学：无严格要求，不具立体专一性
  • 溶剂：极性质子溶剂有利（稳定 R+ 与 L−）
  • 与 SN​1 竞争：同一 R+，水/弱碱亲核 → SN​1；醇/较高温 → E1；高温显著利于 E1（ΔS>0）

2.2 E2 机理（双分子消除，bimolecular elimination）

Tip E2 特点

速率 v=k[RX][B−]，与底物、碱浓度均有关；一步协同完成（与 SN​2 共享过渡态竞争），有过渡态但无中间体。

项目内容
立体条件β-H 与离去基团 L 必须处于 反式共平面（anti-periplanar），此时 C−Hσ 键与 C−Lσ∗ 反键轨道重叠最佳、能量最低；顺式共平面（syn-periplanar）能量较高，一般不发生
立体专一性 立体选择性受反式共平面限制：赤式（erythro）底物得顺式烯烃为主，苏式（threo）底物得反式烯烃为主
与 SN​2 的关系同一过渡态竞争：B 进攻 β-H 走 E2；Nu 进攻 α-C 走 SN​2
不重排因无碳正离子中间体，E2 不发生重排
底物活性3∘>2∘>1∘（β-H 越多、双键越稳定越易）

Example E2 反式共平面示例

2-溴丁烷在 EtO− 作用下，只有 H 与 Br 处于反式共平面构象时优先消除，主产物为反-2-丁烯（更稳定，符合 Saytzeff）。

环己烷体系中要求 Br 与 β-H 同处于 直立键（axial）反式共平面位置：若反式-1,2-二溴环己烷中两个 Br 均为直立键才能消除。这是环系立体化学的常考点。

Tip E2 过渡态连续谱

E2 过渡态可分为”似 E1 型”（离去基团先断裂明显，过渡态碳正离子性增强）与”似 E1cb 型”（C−H 断裂明显，过渡态碳负离子性增强）。底物碳正离子稳定（3∘、苄基）则偏向 E1-like；离去基团差（F−）或 β-H 酸性强（羰基 α-H）则偏向 E1cb-like。

2.3 E1cb 机理（共轭碱单分子消除，unimolecular elimination via conjugate base）

Tip E1cb 特点

速率 v=k[底物]（决速步为碳负离子生成后离去基离去），机理分两步：先拔 β-H 生成碳负离子（共轭碱），再离去基团 L− 离去。中间体为碳负离子而非碳正离子。

步骤描述
第一步（快平衡）强碱拔取 α-H（被吸电子基致活者）生成碳负离子 R−CH−−L
第二步（慢，决速步）碳负离子上孤对电子形成 C=C，同时 L− 离去

Note E1cb 发生条件

  • β-H 酸性强：被 EWG（−NO2​,−COR,−CN,−COOR,−SO2​R 等）致活
  • 离去基团差：如 F−、OR−、NR2−​（E1cb 中 L 不必先离去）
  • 强碱：需强碱才能可逆生成碳负离子
  • 典型实例：β-羟基羰基化合物的脱水（羟醛缩合逆反应的脱水步骤）、β-卤代羰基化合物脱 HX、弱离去基季铵盐的 Hofmann 消除
  • 不重排（无碳正离子）

机理选择的判断

  • 叔卤代烷 + 弱碱 → E1；叔卤代烷 + 强碱 → E2
  • 仲/伯卤代烷 + 强碱 → E2；仲卤代烷 + 弱碱/极性质子溶剂 → E1
  • 底物有强 EWG 致活 β-H + 强碱 + 差离去基 → E1cb
  • 高温、强碱、位阻碱 → 消除（E2/E1cb）为主

三、区域选择性：Saytzeff vs Hofmann 规则

3.1 Saytzeff（Zaitsev）规则

Saytzeff 规则

在消除反应中，主要消除含 H 较少的 β-C 上的 H，生成双键上取代基更多、热力学更稳定的烯烃（Zaitsev 烯烃）。

理论依据：双键上烷基取代越多，超共轭与诱导给电子稳定作用越强，烯烃热力学越稳定（双键上取代基数：四取代 > 三取代 > 二取代 > 一取代）。

Saytzeff 产物示例

2-溴丁烷 + KOH/乙醇：主产物为 2-丁烯（约占 80%，二取代），副产物为 1-丁烯（一取代）。反-2-丁烯>顺-2-丁烯>1-丁烯（稳定性递减）。

3.2 Hofmann 规则（例外）

Hofmann 规则

当使用 大位阻强碱（如 (CH3​)3​CONa (t-BuOK)、R4​N+OH− 季铵碱）或离去基团体积大（季铵基）时，碱优先拔取位阻较小的 β-H（常为端基 H），生成取代基较少的 Hofmann 烯烃（动力学控制产物）。

原因：大位阻碱难接近拥挤的中心 β-H，转而进攻位阻小的端基 H；季铵碱 Hofmann 消除也因 NMe3+​ 体积大且 E2 过渡态偏离典型 Saytzeff 区域。

3.3 两条规则对比

对比项Saytzeff（Zaitsev）规则Hofmann 规则
产物取代基多的烯烃（热力学稳定）取代基少的烯烃（末端双键）
控制类型热力学控制（多数消除）动力学控制
触发条件一般碱（OH−,EtO−）、好的离去基团大位阻碱（t-BuOK）、季铵碱
典型反应卤代烃脱 HX、醇脱水季铵碱 Hofmann 消除
理论解释双键超共轭稳定位阻 / 过渡态偏离

易错点

  • Hofmann 消除不是与 Saytzeff 矛盾，而是动力学控制的不同情形；选用大位阻碱时产物切换。
  • 当底物存在共轭可能时（如生成共轭二烯或与羰基共轭），共轭产物优先，规则需让位于共轭效应。

四、卤代烃的消除反应

4.1 基本反应

X 为 EWG（吸电子基），R−X 的 β-H 具酸性，可在 NaOH/醇（强碱）中消去一分子 HX 得到碳碳双键；从邻二卤或偕二卤代物继续脱 HX 可得碳碳三键。消除是合成烯烃/炔烃的重要途径，也是与取代竞争的主要副反应。

4.2 特殊消除反应

特殊情形

  1. 底物已有双键/三键时，消除烯丙位 H 形成共轭烯炔，产物更稳定（共轭效应优先于规则）
  2. 大位阻碱（t-BuOK）：进攻端基 H，遵循 Hofmann 规则，得末端双键/三键
  3. 氟代烃：C−F 键能大、F− 离去性极差，E2 过渡态偏 E1cb-like；与季铵基类似倾向生成 Hofmann 产物

4.4 消除与取代的竞争

竞争规律

  1. RX 结构：3∘C−X 易消除，1∘C−X 易取代
  2. Nu 性与碱性：Nu 性强（I−,RS−）利于 SN​2；碱性强（RO−,t-BuO−）利于 E2
  3. 碱浓度：浓度越大越利于 E 控pH反应
  4. 大位阻碱：利于消除（易靠近位阻小的 β-H，难靠近拥挤的 α-C）
  5. 温度：高温利于消除（熵增 ΔS>0），低温利于取代 控温
  6. 溶剂：极性质子溶剂利于 SN​1/E1；极性非质子溶剂利于 SN​2/E2

选择策略

  • 想 SN​1：叔卤 + 弱碱亲核试剂（H2​O,ROH）+ 极性质子溶剂
  • 想 SN​2：伯卤 + 强亲核试剂 + 极性非质子溶剂 + 较低温度
  • 想 E1：叔卤 + 弱碱/醇 + 加热
  • 想 E2：仲/叔卤 + 强碱（醇钠、t-BuOK）+ 加热

五、醇的脱水反应（acid-catalyzed dehydration）

5.1 反应类型

类型反应式条件机理与要点
分子内脱水RCH2​CH(OH)R′H2​SO4​Δ​RCH=CHR′Lewis 酸催化（H2​SO4​,H3​PO4​,Al2​O3​）走 E1（叔/仲醇）或 E2（伯醇）；活性 3∘>2∘>1∘；遵循 Saytzeff 规则；Al2​O3​ 高温非均相催化不重排 控温 控pH反应
分子间脱水2ROHH+Δ​R−O−R酸催化、加热走 SN​2（伯醇）；伯/仲醇成醚；叔醇走 E1 不能成醚；低温利于成醚，高温利于成烯 控温 控pH反应

5.2 机理（E1 为主）

醇脱水 E1 机理

  1. 质子化：−OH 被酸质子化为 −OH2+​，成为好的离去基团（H2​O）
  2. 脱水（慢，决速步）：H2​O 离去生成碳正离子 R+
  3. 脱 β-H：碱（HSO4−​、H2​O）拔取 β-H 形成双键
  4. 可能重排：若能生成更稳定碳正离子，发生甲基/负氢迁移

5.3 Zaitsev 选择与重排

区域选择性与重排

  • 生成双键碳上取代基较多的稳定烯烃，遵循 Saytzeff 规则
  • 烯丙型/苄基型醇脱水形成共轭稳定烯烃，常优先
  • 2-丁醇脱水：主产物 2-丁烯（约 90%），副产物 1-丁烯（约 10%）
  • 重排实例：3,3-二甲基-2-丁醇在酸催化脱水时，仲碳正离子经甲基迁移生成更稳定的叔碳正离子，产物为 2,3-二甲基-2-丁烯（非预期的 3,3-二甲基-1-丁烯）

脱水易错点

  • 脱水为 E1 机理，经碳正离子可能重排（甲基/负氢迁移生成更稳定碳正离子）；Al2​O3​ 催化因非均相无碳正离子故不重排
  • 分子内脱水与分子间脱水竞争：低温（约 140 °C）成醚，高温（约 170 °C）成烯——乙醇制乙醚 vs 制乙烯的经典温度控制
  • 叔醇在酸条件下极易消除，不能用叔醇经分子间脱水制醚（应改用 Williamson）
  • 伯醇脱水倾向于 E2（伯碳正离子不稳定），需较高温度；仲/叔醇走 E1

六、季铵碱的 Hofmann 消除

6.1 反应通式

胺经彻底甲基化（与 CH3​I 反应生成季铵盐）后再用 Ag2​O 处理得季铵碱（quaternary ammonium hydroxide），加热发生 Hofmann 消除生成烯烃与叔胺：

R−CH2​−CH2​−N+(CH3​)3​OH−Δ​R−CH=CH2​+(CH3​)3​N+H2​O

6.2 区域选择性与立体化学

Hofmann 消除特点

  • 产物为 Hofmann 烯烃（取代基较少的双键，即末端烯烃为主），因 N+(CH3​)3​ 体积大、E2 过渡态偏离 Saytzeff
  • 机理为 E2（季铵基为离去基团，OH− 为碱）
  • 反式共平面要求：β-H 与 NMe3+​ 处于反式共平面构象才能消除
  • 当存在多种 β-H 时，能形成反式共平面者被消除——这使 Hofmann 消除成为测定胺结构的重要工具（推断碳骨架）

6.3 应用：测定胺的结构

结构推断

  1. 未知胺与过量 CH3​I 反应 → 季铵盐（统计引入的甲基数可推 N 上取代基）
  2. Ag2​O 处理 → 季铵碱
  3. 加热消除 → 烯烃（断裂位置标示原胺的 C-N 键所在）
  4. 重复彻底甲基化—消除循环，可逐步降解并推断原胺的碳骨架，是经典结构鉴定方法之一

与卤代烃消除对比

卤代烃 E2 多得 Saytzeff 产物；季铵碱 E2 多得 Hofmann 产物。差异源于离去基团 NMe3+​ 的强吸电子诱导（−I）使 β-H 酸性差异变小、过渡态偏 E1cb，且体积大阻碍对中央 β-H 的进攻。

七、羧酸及其盐的脱羧反应

7.1 脱羧条件总览

脱羧条件

当 R 为吸电子基（EWG，如 −NO2​,−CO−,−CN,−COOH,C=C）时，β-位存在不饱和键的羧酸易脱羧。β-酮酸、丙二酸、硝基乙酸等尤为容易。

R−COOHNaOHΔ​R−H+CO2​↑

脱羧机理

  • β-羰基酸脱羧：经六元环状过渡态，羰基氧与羧基氢形成氢键，C−C 键断裂同时质子转移，温和条件即可脱羧——是乙酰乙酸乙酯酮式分解、TCA 循环中草酰琥珀酸脱羧的化学基础
  • 一般羧酸脱羧：需高温、强碱；过渡态形成碳负离子，故 EWG 稳定碳负离子时易脱羧
  • 生物体内脱羧：由脱羧酶（含辅酶焦磷酸硫胺素 TPP / 磷酸吡哆醛 PLP）催化，温和进行

7.2 Kolbe 电解合成（Kolbe electrolysis）

Kolbe 电解

电解羧酸盐溶液（阳极氧化脱羧偶联）： 2R−COO−电解​R−R+2CO2​+2e− 两个羧酸根在阳极失去电子、脱羧生成烷基自由基，再偶联生成烷烃 R−R；适用于同种羧酸盐偶联制备对称烷烃，是构建碳碳键的电化学方法。

7.3 Hunsdiecker 反应

Hunsdiecker 反应

羧酸银盐与溴反应得少一个碳的溴代烃： R−COOAg+Br2​→R−Br+CO2​+AgBr↓ 经自由基机理（酰基次溴酸酯均裂 → 羧基自由基脱羧 → 烷基自由基与溴结合），得少一个碳的溴代烷；是由羧酸降解制备卤代烃的方法。改良法（Cristol 改良、Barton 改良）使用 HgO 与卤素或三氯乙酸铅，提高收率。

7.4 β-酮酸与丙二酸的加热脱羧

易脱羧的两类典型

β-酮酸 R−CO−CH2​−COOH 与 丙二酸 HOOC−CH2​−COOH 加热即脱羧： R−CO−CH2​−COOHΔ​R−CO−CH3​+CO2​ HOOC−CH2​−COOHΔ​CH3​−COOH+CO2​

二者均通过六元环状过渡态实现协同脱羧。对应在合成上：乙酰乙酸乙酯合成法（→甲基酮）、丙二酸酯合成法（→取代乙酸）均依赖水解脱羧释放目标产物。

7.5 二元酸的受热反应

碳链长度反应类型产物
乙二酸、丙二酸脱羧一元酸 + CO2​
丁二酸、戊二酸脱水五/六元环状酸酐
己二酸、庚二酸既脱羧又脱水环酮（五/六元环）

Blanc 规律

二元酸受热反应倾向于生成五元环或六元环化合物（成环热力学有利）。

脱羧易错点

  • 脱羧一般生成少一个碳的烃，不是少两个碳；Kolbe/Hunsdiecker 各有不同（前者偶联生成 R−R，后者生成 R−Br）
  • 普通一元羧酸高温才能脱羧，β-酮酸/丙二酸温和即可——切勿混用条件
  • β-酮酸脱羧属协同机理（六元环过渡态），不经过碳负离子中间体

八、α-消除与卡宾

8.1 卡宾的生成

Tip α-消除生成卡宾

同一碳上脱去两个基团（一般为一个 H 与一个离去基）生成 卡宾（carbene，:CR2​，六隅体碳中性物种，单重态或三重态）： CHCl3​+B−→:CCl2​+BH+Cl−

  • CHCl3​ 在强碱（KOH/t-BuOK）下脱 HCl 生成二氯卡宾 :CCl2​
  • CH2​I2​+Zn(Cu)（Simmons-Smith 反应）生成类卡宾 ICH2​ZnI，与烯烃加成环丙烷

8.2 卡宾的反应

反应描述
与烯烃加成（环丙烷化）:CH2​+CH2​=CH2​→ 环丙烷；单重态卡宾为顺式立体专一加成
** insertion 插入反应**插入 C−H 键，生成同碳多一个 CH2​ 的产物，常无选择性

单重态 vs 三重态卡宾

  • 单重态：孤对电子反平行自旋，sp2 杂化，与烯烃加成立体专一（顺式保留）
  • 三重态：两未成键电子平行自旋，行为类似双自由基，加成无立体专一性

九、热消除反应（Ei 机理）

9.1 Cope 消除

Cope 消除

叔胺氧化物（amine oxide）加热发生分子内顺式消除生成烯烃与羟胺： R2​CH−CH2​−N+(O−)R2′​Δ​R2​C=CH2​+R2′​N−OH

  • 经五元环状过渡态（O−H⋯C−N），O 内氢转移，顺式消除（syn-elimination）
  • 反应温度低（约 100–150 °C）、副反应少、不重排
  • 区域选择性偏 Hofmann 产物（位阻小处易形成过渡态）

9.2 Chugaev 消除

Chugaev 消除

黄原酸酯（xanthate ester）加热发生分子内顺式消除生成烯烃、COS、S−R： R2​CH−CH2​−O−CS−S−R′Δ​R2​C=CH2​+COS+R′SH

  • 经六元环状过渡态，顺式消除
  • 由醇经 NaOH,CS2​,CH3​I 制黄原酸酯再热解，是醇脱水制烯烃的无重排方法（避免了 E1 重排），常得 Hofmann 产物

9.3 热消除与 β-消除对比

对比项E1/E2 β-消除Ei 热消除（Cope/Chugaev）
试剂外加酸/碱不需外加酸碱，加热即可
立体专一性E2 反式共平面；E1 无顺式消除
重排E1 可重排不重排
区域选择性Saytzeff 为主常得 Hofmann 产物
典型反应卤代烃、醇脱水Cope、Chugaev、酯热解

类型反应通式典型机理立体化学区域选择应用

脱水-脱氨-脱卤化氢-脱羧对比

  • 脱水：质子化 −OH 为 H2​O 离去，经 E1 重排常见
  • 脱氨：季铵基 NMe3+​ 体积大离去性好，强制 Hofmann 产物
  • 脱卤化氢：X− 离去性好（除 F），机理最丰富（E1/E2/E1cb）
  • 脱羧：断裂 C−C 键放出 CO2​，β-羰基/二元酸易协同脱羧

十一、与生物工程的联系

脱羧与代谢

  • β-酮酸脱羧：乙酰乙酸脱羧生成丙酮，是酮体代谢（糖尿病酮症）的化学基础；TCA 循环中草酰琥珀酸脱羧、异柠檬酸→α-酮戊二酸的脱羧均遵循 β-羰基酸脱羧规律
  • 氨基酸脱羧：氨基酸经脱羧酶（辅酶 PLP，磷酸吡哆醛）催化脱羧生成生物胺，是氨基酸代谢关键去路：
    • 谷氨酸 → γ-氨基丁酸（GABA，中枢抑制性神经递质）
    • 组氨酸 → 组胺（炎症与胃酸分泌介质）
    • 5-羟色氨酸 → 5-羟色胺（血清素）
    • 多巴 → 多巴胺（帕金森相关）
    • 赖氨酸 → 尸胺、鸟氨酸 → 腐胺（腐胺/尸胺，尸检特征胺）
  • 丙酮酸脱羧：丙酮酸脱氢酶系（辅酶 TPP、硫辛酸、CoA、FAD、NAD+）将丙酮酸氧化脱羧为乙酰 CoA，连接糖酵解与 TCA 循环
  • Kolbe 电解思想对应生物体系中通过自由基机理进行的氧化偶联（如酚氧化偶联生成木质素）

脱氨与脱卤化氢在生物合成中的角色

  • E1cb 在生物合成：许多酶催化的消除/脱水反应（如脂肪酸合成中的 β-羟基酰基脱水、异戊烯基转移）涉及先形成碳负离子或烯醇负离子再脱去的步骤，机理上类比 E1cb
  • 磷酸烯醇式丙酮酸（PEP）：糖酵解中 2-磷酸甘油酸经烯醇化脱水生成 PEP，是一高能磷酸化合物（类似 E1cb 中间体稳定化）
  • 脱卤化氢酶：微生物降解卤代污染物（如 2,2-二氯丙酸脱氯化氢）依赖脱卤酶，机理涉及水亲核取代或谷氨酸残基介导的 E2 消除；是生物修复卤代烃污染的关键酶
  • Hofmann 消除思想：在生物胺合成的逆向分析（脱羧 + 识别母体氨基酸）中体现；季铵化合物（如胆碱、乙酰胆碱）的代谢也涉及类似季铵断裂

知识检查与易错点

常见易错点

  1. E2 反式共平面：是立体专一性要求，并非”必须有反式异构体”——而是 β-H 与 L 在过渡态须处于反式共平面构象；环己烷中两者须同为直立键 立体选择性
  2. Hofmann vs Saytzeff：大位阻碱（t-BuOK）及季铵碱才得 Hofmann 产物，不是所有消除都得 Hofmann 烯烃
  3. E1 可重排、E2/E1cb 不可重排：因 E1 经碳正离子中间体，其余机理无碳正离子
  4. 醇脱水重排：E1 中碳正离子可重排（甲基/负氢迁移）；Al2​O3​ 催化不重排
  5. 醇脱水的温度竞争：低温（~140 °C）成醚，高温（~170 °C）成烯；叔醇不能成醚
  6. 邻二卤/偕二卤制炔：脱两分子 HX（不是两分子 H），需 2 当量强碱（如 NaNH2​）
  7. 脱羧碳数：普通脱羧产少一碳烃；Kolbe 电解得偶联 R−R；Hunsdiecker 得少一碳 R−Br——三者切勿混淆
  8. β-酮酸/丙二酸脱羧温和：六元环过渡态协同脱羧，不经过碳负离子中间体；普通羧酸需高温强碱
  9. Cope/Chugaev 为顺式消除：与 E2 反式共平面相反，热消除专属特征
  10. E1cb 条件：需同时满足”β-H 酸性强 + 离去基差 + 强碱”三要素，不可随意套用

思考检查

  1. 为何 2-溴丁烷在 EtO− 下的 E2 主产物为反-2-丁烯？——反式共平面构象 + 反式烯烃更稳定。
  2. 为何 2-溴-2-甲基丁烷在弱碱水中以 SN​1 为主、在乙醇钠中以 E2 为主？——碱性质子溶剂降低 Nu 活性促进 SN​1；强碱+较高温度促进 E。
  3. 如何实现 3,3-二甲基-2-丁醇脱水不重排地得烯烃？——改用 Al2​O3​ 高温催化或转黄原酸酯 Chugaev 消除。
  4. 为何季铵碱消除得 Hofmann 产物？——NMe3+​ 体积大且 −I 强，过渡态偏 E1cb，碱优先拔端基 H。
