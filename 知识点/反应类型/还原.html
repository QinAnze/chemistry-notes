还原反应

Jun 27, 2026, 8 min read

还原反应 (Reduction Reaction)

还原反应是通过增加氢原子或减少氧原子来降低有机分子氧化态的反应，是合成中的重要手段。

一、还原反应的基本概念

定义

还原：有机物获得电子或增加氢原子、减少氧原子的过程。

常见还原路径：

  • −COOH → −CHO → −CH2​OH → −CH3​
  • 羧酸 → 醛 → 醇 → 烷烃（氧化态逐级降低）

二、催化氢化还原

1. 一般催化氢化

H₂/催化剂还原

使用 Ni、Pd、Pt 等催化剂，H₂ 进行催化加氢：

反应物产物催化剂
烯烃烷烃Ni、Pd、Pt
炔烃烯烃/烷烃Pd、Lindlar
醛酮醇Ni、Pd
硝基化合物胺Ni、Pd

注意：

  • 催化氢化不能还原羧酸
  • 双键比酮羰基更容易被还原
  • 顺式加成（syn-addition）

2. Lindlar催化剂——选择性还原炔烃

Lindlar催化剂

组成：Pd/CaCO₃ 或 Pd/BaSO₄，经醋酸铅、硫、喹啉毒化

特点：

  • 活性降低，选择性还原炔烃 → 顺式烯烃
  • 不继续还原烯烃
  • 用于合成 cis-烯烃

R−C≡C−R′Lindlar​R−CH=CH−R′（顺式）

3. 温和条件 vs 加热条件

催化加氢的选择性

条件结果
温和条件（不加热）仅还原双键，酮羰基保留
加热全部还原（双键 + 酮羰基）

→ 双键比酮羰基更容易被催化加氢还原

三、化学还原剂

1. LiAlH₄（氢化铝锂）——强还原剂

LiAlH₄ 的还原范围

强还原剂，可还原多种官能团：

反应物产物
醛、酮伯醇、仲醇
羧酸伯醇
酯伯醇 + 醇
酰胺胺
腈 (R−CN)伯胺

特点：

  • 不还原不饱和烃（双键、三键）
  • 需无水条件，遇水分解
  • 后处理需小心淬灭（稀酸或 NH₄Cl）

LiAlH₄ 机理细节

LiAlH₄ 的四个 H

LiAlH₄ 的四个 H 的去向：

  • 两个 H 与羰基的 O 结合形成 H₂O 离去
  • 两个 H 加到碳上（亲核加成）
  • 水解后，水的 H 加到醇的 O 上

同位素标记结论：

  • 用 LiAlD₄ → 生成 –CD₂–（两个 D 加到碳上）

  • 用 D₂O 淬灭 → 生成 –OD（D 在羟基上）

区分

加到碳上的 H 来自还原剂，羟基上的 H 来自水。

2. NaBH₄（硼氢化钠）——温和还原剂

NaBH₄ 的还原范围

温和还原剂，选择性高：

反应物产物
醛、酮伯醇、仲醇

不还原：

  • 羧酸
  • 酯
  • 酰胺
  • 不饱和烃

特点：

  • 反应条件温和（可在醇溶液中使用）
  • 选择性高，只还原醛酮
  • 比 LiAlH₄ 安全

3. B₂H₆（硼烷）——硼氢化还原

硼氢化-氧化反应

B₂H₆ 用于烯烃的硼氢化还原（协同加成）：

R−CH=CH2​B2​H6​​R−CH2​−CH2​−BH2​O2​/OH−​R−CH2​−CH2​−OH

特点：

  • 反马氏规则：OH 加到取代较少的碳
  • 顺式加成：H 与 OH 在同侧
  • 无中间体，协同机理

四、特殊还原方法

1. Wolff-Kishner 还原

Wolff-Kishner 还原

将羰基还原为亚甲基： R−CO−R′NH2​NH2​,KOH,高温​R−CH2​−R′

条件：

  • 强碱性条件
  • 高温（180–200°C）
  • 肼 (NH₂NH₂) + KOH

适用：

  • 对酸敏感的化合物
  • Clemmensen 还原不能进行的场合

2. Clemmensen 还原

Clemmensen 还原

将羰基还原为亚甲基： R−CO−R′Zn−Hg,HCl​R−CH2​−R′

条件：

  • 酸性条件
  • Zn-Hg（锌汞齐）+ 浓 HCl

适用：

  • 对碱敏感的化合物
  • Wolff-Kishner 还原不能进行的场合

3. Birch 还原

Birch 还原

将芳香环部分还原为 1,4-环己二烯： 苯Na/NH3​(l),醇​1,4−环己二烯

条件：

  • 碱金属（Na、Li、K）
  • 液氨
  • 醇（提供质子）

特点：

  • 电子效应决定还原位置
  • EWG → 1,4-加成（非取代位置还原）
  • EDG → 2,5-加成（取代位置保留）

4. Na/醇还原酯

Na + 醇还原酯

金属钠在醇溶液中还原酯为醇： R−COOR′Na/醇​R−CH2​OH+R′OH

特点：

  • 不还原不饱和烃
  • 可制备不饱和醇
  • Bouveault-Blanc 还原

五、还原反应的选择性

1. 还原剂能力对比

还原剂能力排序

还原剂还原能力选择性
LiAlH₄最强可还原羧酸、酰胺、腈
NaBH₄中只还原醛酮
H₂/催化剂弱-中还原不饱和键、硝基
B₂H₆特殊反马氏硼氢化

还原范围：

  • LiAlH₄：醛酮、羧酸、酯、酰胺、腈
  • NaBH₄：仅醛酮
  • H₂/Ni/Pd：烯炔、醛酮、硝基（不还原羧酸）

2. 官能团还原顺序

还原优先级

官能团还原难易最佳还原剂
烯烃、炔烃最易H₂/Pd、Lindlar
醛、酮易NaBH₄、LiAlH₄
酯中等LiAlH₄、Na/醇
羧酸较难LiAlH₄
酰胺较难LiAlH₄

选择性还原策略：

  • 有双键 + 羰基 → 先用温和条件还原双键，再用 NaBH₄ 还原羰基
  • 有醛酮 + 酯 → 先用 NaBH₄ 还原醛酮，再用 LiAlH₄ 还原酯

六、还原反应实例

典型还原反应

实例1：丙酮的还原 CH3​COCH3​NaBH4​​CH3​CH(OH)CH3​ CH3​COCH3​LiAlH4​​CH3​CH(OH)CH3​

实例2：乙酸乙酯的还原 CH3​COOCH2​CH3​LiAlH4​​CH3​CH2​OH+CH3​CH2​OH

实例3：炔烃的选择性还原 R−C≡C−R′Lindlar​R−CH=CH−R′（顺式） R−C≡C−R′Na/NH3​​R−CH=CH−R′（反式）

实例4：硝基苯的还原 Ph−NO2​H2​/Pd​Ph−NH2​

还原剂还原对象特点

考试重点提醒

  1. ⭐ LiAlH₄ vs NaBH₄：

    • LiAlH₄：可还原羧酸、酰胺、腈
    • NaBH₄：只还原醛酮，不还原羧酸

  2. ⭐ 催化氢化不能还原羧酸：

    • H₂/Ni/Pd 不还原羧酸
    • 需用 LiAlH₄ 还原羧酸

  3. ⭐ 炔烃的选择性还原：

    • Lindlar → cis-烯烃
    • Na/NH₃ → trans-烯烃

  4. ⭐ 硼氢化-氧化：

    • 反马氏规则
    • 顺式加成

  5. ⭐ 同位素标记：

    • LiAlD₄ → –CD₂–（D 在碳上）
    • D₂O 淬灭 → –OD（D 在羟基上）
